For Windows XP, 32-bit Windows Vista, or 32-bit Windows 7:

	Download and install DoubleAgent_x86.msi. 

For 64-bit Windows Vista 64-bit or Windows 7:

	Download and install DoubleAgent_x64x86.msi 

After the installation is complete, use the Double Agent control panel to configure it.  Pay special attention to the "Emulation" setting.
