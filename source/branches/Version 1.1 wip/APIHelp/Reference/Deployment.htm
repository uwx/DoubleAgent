Deploying Double Agent

The Double Agent installation files are Microsoft Installer (MSI) packages. They can be installed using the msiexec program that's part of Windows.  Here are some sample command-lines (refer to Microsoft's MSI documentation for more details):

  • Install on a 64-bit system with the full user interface:

    msiexec /i DoubleAgent_x64x86.msi

  • Install on a 32-bit system with only a progress indicator and no user interaction:

    msiexec /i DoubleAgent_x86.msi /qb-!

  • Install with a progress indicator and cancel button, and a confirmation dialog at the end:

    msiexec /i DoubleAgent_x86.msi /qb+

  • Install with emulation enabled and only a confirmation dialog at the end:

    msiexec /i DoubleAgent_x86.msi /qn+ AUTO_EMULATE=1

If you configure your installation to automatically install Double Agent, you must include the appropriate Open Source acknowledgement (see Licensing and Distribution).

If you choose to deploy Double Agent with less than the full user interface, you should include some kind reference to the GNU license agreement in your installation. The GNU licenses in HTML format are included in the Double agent installation.
