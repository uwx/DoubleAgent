Documentation Conventions

This documentation uses the following typographical conventions:

Convention Description
Sub, Visible, Caption Words in bold with an initial capital letter are keywords.
control, String, Now Italic words are placeholders for information you supply.
control The word "control" in italics represents a Double Agent Control Object.
Enter, F1 Words in all capital letters indicate filenames, key names, and key sequences.
control.Commands.Enabled = True Words in this font are code samples.
' This is a comment An apostrophe (') indicates a code comment.
[words or expression] Items inside square brackets are optional.
This | That A vertical bar indicates a choice between two or more items.

The programming examples in this documentation follow the conventions for Microsoft® VBScript. However, they are generally applicable to other languages as well.
