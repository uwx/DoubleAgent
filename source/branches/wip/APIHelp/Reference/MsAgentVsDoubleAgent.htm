Double Agent SDK

Notable differences between Double Agent and Microsoft® Agent

Double Agent supports the functionality implemented by Microsoft® Agent on Windows XP and Windows Vista, with the following differences:

  • Microsoft® Agent supports help mode on Windows XP but not on Windows Vista. Double Agent does not support help mode (yet).
  • The 32-bit version of Double Agent supports SAPI4 text-to-speech engines, but it does not support SAPI4 voice recognition engines.
  • Microsoft® Agent supports linguistically enhanced sound files for text-to-speech. Double Agent does not support this feature (for now).
  • Microsoft® Agent supports international languages on Windows XP but not on Windows Vista. Double Agent does support international languages.
  • In a 64-bit environment, Double Agent may use a different server architecture than Microsoft Agent. See Targeting 64-bit Windows for details.
  • Double Agent does not include any of the Microsoft® Agent character editing tools. However, since Double Agent uses Microsoft® Agent character files, the original tools can still be used.
