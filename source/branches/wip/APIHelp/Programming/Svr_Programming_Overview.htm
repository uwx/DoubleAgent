Double Agent API

Server Programming Overview

To use the server interfaces, Double Agent must already be installed on the target system. If you want to install Double Agent as part of your application, you can download Double Agent's installation (.MSI) package from the Web site. You can include this file in your installation setup program (with the required acknowledgement). Whenever it is executed, it will install Double Agent on the target system.

The Microsoft® Agent language components can be installed the same way. Similarly, you can use this technique to install the .ACS format of the Microsoft® characters, or your own characters.

To program the server, you will also need to install Double Agent's Application Interfaces components. This is an optional part of the Double Agent installation. To install it, you must download its .CAB file to the same location as the .MSI file.

Once Double Agent and its Application Interfaces components are installed, the following development files will be available in the installation folder:

File Description
Dev\DaServer.tlb The registered Double Agent Server type library
Dev\DaServerOdl.h The generated C/C++ header file for the Double Agent Server type library
Dev\DaServerInterface.h C++ Smart pointer definitions for the Double Agent Server type library (includes DaServerOdl.h)
DoubleAgent.Server.dll The Double Agent Server Primary Interop Assembly
DoubleAgent.Server.xml Intellisense data for DoubleAgent.Server.dll
