Programming the Double Agent Control

Double Agent includes a Microsoft ActiveX® Control that can be used with any language that supports the ActiveX® Control Interface. It provides for easy programming using an interpreted language such as Microsoft Visual Basic®, or a scripting language such as Microsoft Visual Basic® Scripting Edition (VBScript).

The Double Agent control can be used as a simple programming object, either stand-alone or connected to the Double Agent server. As an ActiveX® control, it can provide character animations within a form or Web page.
