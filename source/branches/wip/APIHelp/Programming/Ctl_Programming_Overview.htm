Double Agent API

Control Programming Overview

To access the control, Double Agent must already be installed on the target system. If you want to install Double Agent as part of your application, you can download Double Agent's installation (.MSI) package from the Web site. You can include this file in your installation setup program (with the required acknowledgement). Whenever it is executed, it will install Double Agent on the target system.

To program the control, you will also need to install Double Agent's Application Interfaces components. This is an optional part of the Double Agent installation. To install it, you must download its .CAB file to the same location as the .MSI file.

Once Double Agent and its Application Interfaces components are installed, the following development files will be available in the installation folder:

File Description
Dev\DaControlOdl.h The generated C/C++ header file for the Double Agent Control type library
Dev\DaControlInterface.h C++ Smart pointer definitions for the Double Agent Control type library (includes DaControlOdl.h)
DoubleAgent.AxControl.dll The Double Agent ActiveX® Control Primary Interop Assembly
DoubleAgent.AxControl.xml Intellisense data for DoubleAgent.AxControl.dll
DoubleAgent.Control.dll The Double Agent Control Primary Interop Assembly
DoubleAgent.Control.xml Intellisense data for DoubleAgent.Control.dll
