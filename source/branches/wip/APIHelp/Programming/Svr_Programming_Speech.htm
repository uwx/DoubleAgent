Double Agent API

Using Speech Services

Although Double Agent includes support for synthesized speech output, a compatible text-to-speech (TTS) speech engine must be installed to access Double Agent's speech output features. Similarly, if you want to use Double Agent's speech features to support speech input for a character, you must install a compatible command-and-control speech recognition engine.

To produce text-to-speech output, use the Speak method. Double Agent will automatically attempt to load an engine that matches the character's LanguageID. If the character's definition includes a specific TTSModeID and that engine is available and matches the character's LanguageID, Double Agent loads that engine for the character. If not, Agent loads the first TTS engine that SAPI returns as matching the character's language setting. You can also use the TTSModeID property to load a specific engine. The TTSEngines collection will help you determine which engines are available and which ones are compatible with the character.

To enable speech input support in your application, define a Command object and set its Voice Caption property. Double Agent will automatically load speech services, so that when the user presses the Listening key or you call Listen, the speech recognition engine will be loaded. By default the character's LanguageID will determine which engine is loaded. Double Agent attempts to load the first engine that the Microsoft® Speech API (SAPI) returns as matching this language. Use the SRModeID property if you want to load a specific engine. The SREngines collection will help you determine which engines are available and which ones are compatible with the character.

Typically, Double Agent loads a text-to-speech engine when Speak is first called and a speech recognition when the Listening mode is initiated. However, if you want to preload the speech engine, you can query the properties related to the speech interfaces. For example, querying TTSModeID or SRModeID will attempt to load that type of engine.

Because Double Agent's speech features are based on the Microsoft® Speech API, you can use any engines that support the required speech interfaces. The Double Agent 32-bit build supports Microsoft® Speech (SAPI) versions 4 and 5 for text-to-speech and version 5 for speech recognition. The Double Agent 64-bit build supports only version 5 for both text-to-speech and speech recognition.
