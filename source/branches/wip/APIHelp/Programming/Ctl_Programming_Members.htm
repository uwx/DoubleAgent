Double Agent API

Using the Control's Methods, Properties, and Events

Declaring Objects

Programming languages that provide access to the object types exposed by the Double Agent control, like Visual Basic, allow you to use these in your object declarations. For example, instead of declaring an object as a generic type:

Dim Genie as Object

you can declare an object as a specific type:

Dim Genie as DoubleAgentCtl.DaCtlCharacter

or, when using the control's Primary Interop Assembly:

Dim Genie as DoubleAgent.Control.Character

This allows the programming environment to identify the object's methods, properties, and events, and may improve the overall performance of your application.

Using Collections

Most programming languages that support ActiveX® controls follow indexing conventions similar to Visual Basic. For programming languages that don't support object collections, you can use the get_Item method to access methods and properties of items in the collection.

Initializing Properties

By default, the Double Agent control will automatically create a connection to the Double Agent server when the application runs. If you want the control to run in stand-alone mode, initialize the control's AutoConnect property to False.

You should also initialize the RaiseRequestErrors property depending on how you want to handle errors.

Receiving Events

When using a programming language that supports ActiveX® controls or Windows Forms controls, adding the Double Agent control to a form will automatically make the control's events available, with their appropriate parameters.

You may also be able to use your programming language's object creation syntax to create an instance of the control at runtime. For example, in Visual Basic 6, if you include the Double Agent control in your project's references, you can use a With Events...New declaration (if you don't include the reference, VB raises an error).

' Declare a global variable for the control
Dim WithEvents MyAgentControl as DoubleAgentCtl.DaControl

' Create an instance of the control using New
Set MyAgentControl = New DoubleAgentCtl.DaControl

' Load a character
MyAgentControl.Characters.Load "Genie","Genie.acs"

' Display the character
MyAgentControl.Characters("Genie").Show
