Double Agent API

The IdleState Property

Description

Indicates if the character is currently playing its Idling animations.

Syntax

Visual Basic .NET

[IdleState = ]control.Characters ("CharacterID").IdleState

Visual Basic 6

[IdleState = ]control.Characters ("CharacterID").IdleState

VBScript

[IdleState = ]control.Characters ("CharacterID").IdleState

Visual C#

[IdleState = ]control.Characters.this ["CharacterID"].IdleState

Visual C++

character->get_IdleState (&IdleState)
Part Description
IdleState A boolean value that indicates if the character is idling.

Remarks

Note: This property applies to all clients of the character.
