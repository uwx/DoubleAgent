Double Agent API

The NewFormattedText Method

Description

Returns an initialized, empty FormattedText Object.

Syntax

Visual C++

character->NewFormattedText (&FormattedText)

Visual C#

FormattedText = character.NewFormattedText ()

Visual Basic .NET

FormattedText = character.NewFormattedText ()

Visual Basic 6

Set FormattedText = character.NewFormattedText ()

VBScript

Set FormattedText = character.NewFormattedText ()
Part Description
FormattedText A FormattedText Object.

Remarks

The returned FormattedText Object is initialized to handle speech tags consistent with the Character's current Text-to-Speech engine.

Once you have completed the FormattedText Object, you can pass it to either the SpeakFormatted or ThinkFormatted method. If you plan to use the same text more than once, keep a reference to the FormattedText Object and reuse it. This can improve the performance of your application.
