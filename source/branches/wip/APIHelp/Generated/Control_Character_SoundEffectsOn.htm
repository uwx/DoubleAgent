Double Agent API

The SoundEffectsOn Property

Description

Returns or sets whether sound effects are enabled for your character.

Syntax

Visual Basic .NET

[SoundEffectsOn = ]control.Characters ("CharacterID").SoundEffectsOn

control.Characters ("CharacterID").SoundEffectsOn = SoundEffectsOn

Visual Basic 6

[SoundEffectsOn = ]control.Characters ("CharacterID").SoundEffectsOn

control.Characters ("CharacterID").SoundEffectsOn = SoundEffectsOn

VBScript

[SoundEffectsOn = ]control.Characters ("CharacterID").SoundEffectsOn

control.Characters ("CharacterID").SoundEffectsOn = SoundEffectsOn

Visual C#

[SoundEffectsOn = ]control.Characters ("CharacterID").SoundEffectsOn

control.Characters ("CharacterID").SoundEffectsOn = SoundEffectsOn

Visual C++

character.get_SoundEffectsOn (&SoundEffectsOn)

character.put_SoundEffectsOn (SoundEffectsOn)

Part Description
SoundEffectsOn A boolean value specifying whether sound effects are enabled.

Remarks

This property determines whether the sound effects included as a part of a character's animations will play when an animation plays. This property's setting applies to all clients of the character.
