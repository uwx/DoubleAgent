Double Agent API

The ConfidenceText Property

Description

Specifies the text that appears in the Listening tip.

Syntax

Visual C++

command->get_ConfidenceText (&ConfidenceText)

command->put_ConfidenceText (ConfidenceText)

Visual C#

[ConfidenceText = ]command.ConfidenceText

command.ConfidenceText = ConfidenceText

Visual Basic .NET

[ConfidenceText = ]command.ConfidenceText

command.ConfidenceText = ConfidenceText

Visual Basic 6

[ConfidenceText = ]command.ConfidenceText

command.ConfidenceText = ConfidenceText

VBScript

[ConfidenceText = ]command.ConfidenceText

command.ConfidenceText = ConfidenceText
Part Description
ConfidenceText A string value that specifies the text displayed in the Listening Tip.

Remarks

When the ItemConfidence value of the best match does not exceed the command's ConfidenceThreshold property, Double Agent displays the text specified in ConfidenceText in the Listening tip.
