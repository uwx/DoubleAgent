Double Agent API

The Unload Method

Description

Unloads a Character and releases any resources it uses.

Syntax

Visual C++

server->Unload (CharacterID)

Visual C#

server.Unload (CharacterID)

Visual Basic .NET

server.Unload (CharacterID)

Visual Basic 6

server.Unload CharacterID

VBScript

server.Unload CharacterID
Part Description
CharacterID The unique numeric identifier assigned when the character was loaded.

Remarks

When you no longer need a Character, use this method to free up memory used to store information about the character. To access the character again, use the Load method.
