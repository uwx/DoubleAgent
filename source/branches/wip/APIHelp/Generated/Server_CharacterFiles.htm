The CharacterFiles Object

Description

The CharacterFiles Object provides information that allows you to locate locally installed character definition files. It includes a Filter that specifies where to look for character files, and what kind of files to find. The results are returned in its FilePaths property.

Properties

DefaultFileName Returns the name of the default character's definition file.
DefaultFilePath Returns the full path of the default character's definition file.
DefaultSearchPath Identifies the default search path used to fill the FilePaths property.
FilePaths Returns a list of character file paths that satisfy the filtering criteria defined by other properties in the CharacterFiles Object.
Filter Specifies the filtering criteria applied to the FilePaths and DefaultSearchPath properties.
SearchPath Identifies the actual search path used to fill the FilePaths property. Unless you have special requirements, this is usually the same as the DefaultSearchPath property.
