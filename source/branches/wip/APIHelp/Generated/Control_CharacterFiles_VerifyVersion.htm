Double Agent API

The VerifyVersion Property

Description

Indicates if the the FilePaths property should be limited to version 2.x characters.

Syntax

Visual Basic .NET

[VerifyVersion = ]control.CharacterFiles.VerifyVersion

control.CharacterFiles.VerifyVersion = VerifyVersion

Visual Basic 6

[VerifyVersion = ]control.CharacterFiles.VerifyVersion

control.CharacterFiles.VerifyVersion = VerifyVersion

VBScript

[VerifyVersion = ]control.CharacterFiles.VerifyVersion

control.CharacterFiles.VerifyVersion = VerifyVersion

Visual C#

[VerifyVersion = ]control.CharacterFiles.VerifyVersion

control.CharacterFiles.VerifyVersion = VerifyVersion

Visual C++

characterfiles->get_VerifyVersion (&VerifyVersion)

characterfiles->put_VerifyVersion (VerifyVersion)
Part Description
VerifyVersion A boolean value that indicates if the character files' version should be verified.

Remarks

Double Agent can only use version 2.x characters.

Note: Setting the SpeakingCharacters or NonSpeakingCharacters filters will automatically set this property to True.
