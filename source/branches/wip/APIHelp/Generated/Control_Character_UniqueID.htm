Double Agent API

The UniqueID Property

Description

Returns the unique identifier given to the character when it was defined.

Syntax

Visual Basic .NET

[UniqueID = ]control.Characters ("CharacterID").UniqueID

Visual Basic 6

[UniqueID = ]control.Characters ("CharacterID").UniqueID

VBScript

[UniqueID = ]control.Characters ("CharacterID").UniqueID

Visual C#

[UniqueID = ]control.Characters.this ["CharacterID"].UniqueID

Visual C++

character->get_UniqueID (&UniqueID)
Part Description
UniqueID A string value containing the the character's unique (GUID) identifier.

Remarks

This property returns a string representing the internal identifier that Double Agent uses to uniquely identify the character. A character identifier is set when it is compiled with the Microsoft Agent Character Editor.

Note: This is the value that Double Agent uses to determine if a character has already been loaded. The character's file name or location does not uniquely identify the character.
