Double Agent API

The Characters Collection

Description

The Characters Collection includes methods and properties for loading, unloading, and accessing Character Objects.

Methods

Character Returns a Character Object in the Characters Collection.
Load Loads a character into the Characters Collection.
Unload Unloads the character data for a loaded character.

Properties

Count Returns the count of Character Objects in the Characters Collection.
Index Returns a Character Object in the collection.
Item Returns a Character Object in the collection.

Remarks

This object can be accessed using the Control Object's Characters property.
