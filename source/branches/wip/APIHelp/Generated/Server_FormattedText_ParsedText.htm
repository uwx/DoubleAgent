The ParsedText Property

Description

Returns the text formatted for display.

Syntax

Visual C++

formattedtext->get_ParsedText (&ParsedText)

Visual C#

[ParsedText = ]formattedtext.ParsedText

Visual Basic .NET

[ParsedText = ]formattedtext.ParsedText

Visual Basic 6

[ParsedText = ]formattedtext.ParsedText

VBScript

[ParsedText = ]formattedtext.ParsedText
Part Description
ParsedText A string value containing the text formatted for displaying in a word balloon.
