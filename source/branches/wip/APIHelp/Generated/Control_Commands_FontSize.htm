Double Agent API

The FontSize Property

Description

Returns or sets the font size used in the Voice Commands window.

Syntax

Visual Basic .NET

[FontSize = ]control.Characters ("CharacterID").Commands.FontSize

control.Characters ("CharacterID").Commands.FontSize = FontSize

Visual Basic 6

[FontSize = ]control.Characters ("CharacterID").Commands.FontSize

control.Characters ("CharacterID").Commands.FontSize = FontSize

VBScript

[FontSize = ]control.Characters ("CharacterID").Commands.FontSize

control.Characters ("CharacterID").Commands.FontSize = FontSize

Visual C#

[FontSize = ]control.Characters ("CharacterID").Commands.FontSize

control.Characters ("CharacterID").Commands.FontSize = FontSize

Visual C++

commands->get_FontSize (&FontSize);

commands->put_FontSize (FontSize);

Part Description
FontName A numeric value specifying the font size (in points).

Remarks

The FontSize property defines the point size of the font used to display text in the CommandsWindow. The default value for the font setting is based on the menu font for the character's LanguageID or, if that's not set, the user default language.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
