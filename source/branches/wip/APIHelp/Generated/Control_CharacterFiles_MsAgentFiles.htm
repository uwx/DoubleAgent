Double Agent API

The MsAgentFiles Property

Description

Indicates if the Microsoft Agent install location should be included in the DefaultSearchPath.

Syntax

Visual Basic .NET

[MsAgentFiles = ]control.CharacterFiles.MsAgentFiles

control.CharacterFiles.MsAgentFiles = MsAgentFiles

Visual Basic 6

[MsAgentFiles = ]control.CharacterFiles.MsAgentFiles

control.CharacterFiles.MsAgentFiles = MsAgentFiles

VBScript

[MsAgentFiles = ]control.CharacterFiles.MsAgentFiles

control.CharacterFiles.MsAgentFiles = MsAgentFiles

Visual C#

[MsAgentFiles = ]control.CharacterFiles.MsAgentFiles

control.CharacterFiles.MsAgentFiles = MsAgentFiles

Visual C++

characterfiles.get_MsAgentFiles (&MsAgentFiles)

characterfiles.put_MsAgentFiles (MsAgentFiles)

Part Description
MsAgentFiles A boolean value that indicates if the Microsoft Agent install location should be included in the search for character files.

Remarks

The path used to find character files is the Chars subdirectory of the Microsoft Agent install location. Typically, this would be C:\Windows\MsAgent\Chars.
