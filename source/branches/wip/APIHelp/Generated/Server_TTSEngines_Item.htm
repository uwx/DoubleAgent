Double Agent API

The Item Property

Description

Returns a TTSEngine Object in the collection.

Syntax

Visual C++

ttsengines->get_Item (Index, &TTSEngine)

Visual C#

[TTSEngine = ]ttsengines.this [Index]

Visual Basic .NET

[TTSEngine = ]ttsengines (Index)

Visual Basic 6

[Set TTSEngine = ]ttsengines (Index)

VBScript

[Set TTSEngine = ]ttsengines (Index)
Part Description
Index A numeric value that specifies the zero-based index of the item to retrieve,
or a string value that specifies the TTSModeID of the item to retrieve.
TTSEngine The TTSEngine Object identified by Index.

Remarks

When the TTSEngines Collection is retrieved using matching criteria (for example using the FindTTSEngines method), it is an ordered collection. It's items are in order from best to worst match.
