Double Agent API

The Append Method

Description

Appends one FormattedText Object to another FormattedText Object.

Syntax

Visual C++

formattedtext->Append (FormattedText)

Visual C#

formattedtext.Append (FormattedText)

Visual Basic .NET

formattedtext.Append FormattedText

Visual Basic 6

formattedtext.Append FormattedText

VBScript

formattedtext.Append FormattedText
Part Description
FormattedText The FormattedText Object to append.

Remarks

The appended text starts a new paragraph; a return character separates the existing and appended text.

Note: This method will fail if the FormattedText Object is currently being used by a queued speech or thought action.
