Double Agent API

The FindSREngines Method

Description

Retrieves a collection of Speech Recognition Engine Objects that meet the character's requirements.

Syntax

Visual C++

character->FindSREngines (LanguageID, &SREngines)

Visual C#

[SREngines = ]character.FindSREngines (LanguageID)

Visual Basic .NET

[SREngines = ]character.FindSREngines (LanguageID)

Visual Basic 6

[Set SREngines = ]character.FindSREngines (LanguageID)

VBScript

[Set SREngines = ]character.FindSREngines (LanguageID)
Part Description
LanguageID A numeric value that specifies the language to match, or zero for any language.
SREngines A SREngines Collection that matches the character's text-to-speech requirements and the specified LanguageID.
