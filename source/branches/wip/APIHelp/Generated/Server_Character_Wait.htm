Double Agent API

The Wait Method

Description

Causes the character's animation queue to wait until the specified animation request completes.

Syntax

Visual C++

character->Wait (WaitForRequestID, &RequestID)

Visual C#

character.Wait (WaitForRequestID, out RequestID)

Visual Basic .NET

character.Wait (WaitForRequestID, RequestID)

Visual Basic 6

character.Wait WaitForRequestID[, RequestID]

VBScript

character.Wait WaitForRequestID[, RequestID]
Part Description
WaitForRequestID A numeric value specifying the animation to wait for.
RequestID An optional unique numeric value your application can use to track the method's status.

Remarks

Use this method only when you are using multiple (simultaneous) characters and are trying to sequence the interaction of characters. (For a single character, each animation request is played sequentially--after the previous request completes.)

If you have two characters and you want one character's animation request to wait until the other character's animation completes, use the Wait method with the other character's animation RequestID value.
