The Characters Property

Description

Retrieves the control's Characters Collection.

Syntax

Visual Basic .NET

[Characters = ]control.Characters

Visual Basic 6

[Set Characters = ]control.Characters

VBScript

[Set Characters = ]control.Characters

Visual C#

[Characters = ]control.Characters

Visual C++

control->get_Characters (&Characters)

Part Description
Characters The control's Characters Collection.
