Double Agent API

The SpeechText Property

Description

Returns the text formatted for the Text-to-Speech engine.

Syntax

Visual Basic .NET

[SpeechText = ]formattedtext.SpeechText

Visual Basic 6

[SpeechText = ]formattedtext.SpeechText

VBScript

[SpeechText = ]formattedtext.SpeechText

Visual C#

[SpeechText = ]formattedtext.SpeechText

Visual C++

formattedtext->get_SpeechText (&SpeechText)
Part Description
SpeechText A string value containing the text formatted for sending to the Text-to-Speech engine.

Remarks

This property contains speech tags formatted for the Text-to-Speech engine selected for the Character when the FormattedText Object was created.

For more information about speech tags, see Speech Output Tags.
