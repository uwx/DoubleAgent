Double Agent API

The Command Property

Description

Returns a Command Object in the collection.

Syntax

Visual C++

commands->get_Command (CommandID, &Command)

Visual C#

[Command = ]character.Commands.get_Command (CommandID)

Visual Basic .NET

[Command = ]character.Commands.Command (CommandID)

Visual Basic 6

[Set Command = ]character.Commands.Command (CommandID)

VBScript

[Set Command = ]character.Commands.Command (CommandID)
Part Description
CommandID A numeric value that specifies the CommandID given to the command when it was defined.
Command The Command Object with the specified CommandID.
