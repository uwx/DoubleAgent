Double Agent API

The CtlSize Event

Description

Signalled when a character's size changes.

Syntax

Visual Basic .NET

Event control_CtlSize (ByVal Sender As Object, ByVal E As CtlSizeEvent)

Visual C#

delegate void control_CtlSizeEventHandler (object Sender, CtlSizeEvent E)
Part Description
Sender The source of the event.
E A CtlSizeEvent Object that contains the event data:
CharacterID
The unique identifier of the character that was resized.
Width
The character's new width, in pixels.
Height
The character's new height, in pixels.

Remarks

This event is signalled when an application changes the size of a character. This event is sent only to the clients of the character (applications that have loaded the character).
