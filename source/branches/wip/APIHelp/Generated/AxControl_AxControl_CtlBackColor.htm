Double Agent API

The CtlBackColor Property

Description

Specifies the ActiveX® control's background color.

Syntax

Visual Basic .NET

[CtlBackColor = ]control.CtlBackColor

control.CtlBackColor = CtlBackColor

Visual C#

[CtlBackColor = ]control.CtlBackColor

control.CtlBackColor = CtlBackColor
Part Description
CtlBackColor An System.Drawing.Color value for the control's background color.

Remarks

Note: This property is relevant only when the control is visible (i.e. when it's used in contained mode).
