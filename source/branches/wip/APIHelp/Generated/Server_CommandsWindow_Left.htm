Double Agent API

The Left Property

Description

Returns the horizontal location, in pixels, of the Voice Commands window.

Syntax

Visual C++

commandswindow->get_Left (&Left)

Visual C#

[Left = ]server.CommandsWindow.Left

Visual Basic .NET

[Left = ]server.CommandsWindow.Left

Visual Basic 6

[Left = ]server.CommandsWindow.Left

VBScript

[Left = ]server.CommandsWindow.Left
Part Description
Left A numeric value that specifies the left edge, in pixels, of the Voice Commands window.

Remarks

Double Agent remembers and reuses the window location set by the user.
