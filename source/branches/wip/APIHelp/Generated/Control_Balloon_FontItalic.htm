Double Agent API

The FontItalic Property

Description

Specifies the font style of a character's the word balloon.

Syntax

Visual Basic .NET

[FontItalic = ]control.Characters ("CharacterID").Balloon.FontItalic

control.Characters ("CharacterID").Balloon.FontItalic = FontItalic

Visual Basic 6

[FontItalic = ]control.Characters ("CharacterID").Balloon.FontItalic

control.Characters ("CharacterID").Balloon.FontItalic = FontItalic

VBScript

[FontItalic = ]control.Characters ("CharacterID").Balloon.FontItalic

control.Characters ("CharacterID").Balloon.FontItalic = FontItalic

Visual C#

[FontItalic = ]control.Characters.this ["CharacterID"].Balloon.FontItalic

control.Characters.this ["CharacterID"].Balloon.FontItalic = FontItalic

Visual C++

balloon->get_FontItalic (&FontItalic)

balloon->put_FontItalic (FontItalic)
Part Description
FontItalic A boolean value that indicates if the font is italic.

Remarks

The default values for the font of a character's word balloon are set in the Microsoft Agent Character Editor. In addition, the user can override font settings for all characters in the Advanced Character Options window.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
