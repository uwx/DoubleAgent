Double Agent API

The DefaultCharacterChange Event

Description

Signalled when the user changes the default character.

Syntax

Visual Basic .NET

Event control_DefaultCharacterChange (ByVal CharGUID As String)

Visual Basic 6

Sub control_DefaultCharacterChange (ByVal CharGUID As String)

VBScript

Sub control_DefaultCharacterChange (ByVal CharGUID)

Visual C#

delegate void control_DefaultCharacterChangeEventHandler (string CharGUID)

Visual C++

void control_DefaultCharacterChange (LPCTSTR CharGUID)
Part Description
CharGUID The unique identifier of the new default character.

Remarks

This event indicates that the user has changed the character assigned as the user's default character. Double Agent sends this only to clients that have loaded the default character.

When the new character appears, it assumes the same size as any already loaded instance of the character, or the previous default character (in that order).
