Double Agent API

The Status Property

Description

You can use the Status property to check the status of your request.

Syntax

Visual Basic .NET

[Status = ]request.Status

Visual Basic 6

[Status = ]request.Status

VBScript

[Status = ]request.Status

Visual C#

[Status = ]request.Status

Visual C++

request->get_Status (&Status)

Part Description
Status The current status of the request.
0Request successfully completed
1Request failed
2Request pending (in the queue, but not complete)
3Request interrupted
4Request in progress

Example

Dim MyRequest

Set MyRequest = AgentControl.Characters("Genie").Play ("GetAttention")

If (MyRequest.Status = 2) then
    'do something
Else If (MyRequest.Status = 0) then
    'do something right away
End If
