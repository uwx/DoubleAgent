Double Agent API

The ShowPopupMenu Method

Description

Displays the character's pop-up menu at the specified location.

Syntax

Visual Basic .NET

[Showed = ]control.Characters ("CharacterID").ShowPopupMenu (X, Y)

Visual Basic 6

[Showed = ]control.Characters ("CharacterID").ShowPopupMenu X, Y

VBScript

[Showed = ]control.Characters ("CharacterID").ShowPopupMenu X, Y

Visual C#

[Showed = ]control.Characters.this ["CharacterID"].ShowPopupMenu (X, Y)

Visual C++

character->ShowPopupMenu (X, Y, &Showed)
Part Description
X The horizontal screen coordinate to display the menu, in pixels.
Y The vertical screen coordinate to display the menu, in pixels.
Showed True if the method succeeded.

Remarks

If the AutoPopupMenu property is set to True, Double Agent automatically displays the character's pop-up menu when the user right-clicks the character. If you set AutoPopupMenu to False, you can use ShowPopupMenu to display the menu.

The menu remains displayed until the user selects a command or displays another menu. Only one pop-up menu can be displayed at a time; therefore, calls to this method will cancel (remove) the former menu.

This method should be called only when your client application is the active client of the character; otherwise it fails. To determine the success of this method you can call it as a function and it will return a boolean value indicating whether the method succeeded. 

If Genie.ShowPopupMenu (10,10) = True Then
    ' The menu was displayed
Else
    ' The menu was not displayed
End If
