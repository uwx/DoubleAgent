Double Agent API

The DoubleAgentFiles Property

Description

Indicates if the Double Agent install location should be included in the DefaultSearchPath.

Syntax

Visual Basic .NET

[DoubleAgentFiles = ]control.CharacterFiles.DoubleAgentFiles

control.CharacterFiles.DoubleAgentFiles = DoubleAgentFiles

Visual Basic 6

[DoubleAgentFiles = ]control.CharacterFiles.DoubleAgentFiles

control.CharacterFiles.DoubleAgentFiles = DoubleAgentFiles

VBScript

[DoubleAgentFiles = ]control.CharacterFiles.DoubleAgentFiles

control.CharacterFiles.DoubleAgentFiles = DoubleAgentFiles

Visual C#

[DoubleAgentFiles = ]control.CharacterFiles.DoubleAgentFiles

control.CharacterFiles.DoubleAgentFiles = DoubleAgentFiles

Visual C++

characterfiles->get_DoubleAgentFiles (&DoubleAgentFiles)

characterfiles->put_DoubleAgentFiles (DoubleAgentFiles)
Part Description
DoubleAgentFiles A boolean value that indicates if the Double Agent install location should be included in the search for character files.

Remarks

The path for Double Agent characters is the Chars subdirectory of the Double Agent install location. Typically, this would be C:\Program Files\Double Agent\Chars.
