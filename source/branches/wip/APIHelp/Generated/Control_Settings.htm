Double Agent API

The Settings Object

Description

The Settings Object provides access to various settings maintained by Double Agent. The properties are read-only for client applications, but the user can change them in the Advanced Character Options window.

Properties

AudioStatus Returns the status of the audio output channel.
BalloonEnabled Indicates if the word balloon for the Speak and Think methods is enabled.
BalloonFont Returns the font that will be used for all word balloons, if any.
SoundEffectsEnabled Indicates if sound effects associated with a character's animations will play.
SREnabled Indicates if speech input is enabled.
SRHotKey Returns the user's current setting for the Listening key.
SRHotKeyTime Returns the current timeout delay for the Listening key.
SRListeningPrompt Indicates if a sound is played when Listening mode is activated or deactivated.
SRListeningTip Indicates if the Listening tip is shown when Listening mode is active.
TTSEnabled Indicates if spoken audio output is enabled.
TTSSpeed Returns the text-to-speech rate configured by the user.

Remarks

The Settings Object replaces and extends the AudioOutput and SpeechInput Objects.

This object can be accessed using the Control Object's Settings property.

Note: For some properties, Double Agent returns a relevant value only if a compatible speech engine has been installed and is enabled.
