Double Agent API

The Settings Object

The Settings object provides access to various settings maintained by Double Agent. The properties are read-only for client applications, but the user can change them in the Advanced Character Options window.

This object can be accessed using the Control's Settings property.

  • The Settings object has the following properties:

    AudioStatus, BalloonEnabled, BalloonFont, SoundEffectsEnabled, SREnabled, SRHotKey, SRHotKeyTime, SRListeningPrompt, SRListeningTip, TTSEnabled, TTSSpeed

Remarks

For some properties, Double Agent return a relevant value only if a compatible speech engine has been installed and is enabled.
