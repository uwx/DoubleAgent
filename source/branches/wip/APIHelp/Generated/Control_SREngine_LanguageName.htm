Double Agent API

The LanguageName Property

Description

Returns the speech recognition engine's human-readable primary language name.

Syntax

Visual Basic .NET

[LanguageName = ]srengine.LanguageName[ [EnglishName]]

Visual Basic 6

[LanguageName = ]srengine.LanguageName[ [EnglishName]]

VBScript

[LanguageName = ]srengine.LanguageName[ [EnglishName]]

Visual C#

[LanguageName = ]srengine.get_LanguageName (EnglishName)

Visual C++

srengine->get_LanguageName (EnglishName, &LanguageName)
Part Description
LanguageName A string value that identifies the engine's primary language in a form suitable for the user.
EnglishName An optional boolean value that specifies if the language name should be in English, or in the user's language. The default value is True.
