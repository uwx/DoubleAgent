Double Agent API

The Caption Property

Description

Specifies the text displayed for a Command in the character's pop-up menu.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Commands("Name").Caption[ = String]
Visual Basic 6
control.Characters ("CharacterID").Commands("Name").Caption[ = String]
VBScript
control.Characters ("CharacterID").Commands("Name").Caption[ = String]
Visual C#
[Caption = ]control.Characters ("CharacterID").Commands("Name").Caption control.Characters ("CharacterID").Commands("Name").Caption = Caption
Visual C++

command.get_Caption (&Caption)

command.put_Caption (Caption)

Part Description
Caption A string value that specifies the text displayed in the menu.

Remarks

To specify an access key (underlined mnemonic) for your Caption, include an ampersand (&) before the mnemonic character.

If you don't define a VoiceCaption for your command, the Caption property will be used.
