Double Agent API

The Pitch Property

Description

Returns the character's default speech output (text-to-speech) pitch setting.

Syntax

Visual Basic .NET

[Pitch = ]control.Characters ("CharacterID").Pitch

Visual Basic 6

[Pitch = ]control.Characters ("CharacterID").Pitch

VBScript

[Pitch = ]control.Characters ("CharacterID").Pitch

Visual C#

[Pitch = ]control.Characters.this ["CharacterID"].Pitch

Visual C++

character->get_Pitch (&Pitch)
Part Description
Pitch A numeric value that specifies the pitch (in hertz) of the character's default text-to-speech engine.

Remarks

This property is of limited value since it only supplies the default pitch from the character's definition. It's supported for compatibility.

Although your application cannot write this value, you can include Pit (pitch) tags in your speech output text that will temporarily change the pitch for a particular utterance. However, using the Pit tag to change the pitch will not change the Pitch property value. For further information, see Speech Output Tags.
