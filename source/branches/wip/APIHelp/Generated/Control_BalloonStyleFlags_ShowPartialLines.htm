Double Agent API

The ShowPartialLines Member

Description

Partial lines are shown when the word balloon text scrolls (for a word balloon that's not SizeToText).

Value

&H00000010

Remarks

When the ShowPartialLines style is set, the word balloon will show partial lines at the bottom as it scrolls its text. When it's not set, only full lines are shown. This style only applies when the SizeToText style is not set.
