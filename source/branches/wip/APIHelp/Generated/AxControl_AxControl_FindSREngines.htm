The FindSREngines Method

Description

Retrieves a collection of Speech Recognition Engine Objects for a specific language.

Syntax

Visual Basic .NET

[SREngines = ]control.FindSREngines (LanguageID)

Visual C#

[SREngines = ]control.FindSREngines (LanguageID)
Part Description
LanguageID A numeric value that specifies the language to match, or zero for any language.
SREngines A SREngines Collection that matches the the specified LanguageID.

Remarks

This method allows you to query available speech recognition engines without loading a character.

Note: Using this method with zero for the LanguageID is equivalent to using the SREngines property.
