Double Agent API

The WordIndex Property

Description

Returns the index of the word currently being spoken and/or displayed.

Syntax

Visual C++

formattedtext->get_WordIndex (&WordIndex)

Visual C#

[WordIndex = ]formattedtext.WordIndex

Visual Basic .NET

[WordIndex = ]formattedtext.WordIndex

Visual Basic 6

[WordIndex = ]formattedtext.WordIndex

VBScript

[WordIndex = ]formattedtext.WordIndex
Part Description
WordCount A numeric value that indicates the zero-based index of the current word.

Remarks

Note: If speech or thought output has not yet begun, this property's value will be -1.
