Double Agent API

The GetIconIdentity Method

Description

Retrieves the identity that Windows will use to remember the character's notification icon.

Syntax

Visual C++

character->GetIconIdentity (&IconIdentity)

Visual C#

character.GetIconIdentity (out IconIdentity)

Visual Basic .NET

character.GetIconIdentity (IconIdentity)

Visual Basic 6

character.GetIconIdentity IconIdentity

VBScript

character.GetIconIdentity IconIdentity
Part Description
IconIdentity The GUID value that Windows uses to uniquely identify this character's notification icon.

Remarks

Every notification icon has an identity that allows Windows to remember it's state (shown, hidden, active, etc.). By default, Windows associates a notification icon with the application that creates it.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
