Double Agent API

The StopAll Method

Description

Stops all animation requests or specified types of requests.

Syntax

Visual Basic .NET

control.Characters ("CharacterID").StopAll (Types)

Visual Basic 6

control.Characters ("CharacterID").StopAll [Types]

VBScript

control.Characters ("CharacterID").StopAll [Types]

Visual C#

control.Characters.this ["CharacterID"].StopAll (Types)

Visual C++

character->StopAll (Types)
Part Description
Types You can omit this parameter you or use any combination of the following members of the StopAllFlags Enumeration:
&H00000001 = Play Stop any pending or running animations.
&H00000002 = Move Stop any pending or running move requests.
&H00000004 = Speak Stop any pending or running speech requests.
&H00000008 = QueuedPrepare Stop any queued animation or file downloads (Prepare method with Queue parameter set to True).
&H00000010 = ImmediatePrepare Stop any non-queued animation or file downloads (Prepare method with Queue parameter set to False)
&H00000020 = Visibility Stop any pending show or hide requests.
&HFFFFFFFF = Everything Stop all pending actions.

You can use a string value instead, with one or more of the following values, separated with commas:

"Play"StopAllFlags.Play
"Move"StopAllFlags.Move
"Speak"StopAllFlags.Speak
"QueuedPrepare" (or "Get" for compatibility)StopAllFlags.QueuedPrepare
"ImmediatePrepare" (or "NonQueuedGet" for compatibility)StopAllFlags.ImmediatePrepare
"Visibility" or "Visible"StopAllFlags.Visibility
"Everything" or "All"StopAllFlags.Everything

Remarks

If you don't set the Type parameter, Double Agent stops all animations for the character, including queued and non-queued Prepare requests, and clears its animation queue. It also stops playing a character's Hiding or Showing animation.

This method is performed immediately and does not generate a Request Object.
