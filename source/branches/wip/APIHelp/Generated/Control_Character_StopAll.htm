Double Agent API

The StopAll Method

Description

Stops all animation requests or specified types of requests for a character.

Syntax

Visual Basic .NET

control.Characters ("CharacterID").StopAll (Types)

Visual Basic 6

control.Characters ("CharacterID").StopAll [Types]

VBScript

control.Characters ("CharacterID").StopAll [Types]

Visual C#

control.Characters ("CharacterID").StopAll (Types)

Visual C++

character.StopAll (Types)

Part Description
Types You can omit this parameter you or use any of the following values.
"Get"
To stop all queued Get requests.
"NonQueuedGet"
To stop all non-queued Get requests (Get method with Queue parameter set to False).
"Move"
To stop all queued MoveTo requests.
"Play"
To stop all queued Play requests.
"Speak"
To stop all queued Speak requests.
You can also specify multiple types by separating them with commas.

Remarks

If you don't set the Type parameter, Double Agent stops all animations for the character, including queued and non-queued Prepare requests, and clears its animation queue. It also stops playing a character's Hiding or Showing animation.

This method is performed immediately and does not generate a Request object.
