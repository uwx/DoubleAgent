Double Agent API

The FontSize Property

Description

Returns or sets the font size supported for the word balloon for the specified character.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Balloon.FontSize[ = Integer]
Visual Basic 6
control.Characters ("CharacterID").Balloon.FontSize[ = Integer]
VBScript
control.Characters ("CharacterID").Balloon.FontSize[ = Integer]
Visual C#
control.Characters ("CharacterID").Balloon.FontSize[ = int];
Visual C++

balloon->get_FontSize (&long);

balloon->put_FontSize (long);

Remarks

The FontSize property returns a Long integer value specifying the current font size in points. The maximum value for FontSize is 2160 points.

The default value for the font settings of a character's word balloon are set in the Microsoft Agent Character Editor. In addition, the user can override font settings for all characters in the Double Agent property sheet.
