Double Agent API

The Top Property

Description

Specifies the top edge of the character's image.

Syntax

Visual Basic .NET

[Top = ]control.Characters ("CharacterID").Top

control.Characters ("CharacterID").Top = Top

Visual Basic 6

[Top = ]control.Characters ("CharacterID").Top

control.Characters ("CharacterID").Top = Top

VBScript

[Top = ]control.Characters ("CharacterID").Top

control.Characters ("CharacterID").Top = Top

Visual C#

[Top = ]control.Characters.this ["CharacterID"].Top

control.Characters.this ["CharacterID"].Top = Top

Visual C++

character->get_Top (&Top)

character->put_Top (Top)
Part Description
Top A numeric value that specifies the top left edge of the character's image.

Remarks

The Top property is always expressed in pixels, relative to screen origin (upper left).

Even though the character appears in an irregularly shaped region window, the location of the character is based on the external dimensions of the rectangular animation frame used when the character was compiled with the Microsoft Agent Character Editor.

Note: This property applies to all clients of the character.
