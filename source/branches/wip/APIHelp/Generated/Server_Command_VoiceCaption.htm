Double Agent API

The VoiceCaption Property

Description

Specifies the text displayed for the Command Object in the Voice Commands window.

Syntax

Visual C++

command->get_VoiceCaption (&VoiceCaption)

command->put_VoiceCaption (VoiceCaption)

Visual C#

[VoiceCaption = ]command.VoiceCaption

command.VoiceCaption = VoiceCaption

Visual Basic .NET

[VoiceCaption = ]command.VoiceCaption

command.VoiceCaption = VoiceCaption

Visual Basic 6

[VoiceCaption = ]command.VoiceCaption

command.VoiceCaption = VoiceCaption

VBScript

[VoiceCaption = ]command.VoiceCaption

command.VoiceCaption = VoiceCaption

Remarks

This text will appear in the Voice Commands window when your client application is input-active and the character is visible. If this property is not set, the Caption property is used instead. When neither the VoiceCaption nor Caption property is set, the command does not appear in the Voice Commands window.

If you define a Command Object in a Commands Collection with a non-blank VoiceCaption property, you will typically also set the Command Object's VoiceCaption property.
