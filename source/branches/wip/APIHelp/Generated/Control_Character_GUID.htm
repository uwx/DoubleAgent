Double Agent API

The GUID Property

Description

Returns the unique identifier for the character.

Syntax

Visual Basic .NET

[CharGUID = ]control.Characters ("CharacterID").GUID

Visual Basic 6

[CharGUID = ]control.Characters ("CharacterID").GUID

VBScript

[CharGUID = ]control.Characters ("CharacterID").GUID

Visual C#

[CharGUID = ]control.Characters ("CharacterID").GUID

Visual C++

character.get_GUID (&CharGUID)

Part Description
CharGUID The unique identifier for the character

Remarks

This property returns a string representing the internal identifier that Double Agent uses to refer to uniquely identify the character. A character identifier is set when it is compiled with the Microsoft Agent Character Editor. The property is read-only.
