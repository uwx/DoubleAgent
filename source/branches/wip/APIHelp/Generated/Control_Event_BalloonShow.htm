Double Agent API

The BalloonShow Event

Description

Signalled when a character's word balloon is shown.

Syntax

Visual Basic .NET

Event control_BalloonShow (ByVal CharacterID As String)

Visual Basic 6

Sub control_BalloonShow (ByVal CharacterID As String)

VBScript

Sub control_BalloonShow (ByVal CharacterID)

Visual C#

delegate void control_BalloonShowEventHandler (string CharacterID)

Visual C++

void control_BalloonShow (LPCTSTR CharacterID)

Part Description
CharacterID The unique identifier of the character who's balloon was shown.

Remarks

Double Agent sends this event only to the clients of the character (applications that have loaded the character) that use the word balloon.
