Double Agent API

The FontUnderline Property

Description

Returns or sets the font style of a character's the word balloon.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Balloon.FontUnderline[ = Boolean]
Visual Basic 6
control.Characters ("CharacterID").Balloon.FontUnderline[ = Boolean]
VBScript
control.Characters ("CharacterID").Balloon.FontUnderline[ = Boolean]
Visual C#
control.Characters ("CharacterID").Balloon.FontUnderline[ = bool];
Visual C++

balloon->get_FontUnderline (&VARIANT_BOOL);

balloon->put_FontUnderline (VARIANT_BOOL);

Remarks

The default values for the font settings of a character's word balloon are set in the Microsoft Agent Character Editor. In addition, the user can override font settings for all characters in the Double Agent property sheet.
