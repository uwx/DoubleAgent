Double Agent API

The AxControl Object

Description

The AxControl Object is a managed class that makes the Double Agent control available as a Windows Forms control. It derives from System.Windows.Forms.Control.

Methods

FindCharacterSREngines Retrieves a collection of Speech Recognition Engine Objects that meet a specific character's requirements.
FindCharacterTTSEngines Retrieves a collection of Text-To-Speech Engine Objects that meet a specific character's requirements.
FindSREngines Retrieves a collection of Speech Recognition Engine Objects for a specific language.
FindTTSEngines Retrieves a collection of Text-To-Speech Engine Objects for a specific language and gender.
GetCharacterSREngine Retrieves a Speech Recognition Engine Object thats meet a specific character's requirements.
GetCharacterTTSEngine Retrieves a Text-To-Speech Engine Object that meets a specific character's requirements.
ShowDefaultCharacterProperties Shows the Gallery page of the Advanced Character Options window.

Properties

AutoConnect Specifies if the Double Agent control will automatically connect to the Double Agent server.
CharacterStyle Specifies the initial style to use when a Character is loaded.
Connected Specifies if the control is currently connected to the Double Agent server.
ControlCharacter Specifies the control's contained Character.
RaiseRequestErrors Specifies if an error condition is raised (an exception thrown) for failed requests.

The AxControl Object also includes properties to retrieve other Double Agent objects:

CharacterFiles Returns the control's CharacterFiles Object.
Characters Returns the control's Characters Collection.
CommandsWindow Returns the control's CommandsWindow Object.
PropertySheet Returns the control's PropertySheet Object.
Settings Returns the control's Settings Object.
SREngines Returns a collection of Speech Recognition Engine Objects for all available speech recognition engines.
TTSEngines Returns a collection of Text-To-Speech Engine Objects for all available text-to-speech engines.

The AxControl Object supports the following ActiveX® appearence properties when the control is used in contained mode:

CtlAutoSize Specifies if the ActiveX® control is automatically sized to fit the ControlCharacter.
CtlBackColor Specifies the ActiveX® control's background color.
CtlBorderColor Specifies the ActiveX® control's border color.
CtlBorderStyle Specifies the style of the ActiveX® control's border.
CtlBorderVisible Specifies if the ActiveX® control's has a visible border.
CtlBorderWidth Specifies the width of the ActiveX® control's border.
CtlMousePointer Specifies the pointer to show when the mouse moves over the ActiveX® control.

Events

CtlActivateInput Signalled when a client becomes input-active.
CtlActiveClientChange Signalled when the active client of the character changes.
CtlAgentPropertyChange Signalled when the user changes a property in the Advanced Character Options window.
CtlBalloonHide Signalled when a character's word balloon is hidden.
CtlBalloonShow Signalled when a character's word balloon is shown.
CtlBookmark Signalled when a bookmark defined by your application in a speech text string is reached.
CtlClick Signalled when the user clicks a character, or the character's icon.
CtlCommand Signalled when the user chooses a (client's) command.
CtlDblClick Signalled when the user double-clicks a character.
CtlDeactivateInput Signalled when a client becomes non-input-active.
CtlDefaultCharacterChange Signalled when the user changes the default character.
CtlDragComplete Signalled when the user stops dragging a character.
CtlDragStart Signalled when the user starts dragging a character.
CtlHide Signalled when a character is hidden.
CtlIdleComplete Signalled when Double Agent ends the Idling state of a character.
CtlIdleStart Signalled when Double Agent sets a character to the Idling state.
CtlListenComplete Signalled when Listening mode (speech recognition) has ended.
CtlListenStart Signalled when Listening mode (speech recognition) begins.
CtlMove Signalled when a character has moved.
CtlRequestComplete Signalled when Double Agent completes a queued request.
CtlRequestStart Signalled when Double Agent begins a queued request.
CtlShow Signalled when a character is displayed.
CtlSize Signalled when a character's size changes.
CtlSpeechEnd Signalled when the character stops speaking for the SpeakFormatted method.
CtlSpeechStart Signalled when the character starts speaking for the SpeakFormatted method.
CtlSpeechWord Signalled when the character reaches a new word for the SpeakFormatted method.

Remarks

The AxControl Object implements the same methods, properties, and events as the Control Object.

The ActiveX® appearence property names and the event names are prefixed with "Ctl", to avoid name collisions and to make them stand out. Also, the events receive their event data according to the pattern for Windows Forms controls.
