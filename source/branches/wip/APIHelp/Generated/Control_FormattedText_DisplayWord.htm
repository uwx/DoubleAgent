Double Agent API

The DisplayWord Property

Description

Returns the ParsedText for a specific WordIndex.

Syntax

Visual Basic .NET

[DisplayWord = ]formattedtext.DisplayWord (WordIndex)

Visual Basic 6

[DisplayWord = ]formattedtext.DisplayWord (WordIndex)

VBScript

[DisplayWord = ]formattedtext.DisplayWord (WordIndex)

Visual C#

[DisplayWord = ]formattedtext.DisplayWord [WordIndex]

Visual C++

formattedtext->get_DisplayWord (WordIndex, &DisplayWord)
Part Description
WordIndex A numeric value specifying the zero-based index of the word to return. Must be less than WordCount.
DisplayWord A string value that contains the ParsedText at the specified WordIndex.

Remarks

The returned string includes trailing white-space, but not leading white-space.
