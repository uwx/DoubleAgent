Double Agent API

The CtlMousePointer Property

Description

Specifies the pointer to show when the mouse moves over the ActiveX® control.

Syntax

Visual Basic .NET

[CtlMousePointer = ]control.CtlMousePointer

control.CtlMousePointer = CtlMousePointer

Visual C#

[CtlMousePointer = ]control.CtlMousePointer

control.CtlMousePointer = CtlMousePointer
Part Description
CtlMousePointer A numeric value that specifies the pointer style as follows:
0 = DefaultThe default pointer style (default for the target environment)
1 = ArrowThe arrow-shaped pointer
2 = CrossThe cross-shaped pointer
11 = HourglassThe "wait" pointer
12 = No DropThe pointer that indicates that drag-and-drop is not possible here
13 = Arrow and HourglassThe pointer that indicates that an application is starting
14 = Arrow and QuestionThe pointer used for "What's this?" help
16 = HandThe hand-shaped pointer

Remarks

Note: This property is relevant only when the control is visible (i.e. when it's used in contained mode).
