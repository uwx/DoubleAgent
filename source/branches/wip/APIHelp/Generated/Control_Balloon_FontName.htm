Double Agent API

The FontName Property

Description

Returns or sets the font used in the word balloon for the specified character.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Balloon.FontName[ = String]
Visual Basic 6
control.Characters ("CharacterID").Balloon.FontName[ = String]
VBScript
control.Characters ("CharacterID").Balloon.FontName[ = String]
Visual C#
control.Characters ("CharacterID").Balloon.FontName[ = string];
Visual C++

balloon->get_FontName (&BSTR);

balloon->put_FontName (BSTR);

Remarks

The FontName property defines the font used to display text in the word balloon window as a string. The default value for the font settings of a character's word balloon are set in the Microsoft Agent Character Editor. In addition, the user can override font settings for all characters in the Double Agent property sheet.

This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.

Note:  If you're using a character that you didn't compile, check the FontName and FontCharSet properties for the character to determine whether they are appropriate for your locale. You may need to set these values before using the Speak method to ensure appropriate text display within the word balloon.
