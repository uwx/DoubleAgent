Double Agent API

The DefaultCommand Property

Description

Specifies the default command of the Commands Collection.

Syntax

Visual C++

commands->get_DefaultCommand (&CommandID)

commands->put_DefaultCommand (CommandID)

Visual C#

[CommandID = ]character.Commands.DefaultCommand

character.Commands.DefaultCommand = CommandID

Visual Basic .NET

[CommandID = ]character.Commands.DefaultCommand

character.Commands.DefaultCommand = CommandID

Visual Basic 6

[CommandID = ]character.Commands.DefaultCommand

character.Commands.DefaultCommand = CommandID

VBScript

[CommandID = ]character.Commands.DefaultCommand

character.Commands.DefaultCommand = CommandID
Part Description
CommandID The CommandID of the Command that's shown bold in the pop-up menu (or zero if none).

Remarks

This property allows you to set a Command in your Commands Collection as the default command, rendering it bold. This does not actually change command handling or double-click events.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
