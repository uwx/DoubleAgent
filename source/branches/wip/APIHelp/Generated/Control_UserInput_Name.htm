Double Agent API

The Name Property

Description

Identifies the chosen Command, if chosen from the popup-menu, or the best matching Command, if voice input triggers the Command event.

Syntax

Visual Basic .NET

[Name = ]userinput.Name

Visual Basic 6

[Name = ]userinput.Name

VBScript

[Name = ]userinput.Name

Visual C#

[Name = ]userinput.Name

Visual C++

userinput->get_Name (&Name)

Part Description
Name The name of the chosen Command, or an empty string if the chose command was pre-defined (for example, the Hide command).
