Double Agent API

The FontCharSet Property

Description

Returns or sets the character set for the font used in a character's word balloon.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Balloon.FontCharSet[ = Integer]
Visual Basic 6
control.Characters ("CharacterID").Balloon.FontCharSet[ = Integer]
VBScript
control.Characters ("CharacterID").Balloon.FontCharSet[ = Integer]
Visual C#
control.Characters ("CharacterID").Balloon.FontCharSet[ = int];
Visual C++

balloon->get_FontCharSet (&long);

balloon->put_FontCharSet (long);

Remarks

The default value for the character set of a character's word balloon is set in the Microsoft Agent Character Editor. In addition, the user can override the character-set settings for all characters in the Double Agent property sheet.

This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.

Note:  If you're using a character that you didn't compile, check the FontName and FontCharSet properties for the character to determine whether they are appropriate for your locale. You may need to set these values before using the Speak method to ensure appropriate text display within the word balloon.
