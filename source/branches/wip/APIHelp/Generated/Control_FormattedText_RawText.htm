Double Agent API

The RawText Property

Description

Returns the unformatted text.

Syntax

Visual Basic .NET

[RawText = ]formattedtext.RawText

Visual Basic 6

[RawText = ]formattedtext.RawText

VBScript

[RawText = ]formattedtext.RawText

Visual C#

[RawText = ]formattedtext.RawText

Visual C++

formattedtext->get_RawText (&RawText)
Part Description
RawText A string value containing the unformatted text.

Remarks

After you use the Parse Method, this property will return the text passed in the RawText parameter.

If you use this property to set the RawText value, it does not get parsed. The RawText value will be used for both the ParsedText and the SpeechText.
