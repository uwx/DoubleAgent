Double Agent API

The Interrupt Method

Description

Interrupts an animation for another character.

Syntax

Visual C++

character->Interrupt (InterruptRequestID, &RequestID)

Visual C#

character.Interrupt (InterruptRequestID, out RequestID)

Visual Basic .NET

character.Interrupt (InterruptRequestID, RequestID)

Visual Basic 6

character.Interrupt InterruptRequestID[, RequestID]

VBScript

character.Interrupt InterruptRequestID[, RequestID]
Part Description
InterruptRequestID The RequestID for the animation call you want to interrupt.
RequestID An optional unique numeric value your application can use to track the method's status.

Remarks

You can use this to synchronize animation between characters. For example, if another character is in a looping animation, this method will stop the loop and move to the next animation in the character's queue.

Details

You can only use the Interrupt method to interrupt another character. To stop an animation for a single character, use the Stop method.

Note: The Interrupt method does not flush the character's queue; it halts the specified animation and moves on to the next animation in the character's queue. To halt and flush a character's queue, use that character's StopAll method.

Note: You cannot interrupt a character animation that you didn't start (that was started by another client).
