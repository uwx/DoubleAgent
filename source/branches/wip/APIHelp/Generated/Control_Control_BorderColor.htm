Double Agent API

The BorderColor Property

Description

Specifies the ActiveX® control's border color.

Syntax

Visual Basic .NET

[BorderColor = ]control.BorderColor

control.BorderColor = BorderColor

Visual Basic 6

[BorderColor = ]control.BorderColor

control.BorderColor = BorderColor

VBScript

[BorderColor = ]control.BorderColor

control.BorderColor = BorderColor

Visual C#

[BorderColor = ]control.BorderColor

control.BorderColor = BorderColor

Visual C++

control->get_BorderColor (&BorderColor)

control->put_BorderColor (BorderColor)
Part Description
BorderColor An OLECOLOR value for the control's border color.

Remarks

The valid range for a normal RGB color is 0 to 16,777,215 (&HFFFFFF). The high byte of a number in this range equals 0; the lower 3 bytes, from least to most significant byte, determine the amount of red, green, and blue, respectively. The red, green, and blue components are each represented by a number between 0 and 255 (&HFF).

If the high byte of an OLECOLOR is &H80, the lower 3 bytes contain a system color index.

Note: This property is relevant only when the control is visible (i.e. when it's used in contained mode).
