Double Agent API

The Caption Property

Description

Specifies the text displayed for the Commands object in the character's pop-up menu.

Syntax

Visual Basic .NET

Caption = control.Characters ("CharacterID").Commands.Caption

control.Characters ("CharacterID").Commands.Caption = Caption

Visual Basic 6

Caption = control.Characters ("CharacterID").Commands.Caption

control.Characters ("CharacterID").Commands.Caption = Caption

VBScript

Caption = control.Characters ("CharacterID").Commands.Caption

control.Characters ("CharacterID").Commands.Caption = Caption

Visual C#

Caption = control.Characters ("CharacterID").Commands.Caption

control.Characters ("CharacterID").Commands.Caption = Caption

Visual C++

commands.get_Caption (&Caption)

commands.put_Caption (Caption)

Part Description
Caption A string value that specifies the text displayed in the menu.

Remarks

Setting the Caption property for your Commands collection defines how it will appear on the character's pop-up menu when its Visible property is set to True and your application is not the input-active client. To specify an access key (underlined mnemonic) for your Caption, include an ampersand (&) before the mnemonic character.

If you define Command objects for a Commands collection, you typically also define a Caption for the Commands collection.
