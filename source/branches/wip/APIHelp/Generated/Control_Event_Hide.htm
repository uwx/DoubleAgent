Double Agent API

The Hide Event

Description

Signalled when a character is hidden.

Syntax

Visual Basic .NET

Event control_Hide (ByVal CharacterID As String, ByVal Cause As DoubleAgent.Control.VisibilityCauseType)

Visual Basic 6

Sub control_Hide (ByVal CharacterID As String, ByVal Cause As VisibilityCauseType)

VBScript

Sub control_Hide (ByVal CharacterID, ByVal Cause)

Visual C#

delegate void control_HideEventHandler (string CharacterID, DoubleAgent.Control.VisibilityCauseType Cause)

Visual C++

void control_Hide (LPCTSTR CharacterID, VisibilityCauseType Cause)

Part Description
CharacterID The unique identifier of the character that was hidden.
Cause Indicates what caused the character to be hidden.
1User hid the character by selecting the command on the character's taskbar icon pop-up menu or using speech input.
3Your client application hid the character.
5Another client application hid the character.
7User hid the character by selecting the command on the character's pop-up menu.

Remarks

Double Agent sends this event to all clients of the character. To query the current state of the character, use the Visible property.
