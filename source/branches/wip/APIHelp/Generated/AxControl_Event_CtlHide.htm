Double Agent API

The CtlHide Event

Description

Signalled when a character is hidden.

Syntax

Visual Basic .NET

Event control_CtlHide (ByVal Sender As Object, ByVal E As CtlHideEvent)

Visual C#

delegate void control_CtlHideEventHandler (object Sender, CtlHideEvent E)
Part Description
Sender The source of the event.
E A CtlHideEvent Object that contains the event data:
CharacterID
The unique identifier of the character that was hidden.
Cause
A VisibilityCauseType value that indicates what caused the character to be hidden:
1 = UserHid The user hid the character using the command on the character's taskbar icon pop-up menu or using speech input.
3 = ProgramHid Your application hid the character.
5 = OtherProgramHid Another client application hid the character.

Remarks

Double Agent sends this event to all clients of the character. To query the current state of the character, use the Visible property.
