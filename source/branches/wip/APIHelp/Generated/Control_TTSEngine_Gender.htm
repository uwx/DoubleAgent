Double Agent API

The Gender Property

Description

Returns the text-to-speech engine's gender.

Syntax

Visual Basic .NET

[Gender = ]ttsengine.Gender

Visual Basic 6

[Gender = ]ttsengine.Gender

VBScript

[Gender = ]ttsengine.Gender

Visual C#

[Gender = ]ttsengine.Gender

Visual C++

ttsengine->get_Gender (&Gender)
Part Description
Gender A numeric value that identifies the speech engine's gender:
1 = Female A female voice.
2 = Male A male voice.
