Double Agent API

The FontName Property

Description

Specifies the font used in the word balloon for the specified character.

Syntax

Visual C++

balloon->get_FontName (&FontName)

balloon->put_FontName (FontName)

Visual C#

[FontName = ]character.Balloon.FontName

character.Balloon.FontName = FontName

Visual Basic .NET

[FontName = ]character.Balloon.FontName

character.Balloon.FontName = FontName

Visual Basic 6

[FontName = ]character.Balloon.FontName

character.Balloon.FontName = FontName

VBScript

[FontName = ]character.Balloon.FontName

character.Balloon.FontName = FontName
Part Description
FontName A string value that specifies the font name.

Remarks

The default values for the font of a character's word balloon are set in the Microsoft Agent Character Editor. In addition, the user can override font settings for all characters in the Advanced Character Options window.

Note: If you're using a character that you didn't compile, check the FontName and FontCharSet properties for the character to determine whether they are appropriate for your locale. You may need to set these values before using the Speak method to ensure appropriate text display within the word balloon.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
