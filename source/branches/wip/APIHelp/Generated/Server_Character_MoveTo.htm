Double Agent API

The MoveTo Method

Description

Moves the character to the specified location.

Syntax

Visual C++

character->MoveTo (X, Y, Speed, &RequestID)

Visual C#

character.MoveTo (X, Y, Speed, out RequestID)

Visual Basic .NET

character.MoveTo (X, Y, Speed, RequestID)

Visual Basic 6

character.MoveTo X, Y[, Speed][, RequestID]

VBScript

character.MoveTo X, Y[, Speed][, RequestID]
Part Description
X The horizontal location, in pixels, relative to the left side of the screen.
Y The vertical location, in pixels, relative to the top of the screen.
Speed An optional numeric value specifying how quickly the character's image moves, in milliseconds. The default value is 1000. Specifying zero moves the image without playing an animation.
RequestID An optional unique numeric value your application can use to track the method's status.

Remarks

The location of a character is based on the upper left corner of its image. Double Agent automatically plays the appropriate animation assigned to the Moving states.

Note: The Speed parameter's actual effect may vary based on the computer's processor speed and the priority of other tasks running on the system.
