Double Agent API

The AudioStatus Property

Description

Returns the status of the audio output channel.

Syntax

Visual Basic .NET

[AudioStatus = ]control.Settings.AudioStatus

Visual Basic 6

[AudioStatus = ]control.Settings.AudioStatus

VBScript

[AudioStatus = ]control.Settings.AudioStatus

Visual C#

[AudioStatus = ]control.Settings.AudioStatus

Visual C++

settings->get_AudioStatus (&AudioStatus)
Part Description
AudioStatus A member of the AudioStatusType Enumeration that indicates the current audio output status:
0 = Available Text-to-speech is available.
1 = Disabled Text-to-speech is disabled in the Advanced Character Options window.
3 = UserSpeaking Listening mode is active and the user is currently speaking.
4 = CharacterSpeaking A character is currently speaking.
5 = CharacterListening Listening mode is active.
6 = Error The audio status could not be determined.

Remarks

Your client application can use this property to determine whether it is appropriate to have your character speak or whether it is appropriate to try to turn on Listening mode (using the Listen method).
