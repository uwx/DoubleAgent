Double Agent API

The Count Property

Description

Returns the count of TTSEngine objects in the TTSEngines collection.

Syntax

Visual Basic .NET

[Count = ]ttsengines.Count

Visual Basic 6

[Count = ]ttsengines.Count

VBScript

[Count = ]ttsengines.Count

Visual C#

[Count = ]ttsengines.Count

Visual C++

ttsengines.get_Count (&Count)

Part Description
Count A numeric value that indicates the number of TTSEngine objects in the collection.
