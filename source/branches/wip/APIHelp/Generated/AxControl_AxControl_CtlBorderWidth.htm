Double Agent API

The CtlBorderWidth Property

Description

Specifies the width of the ActiveX® control's border.

Syntax

Visual Basic .NET

[CtlBorderWidth = ]control.CtlBorderWidth

control.CtlBorderWidth = CtlBorderWidth

Visual C#

[CtlBorderWidth = ]control.CtlBorderWidth

control.CtlBorderWidth = CtlBorderWidth
Part Description
CtlBorderWidth An numeric value that specifies the border width, in pixels.

Remarks

Note: This property is relevant only when the control is visible (i.e. when it's used in contained mode).
