Double Agent API

The GetVersion Method

Description

Retrieves the Double Agent Server's version number.

Syntax

Visual C++

server->GetVersion (&MajorVersion, &MinorVersion)

Visual C#

server.GetVersion (out MajorVersion, out MinorVersion)

Visual Basic .NET

server.GetVersion (MajorVersion, MinorVersion)

Visual Basic 6

server.GetVersion MajorVersion, MinorVersion

VBScript

server.GetVersion MajorVersion, MinorVersion
Part Description
MajorVersion A numeric value that indicates the Double Agent server's major version number (e.g. the 1 in 1.0).
MinorVersion A numeric value that indicates the Double Agent server's minor version number (e.g. the 0 in 1.0).
