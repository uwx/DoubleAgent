Double Agent API

The Animations Property

Description

Returns the names of the character's animations.

Syntax

Visual C++

character->get_Animations (&Animations)

Visual C#

[Animations = ]character.Animations

Visual Basic .NET

[Animations = ]character.Animations

Visual Basic 6

[Animations = ]character.Animations

VBScript

[Animations = ]character.Animations
Part Description
Animations An array of string values that name all of the animations the character can play.

Remarks

For .ACF characters, the array contains all the animations defined for the character, not just the ones that have been retrieved with the Prepare method.
