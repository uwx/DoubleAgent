Double Agent API

The BorderWidth Property

Description

Specifies the width of the ActiveX® control's border.

Syntax

Visual Basic .NET

[BorderWidth = ]control.BorderWidth

control.BorderWidth = BorderWidth

Visual Basic 6

[BorderWidth = ]control.BorderWidth

control.BorderWidth = BorderWidth

VBScript

[BorderWidth = ]control.BorderWidth

control.BorderWidth = BorderWidth

Visual C#

[BorderWidth = ]control.BorderWidth

control.BorderWidth = BorderWidth

Visual C++

control->get_BorderWidth (&BorderWidth)

control->put_BorderWidth (BorderWidth)
Part Description
BorderWidth An numeric value that specifies the border width, in pixels.

Remarks

Note: This property is relevant only when the control is visible (i.e. when it's used in contained mode).
