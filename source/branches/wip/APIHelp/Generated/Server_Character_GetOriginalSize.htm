Double Agent API

The GetOriginalSize Method

Description

Retrieves the character's default dimensions.

Syntax

Visual C++

character->GetOriginalSize (&Width, &Height)

Visual C#

character.GetOriginalSize (out Width, out Height)

Visual Basic .NET

character.GetOriginalSize (Width, Height)

Visual Basic 6

character.GetOriginalSize Width, Height

VBScript

character.GetOriginalSize Width, Height
Part Description
Width The character's image width as defined with the Microsoft Agent Character Editor.
Height The character's image height as defined with the Microsoft Agent Character Editor.
