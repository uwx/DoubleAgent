Double Agent API

The TTSEngine Property

Description

Returns the character's current or default Text-To-Speech Engine Object.

Syntax

Visual Basic .NET

[TTSEngine = ]control.Characters ("CharacterID").TTSEngine[ [GetDefault]]

Visual Basic 6

[Set TTSEngine = ]control.Characters ("CharacterID").TTSEngine[ [GetDefault]]

VBScript

[Set TTSEngine = ]control.Characters ("CharacterID").TTSEngine[ [GetDefault]]

Visual C#

[TTSEngine = ]control.Characters.this ["CharacterID"].get_TTSEngine (GetDefault)

Visual C++

character->get_TTSEngine (GetDefault, &TTSEngine)
Part Description
GetDefault An optional boolean value that indicates if you want the character's current or default text-to-speech engine. The default value is False.
TTSEngine A TTSEngine Object that corresponds to the character's current or default TTSModeID.

Remarks

If you change the character's TTSModeID you must re-access this property to get the correct current TTSEngine Object.
