Double Agent API

The PropertySheet Object

The PropertySheet object provides access to the Advanced Character Options window. It allows you to show and hide the window, and retrieve and update the window's location and current page. It can be accessed using the Control's PropertySheet property.

  • The PropertySheet object has the following properties:

    Left, Height, Page, Top, Visible, Width

Remarks

The Advanced Character Options window is shared among all Double Agent clients. Changes to the window's location and visibility should be made with this in mind.

Note: The Advanced Character Options window combines the Microsoft Agent Advanced Character Options and Default Character Properties windows.
