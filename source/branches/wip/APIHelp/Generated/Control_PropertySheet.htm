Double Agent API

The PropertySheet Object

Description

The PropertySheet Object provides access to the Advanced Character Options window. It allows you to show and hide the window, and retrieve and update the window's location and current page.

Properties

Height Returns the current height, in pixels, of the Advanced Character Options window.
Left Specifies the horizontal location, in pixels, of the Advanced Character Options window.
Page Specifies the page displayed in the Advanced Character Options window.
Top Specifies the vertical location, in pixels, of the Advanced Character Options window.
Visible Specifies if the Advanced Character Options window is visible (open).
Width Returns current width, in pixels, of the Advanced Character Options window.

Remarks

The Advanced Character Options window is shared among all Double Agent clients. Changes to the window's location and visibility should be made with this in mind.

This object can be accessed using the Control Object's PropertySheet property.

Note: The Advanced Character Options window combines the Microsoft Agent Advanced Character Options and Default Character Properties windows.
