Double Agent API

The PropertySheet Object

The PropertySheet object provides several properties you can use if you want to manipulate the character relative to the Microsoft Agent property sheet (also known as the Advanced Character Options window).

  • The PropertySheet object has the following properties:

    Height, Left, Page, Top, Visible, Width

Remarks

If you query Left, Top, Width, and Height properties before the property sheet has ever been shown, their values return as zero. Once shown, these properties return the last position and size of the window (relative to your current screen resolution).
