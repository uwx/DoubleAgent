Double Agent API

The ThinkFormatted Method

Description

Displays the specified text in the character's thought word balloon.

Syntax

Visual Basic .NET

[Request = ]control.Characters ("CharacterID").ThinkFormatted (FormattedText)

Visual Basic 6

[Set Request = ]control.Characters ("CharacterID").ThinkFormatted FormattedText

VBScript

[Set Request = ]control.Characters ("CharacterID").ThinkFormatted FormattedText

Visual C#

[Request = ]control.Characters.this ["CharacterID"].ThinkFormatted (FormattedText)

Visual C++

character->ThinkFormatted (FormattedText, &Request)
Part Description
FormattedText A FormattedText Object object containing the text to be displayed.
Request A Request Object your application can use to track the method's status.

Remarks

Calling the Think method with a specific text string is equivalent to:

  1. calling NewFormattedText to get a FormattedText object;
  2. calling the FormattedText object's Parse method with the same text string;
  3. calling ThinkFormatted with the FormattedText object.

Note: This methods does not trigger the SpeechStart, SpeechEnd, and SpeechWord events.
