Double Agent API

The SoundEffectsEnabled Property

Description

Indicates if sound effects associated with a character's animations will play.

Syntax

Visual Basic .NET

[SoundEffectsEnabled = ]control.Settings.SoundEffectsEnabled

Visual Basic 6

[SoundEffectsEnabled = ]control.Settings.SoundEffectsEnabled

VBScript

[SoundEffectsEnabled = ]control.Settings.SoundEffectsEnabled

Visual C#

[SoundEffectsEnabled = ]control.Settings.SoundEffectsEnabled

Visual C++

settings->get_SoundEffectsEnabled (&SoundEffectsEnabled)
Part Description
SoundEffectsEnabled A boolean value that indicates if sound effects are enabled.

Remarks

This property reflects the "Play character sound effects" option on the Output page of the Advanced Character Options window.

Note: The property setting affects all characters and is read-only; only the user can set this property value.
