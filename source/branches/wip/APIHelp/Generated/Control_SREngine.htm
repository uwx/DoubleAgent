The SREngine Object

Description

The SREngine Object provides information about an installed speech recognition engine. You can use it to help choose a character's speech recognition engine.

Methods

GetVersion Returns the speech recognition engine's version.

Properties

DisplayName Returns the speech recognition engine's human-readable name.
LanguageID Returns the speech recognition engine's primary language identifier.
LanguageIDs Returns the speech recognition engine's supported language identifiers.
LanguageName Returns the speech recognition engine's human-readable primary language name.
LanguageNames Returns the speech recognition engine's human-readable supported language names.
Manufacturer Returns the speech recognition engine's manufacturer's name.
SRModeID Returns the unique identifier used to select this speech recognition engine.
