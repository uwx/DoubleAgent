The SREngine Object

The SREngine object provides information about an installed speech recognition engine. You can use it to help choose a character's speech recognition engine.

  • The SREngine object has the following method:

    GetVersion

  • The SREngine object has the following properties:

    DisplayName, LanguageID, LanguageIDs, LanguageName, LanguageNames, Manufacturer, SRModeID
