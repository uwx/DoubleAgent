Double Agent API

The MsOfficeFiles Property

Description

Indicates if the Microsoft Office install location should be included in the DefaultSearchPath.

Syntax

Visual Basic .NET

[MsOfficeFiles = ]control.CharacterFiles.MsOfficeFiles

control.CharacterFiles.MsOfficeFiles = MsOfficeFiles

Visual Basic 6

[MsOfficeFiles = ]control.CharacterFiles.MsOfficeFiles

control.CharacterFiles.MsOfficeFiles = MsOfficeFiles

VBScript

[MsOfficeFiles = ]control.CharacterFiles.MsOfficeFiles

control.CharacterFiles.MsOfficeFiles = MsOfficeFiles

Visual C#

[MsOfficeFiles = ]control.CharacterFiles.MsOfficeFiles

control.CharacterFiles.MsOfficeFiles = MsOfficeFiles

Visual C++

characterfiles.get_MsOfficeFiles (&MsOfficeFiles)

characterfiles.put_MsOfficeFiles (MsOfficeFiles)

Part Description
MsOfficeFiles A boolean value that indicates if the Microsoft Office install location should be included in the search for character files.

Remarks

Typically, the Microsoft Office characters would be in C:\Program Files\Microsoft Office\Office10.

Note: Only older versions of Microsoft Office include Agent character files.
