Double Agent API

The Character Property

Description

Retrieves a Character Object for a loaded character.

Syntax

Visual C++

server->get_Character (CharacterID, &Character)

Visual C#

[Character = ]server.Character (CharacterID)

Visual Basic .NET

[Character = ]server.Character (CharacterID)

Visual Basic 6

[Set Character = ]server.Character CharacterID

VBScript

[Set Character = ]server.Character CharacterID
Part Description
CharacterID The unique numeric identifier assigned when the character was loaded.
Character A Character Object for the specified CharacterID

Remarks

If the character was loaded using the HTTP protocol, you can use the RequestID returned by the Load method to determine if the download is complete before using this method.
