Double Agent API

The NumberOfLines Property

Description

Specifies the number of lines used for a character's word balloon.

Syntax

Visual C++

balloon->get_NumberOfLines (&NumberOfLines)

balloon->put_NumberOfLines (NumberOfLines)

Visual C#

[NumberOfLines = ]character.Balloon.NumberOfLines

character.Balloon.NumberOfLines = NumberOfLines

Visual Basic .NET

[NumberOfLines = ]character.Balloon.NumberOfLines

character.Balloon.NumberOfLines = NumberOfLines

Visual Basic 6

[NumberOfLines = ]character.Balloon.NumberOfLines

character.Balloon.NumberOfLines = NumberOfLines

VBScript

[NumberOfLines = ]character.Balloon.NumberOfLines

character.Balloon.NumberOfLines = NumberOfLines
Part Description
NumberOfLines A numeric value that specifies the number of lines in a fixed-size word balloon.

Remarks

The NumberOfLines property is only used if the word balloon doesn't have the AutoSize style.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
