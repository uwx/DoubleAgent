Double Agent API

The Play Method

Description

Plays the specified animation.

Syntax

Visual C++

character->Play (Animation, &RequestID)

Visual C#

character.Play (Animation, out RequestID)

Visual Basic .NET

character.Play (Animation, RequestID)

Visual Basic 6

character.Play Animation[, RequestID]

VBScript

character.Play Animation[, RequestID]
Part Description
Animation A string value that specifies the name of an animation sequence.
RequestID An optional unique numeric value your application can use to track the method's status.

Remarks

An animation's name is defined when the character is authored. Before playing the specified animation, Double Agent attempts to play the Return animation for the previous animation, if one has been assigned.

The Play method does not make the character visible. If the character is not visible, Double Agent plays the animation invisibly, and sets the Result parameter of the RequestComplete Event.

When accessing a character's animations using a conventional file protocol, you can simply use the Play method specifying the name of the animation. However, if you're using the HTTP protocol to access character animation data, use the Prepare method to load the animation before calling the Play method.
