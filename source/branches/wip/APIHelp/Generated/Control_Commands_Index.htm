Double Agent API

The Index Property

Description

Returns a Command Object in the collection.

Syntax

Visual Basic .NET

[Command = ]control.Characters ("CharacterID").Commands.Index (Index)

Visual Basic 6

[Set Command = ]control.Characters ("CharacterID").Commands.Index (Index)

VBScript

[Set Command = ]control.Characters ("CharacterID").Commands.Index (Index)

Visual C#

[Command = ]control.Characters.this ["CharacterID"].Commands.get_Index (Index)

Visual C++

commands.get_Index (Index, &Command)

Part Description
Index A numeric value that specifies the zero-based index of the item to retrieve.
Character The Command Object at the spedified Index.

Remarks

When you want to use the Commands Collection as an ordered collection, you can use the Index property instead of the Item property.
