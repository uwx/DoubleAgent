Double Agent API

The TextColor Property

Description

Returns the text color displayed in a character's word balloon.

Syntax

Visual Basic .NET

[TextColor = ]control.Characters ("CharacterID").Balloon.TextColor

control.Characters ("CharacterID").Balloon.TextColor = TextColor

Visual Basic 6

[TextColor = ]control.Characters ("CharacterID").Balloon.TextColor

control.Characters ("CharacterID").Balloon.TextColor = TextColor

VBScript

[TextColor = ]control.Characters ("CharacterID").Balloon.TextColor

control.Characters ("CharacterID").Balloon.TextColor = TextColor

Visual C#

[TextColor = ]control.Characters.this ["CharacterID"].Balloon.TextColor

control.Characters.this ["CharacterID"].Balloon.TextColor = TextColor

Visual C++

balloon->get_TextColor (&TextColor)

balloon->put_TextColor (TextColor)
Part Description
TextColor A numeric value that specifies the text color.

Remarks

The valid range for a normal RGB color is 0 to 16,777,215 (&HFFFFFF). The high byte of a number in this range equals 0; the lower 3 bytes, from least to most significant byte, determine the amount of red, green, and blue, respectively. The red, green, and blue components are each represented by a number between 0 and 255 (&HFF).
