The PropertySheet Object Properties

Description

Part Description
Left A numeric value that specifies the left edge, in pixels, of the Advanced Character Options window.
Top A numeric value that specifies the left edge, in pixels, of the Advanced Character Options window.
Width A numeric value that indicates the width, in pixels, of the Advanced Character Options window.
Height A numeric value that indicates the height, in pixels, of the Advanced Character Options window.
Page A string value indicating the visible page.
"Output" Selects the Output page.
"Speech" Selects the Speech Input page.
"Character" Selects the Gallery (default character selection) page.
"Copyright" Selects the Copyright page.
Visible A boolean value indicating whether or not the Advanced Character Options is visible.

Remarks

The Advanced Character Options window is shared among all Double Agent clients. Changes to the window's location and visibility should be made with this in mind.
