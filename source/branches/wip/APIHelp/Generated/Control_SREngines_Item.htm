Double Agent API

The Item Property

Description

Returns a SREngine Object in the collection.

Syntax

Visual Basic .NET

[SREngine = ]srengines (Index)

Visual Basic 6

[Set SREngine = ]srengines (Index)

VBScript

[Set SREngine = ]srengines (Index)

Visual C#

[SREngine = ]srengines.this [Index]

Visual C++

srengines.get_Item (Index, &SREngine)

Part Description
Index A numeric value that specifies the zero-based index of the item to retrieve,
or a string value that specifies the SRModeID of the item to retrieve.
SREngine The SREngine Object identified by Index.

Remarks

When the SREngines Collection is retrieved using matching criteria (for example using the FindSREngines method), it is an ordered collection. It's items are in order from best to worst match.
