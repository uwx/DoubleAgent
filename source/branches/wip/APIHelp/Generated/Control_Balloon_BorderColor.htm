Double Agent API

The BorderColor Property

Description

Specifies the border color of a character's word balloon.

Syntax

Visual Basic .NET

[BorderColor = ]control.Characters ("CharacterID").Balloon.BorderColor

control.Characters ("CharacterID").Balloon.BorderColor = BorderColor

Visual Basic 6

[BorderColor = ]control.Characters ("CharacterID").Balloon.BorderColor

control.Characters ("CharacterID").Balloon.BorderColor = BorderColor

VBScript

[BorderColor = ]control.Characters ("CharacterID").Balloon.BorderColor

control.Characters ("CharacterID").Balloon.BorderColor = BorderColor

Visual C#

[BorderColor = ]control.Characters.this ["CharacterID"].Balloon.BorderColor

control.Characters.this ["CharacterID"].Balloon.BorderColor = BorderColor

Visual C++

balloon->get_BorderColor (&BorderColor)

balloon->put_BorderColor (BorderColor)
Part Description
BorderColor A numeric value that specifies the outline color.

Remarks

The valid range for a normal RGB color is 0 to 16,777,215 (&HFFFFFF). The high byte of a number in this range equals 0; the lower 3 bytes, from least to most significant byte, determine the amount of red, green, and blue, respectively. The red, green, and blue components are each represented by a number between 0 and 255 (&HFF).

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
