Double Agent API

The SREngine Property

Description

Returns the character's current or default Speech Recognition Engine Object.

Syntax

Visual Basic .NET

[SREngine = ]control.Characters ("CharacterID").SREngine[ [GetDefault]]

Visual Basic 6

[Set SREngine = ]control.Characters ("CharacterID").SREngine[ [GetDefault]]

VBScript

[Set SREngine = ]control.Characters ("CharacterID").SREngine[ [GetDefault]]

Visual C#

[SREngine = ]control.Characters.this ["CharacterID"].get_SREngine (GetDefault)

Visual C++

character->get_SREngine (GetDefault, &SREngine)
Part Description
GetDefault An optional boolean value that indicates if you want the character's current or default speech recognition engine. The default value is False.
SREngine A SREngine Object that corresponds to the character's current or default SRModeID.

Remarks

If you change the character's SRModeID you must re-access this property to get the correct current SREngine Object.
