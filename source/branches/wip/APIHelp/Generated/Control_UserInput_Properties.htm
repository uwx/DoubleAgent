The UserInput Object Properties

Description

The UserInput object provides properties that identify a Command chosen from a character's pop-up menu or by spoken input.

Part Description
CharacterID Identifies the Character whose Command was chosen.
Count The number of possible matching commands, from 0 to 3.
Name The name of the chosen Command, or an empty string if the chose command was pre-defined (for example, the Hide command).
Confidence The relative confidence scoring of best match, from -100 to 100 (100 if the Command was chosen from the pop-up menu)
Voice The best matching voice text (or an empty string if the Command was chosen from the popup-menu)
Alt1Name The name of the second-best matching Command.
Alt1Confidence The relative confidence scoring of second-best match, from -100 to 100 (0 if the Command was chosen from the pop-up menu)
Alt1Voice The second-best matching voice text
Alt2Name The name of the third-best matching Command.
Alt2Confidence The relative confidence scoring of third-best match, from -100 to 100 (0 if the Command was chosen from the pop-up menu)
Alt2Voice The third-best matching voice text

Remarks

Not all speech recognition engines may return all the values for all the properties of this object. Check with your engine vendor to determine whether the engine supports the Microsoft Speech API interface for returning alternatives and confidence scores.
