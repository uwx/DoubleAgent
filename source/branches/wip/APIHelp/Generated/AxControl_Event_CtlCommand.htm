Double Agent API

The CtlCommand Event

Description

Signalled when the user chooses a (client's) command.

Syntax

Visual Basic .NET

Event control_CtlCommand (ByVal Sender As Object, ByVal E As DoubleAgent.AxControl.CtlCommandEvent)

Visual C#

delegate void control_CtlCommandEventHandler (object Sender, DoubleAgent.AxControl.CtlCommandEvent E)
Part Description
Sender The source of the event.
E A CtlCommandEvent Object that contains the event data:
UserInput
A UserInput Object that identifies the chosen command.

Remarks

Double Agent sends this event when your application is input-active and the user chooses a command from the character's pop-up menu, or by spoken input.
