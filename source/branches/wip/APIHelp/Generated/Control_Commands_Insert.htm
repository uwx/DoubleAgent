Double Agent API

The Insert Method

Description

Inserts a Command object in the Commands collection.

Syntax

Visual Basic .NET
[Command = ]control.Characters ("CharacterID").Commands.Insert ("Name", "RefName", Before, Caption, VoiceGrammar, Enabled, Visible)
Visual Basic 6
[Command = ]control.Characters ("CharacterID").Commands.Insert "Name", "RefName", Before, Caption, VoiceGrammar, Enabled, Visible
VBScript
[Command = ]control.Characters ("CharacterID").Commands.Insert "Name", "RefName", Before, Caption, VoiceGrammar, Enabled, Visible
Visual C#
[Command = ]control.Characters.this ["CharacterID"].Commands.Insert ("Name", "RefName", Before, Caption, VoiceGrammar, Enabled, Visible)
Visual C++
commands.Insert (Name, RefName, Before, Caption, VoiceGrammar, Enabled, Visible, &Command)
Part Description
Name A string value that uniquely identifies the command.
RefName A string value that specifies the name of the command just above or below where you want to insert the new command.
Before A boolean value indicating whether to insert the new command before the command specified by RefName.
Caption A string value specifying the name that will appear in the character's pop-up menu and in the CommandsWindow when the client application is input-active. For more information, see the Command object's Caption property.
VoiceGrammar A string value specifying the words or phrase used by the speech engine for recognizing this command. For more information on formatting alternatives for the string, see the Command object's VoiceGrammar property.
Enabled A boolean value indicating whether the command is enabled. The default value is True. For more information, see the Command object's Enabled property.
Visible A boolean value indicating whether the command is visible in the character's pop-up menu when the client application is input-active. The default value is True. For more information, see the Command object's Visible property.
Command The Command object inserted, if successful.

Remarks

The value of a Command object's Name property must be unique within its Commands collection. You must remove a Command before you can create a new Command with the same Name property setting. Attempting to create a Command with a Name property that already exists raises an error.

This method returns a Command object. This allows you to declare an object and assign a Command to it when you call the Insert method. 

Dim Command2 as DaControl.DaCtlCommand
Set Command2 = Genie.Commands.Insert ("my second command", "my first command",_ True, "Test", "Test", True, True)
Command2.VoiceCaption = "this is a test"
