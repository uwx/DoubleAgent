Double Agent API

The CtlMoveEvent Object

Description

Event arguments for the CtlMove Event.

Members

CharacterID
The unique identifier of the character that was moved.
X
The character's new horizontal location, in pixels, relative to the left side of the screen.
Y
The character's new vertical location, in pixels, relative to the top of the screen.
Cause
The reason the character moved:
1 = UserMoved The user dragged the character.
2 = ProgramMoved Your client application moved the character.
3 = OtherProgramMoved Another client application moved the character.
