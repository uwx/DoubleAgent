Double Agent API

The NonSpeakingCharacters Property

Description

Indicates if the FilePaths property should include characters that are not designed to speak.

Syntax

Visual Basic .NET

[NonSpeakingCharacters = ]control.CharacterFiles.NonSpeakingCharacters

control.CharacterFiles.NonSpeakingCharacters = NonSpeakingCharacters

Visual Basic 6

[NonSpeakingCharacters = ]control.CharacterFiles.NonSpeakingCharacters

control.CharacterFiles.NonSpeakingCharacters = NonSpeakingCharacters

VBScript

[NonSpeakingCharacters = ]control.CharacterFiles.NonSpeakingCharacters

control.CharacterFiles.NonSpeakingCharacters = NonSpeakingCharacters

Visual C#

[NonSpeakingCharacters = ]control.CharacterFiles.NonSpeakingCharacters

control.CharacterFiles.NonSpeakingCharacters = NonSpeakingCharacters

Visual C++

characterfiles.get_NonSpeakingCharacters (&NonSpeakingCharacters)

characterfiles.put_NonSpeakingCharacters (NonSpeakingCharacters)

Part Description
NonSpeakingCharacters A boolean value that indicates if non-speaking characters should be included.

Remarks

Unlike Microsoft Agent, Double Agents supports the Speak method for characters that are not designed to speak. However, there won't be any mouth animations because they aren't included in the character design.
