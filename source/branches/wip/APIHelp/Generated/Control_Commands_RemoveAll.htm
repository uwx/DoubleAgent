Double Agent API

The RemoveAll Method

Description

Removes all Command Objects from the Commands Collection.

Syntax

Visual Basic .NET

control.Characters ("CharacterID").Commands.RemoveAll ()

Visual Basic 6

control.Characters ("CharacterID").Commands.RemoveAll

VBScript

control.Characters ("CharacterID").Commands.RemoveAll

Visual C#

control.Characters.this ["CharacterID"].Commands.RemoveAll ()

Visual C++

commands->RemoveAll ()

Remarks

An empty Commands Collection no longer appears when the character's Pop-up Menu is displayed, nor in the Voice Commands window when your client application is input-active.
