Double Agent API

The RemoveAll Method

Description

Removes all Command objects from the Commands collection.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Commands.RemoveAll ()
Visual Basic 6
control.Characters ("CharacterID").Commands.RemoveAll
VBScript
control.Characters ("CharacterID").Commands.RemoveAll
Visual C#
control.Characters ("CharacterID").Commands.RemoveAll ()
Visual C++
commands.RemoveAll ()

Remarks

An empty Commands collection no longer appears when the character's pop-up menu is displayed, nor in the CommandsWindow when your client application is input-active.
