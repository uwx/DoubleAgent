Double Agent API

The CtlBorderStyle Property

Description

Specifies the style of the ActiveX® control's border.

Syntax

Visual Basic .NET

[CtlBorderStyle = ]control.CtlBorderStyle

control.CtlBorderStyle = CtlBorderStyle

Visual C#

[CtlBorderStyle = ]control.CtlBorderStyle

control.CtlBorderStyle = CtlBorderStyle
Part Description
CtlBorderStyle A numeric value that specifies the border style as follows:
0 = DefaultThe default border style (default for the target environment)
1 = FixedSingle A fixed-width single line of width CtlBorderWidth and color CtlBorderColor

Remarks

Note: This property is relevant only when the control is visible (i.e. when it's used in contained mode).
