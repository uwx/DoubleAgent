Double Agent API

The CtlSpeechWord Event

Description

Signalled when the character reaches a new word for the SpeakFormatted method.

Syntax

Visual Basic .NET

Event control_CtlSpeechWord (ByVal Sender As Object, ByVal E As CtlSpeechWordEvent)

Visual C#

delegate void control_CtlSpeechWordEventHandler (object Sender, CtlSpeechWordEvent E)
Part Description
Sender The source of the event.
E A CtlSpeechWordEvent Object that contains the event data:
CharacterID
The unique identifier of the speaking character.
FormattedText
The FormattedText that's being spoken.
WordIndex
A numeric value that identifies the word that is currently being spoken.

Remarks

Because events are signalled asynchronously, the synchronization between the speech output and the receipt of this event cannot be exact. However, WordIndex will generally identify the next word to be spoken.

Note: The FormattedText parameter may not contain the same reference pointer as the FormattedText parameter passed to the SpeakFormatted method, but the underlying data is the same.
