Double Agent API

The SpeechEnd Event

Description

Signalled when the character stops speaking for the SpeakFormatted method.

Syntax

Part Description
CharacterID The unique identifier of the speaking character.
FormattedText The FormattedText that's being spoken.
Stopped A boolean value that indicates if the speech was stopped before completion.

Remarks

Note: The FormattedText parameter may not contain the same reference pointer as the FormattedText parameter passed to the SpeakFormatted method, but the underlying data is the same.
