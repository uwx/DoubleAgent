Double Agent API

The TTSModeID Property

Description

Specifies the text-to-speech engine used for the character.

Syntax

Visual C++

character->get_TTSModeID (&TTSModeID)

character->put_TTSModeID (TTSModeID)

Visual C#

[TTSModeID = ]character.TTSModeID

character.TTSModeID = "TTSModeID"

Visual Basic .NET

[TTSModeID = ]character.TTSModeID

character.TTSModeID = "TTSModeID"

Visual Basic 6

[TTSModeID = ]character.TTSModeID

character.TTSModeID = "TTSModeID"

VBScript

[TTSModeID = ]character.TTSModeID

character.TTSModeID = "TTSModeID"
Part Description
TTSModeID A string value that uniquely identifies a text-to-speech engine and synthesis mode.

Remarks

This property determines the TTS (text-to-speech) engine for a character's spoken output. TheTTSModeID for a TTS engine is a formatted string defined by the vendor that uniquely identifies the engine and it's synthesis mode.

Details

Setting this property overrides Double Agent's attempt to load an engine based on the character's compiled TTS setting and the character's current LanguageID setting. However, if you specify a mode ID for an engine that isn't installed or if the user has disabled speech output in the Advanced Character Options window, Double Agent raises an error.

If you do not (or have not successfully) set the character's TTSModeID, Double Agent checks to see if the character's compiled TTS mode setting matches the character's LanguageID setting, and if the associated TTS engine is installed. If so, the character's TTSModeID is initialized accordingly. If not, Double Agent looks for a compatible SAPI speech engine that matches the character's LanguageID and gender. If you have not set the character's LanguageID, the current user language is used. If no matching engine can be found, this property returns an empty string. Similarly, if you query this property when the user has disabled speech output in the Advanced Character Options window, the value will be an empty string.

Querying or setting this property will load the associated engine (if it is not already loaded). However, if the engine specified in the character's compiled TTS setting is installed and matches the character's LanguageID setting, the engine will be loaded when the character loads.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.

Note: Double Agent's speech engine requirements are based on the Microsoft Speech API. Engines supporting Microsoft Agent's SAPI requirements can be installed and used with Double Agent.

Note: This property also returns the empty string if you have no compatible sound support installed on your system.

Note: Querying this property does not typically return an error. However, if the speech engine takes an abnormally long time to load, you may get an error indicating that the query timed out.
