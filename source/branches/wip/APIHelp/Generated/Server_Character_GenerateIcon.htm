Double Agent API

The GenerateIcon Method

Description

Generates a notification icon from one frame of the character's animations.

Syntax

Visual C++

character->GenerateIcon (ClipLeft, ClipTop, ClipWidth, ClipHeight)

Visual C#

character.GenerateIcon (ClipLeft, ClipTop, ClipWidth, ClipHeight)

Visual Basic .NET

character.GenerateIcon (ClipLeft, ClipTop, ClipWidth, ClipHeight)

Visual Basic 6

character.GenerateIcon [ClipLeft][, ClipTop][, ClipWidth][, ClipHeight]

VBScript

character.GenerateIcon [ClipLeft][, ClipTop][, ClipWidth][, ClipHeight]
Part Description
ClipLeft An optional numeric value that specifies the left side of the clipping area. The default value is 0.
ClipTop An optional numeric value that specifies the top of the clipping area. The default value is 0.
ClipWidth An optional numeric value that specifies the width of the clipping area. The default value is -1, meaning the full width of the animation frame.
ClipHeight An optional numeric value that specifies the height of the clipping area. The default value is -1, meaning the full width of the animation frame.

Remarks

The icon is generated using the first frame of one of the character's animations as the source of the icon image. Which animation is used depends on which animations the character supports.

You can specify an optional clipping area to generate the icon from only a part of the animation frame.

If you don't specify a clipping area, the entire image is used (with any empty space around the edges removed).

Note: You can only use this method once per character. Once a notification icon has been generated, it cannot be changed.

Note: If you use this method for a character that has a notification icon in its definition, the defined icon is replaced by the generated icon for as long as the character is loaded.
