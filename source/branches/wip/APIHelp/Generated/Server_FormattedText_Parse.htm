Double Agent API

The Parse Method

Description

Parses text containing speech tags.

Syntax

Visual C++

formattedtext->Parse (RawText)

Visual C#

formattedtext.Parse (RawText)

Visual Basic .NET

formattedtext.Parse RawText

Visual Basic 6

formattedtext.Parse RawText

VBScript

formattedtext.Parse RawText
Part Description
RawText A string value containing text with optional speech output tags.

Remarks

The kind of speech tags accepted depends on the Text-to-Speech engine selected for the Character when the FormattedText Object was created. If the Character is using a SAPI4 speech engine, then SAPI4 speech tags are accepted. If the Character is using a SAPI5 speech engine, then SAPI5 speech tags and supported SAPI4 speech tags are accepted. Any supported SAPI4 tags are converted to SAPI5 tags.

For more information about speech tags, see Speech Output Tags.

Note: This method will fail if the FormattedText Object is currently being used by a queued speech or thought action.
