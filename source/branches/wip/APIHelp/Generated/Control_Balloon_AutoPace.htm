Double Agent API

The AutoPace Property

Description

Indicates if the word balloon paces its text display to the speech output.

Syntax

Visual Basic .NET

[AutoPace = ]control.Characters ("CharacterID").Balloon.AutoPace

control.Characters ("CharacterID").Balloon.AutoPace = AutoPace

Visual Basic 6

[AutoPace = ]control.Characters ("CharacterID").Balloon.AutoPace

control.Characters ("CharacterID").Balloon.AutoPace = AutoPace

VBScript

[AutoPace = ]control.Characters ("CharacterID").Balloon.AutoPace

control.Characters ("CharacterID").Balloon.AutoPace = AutoPace

Visual C#

[AutoPace = ]control.Characters.this ["CharacterID"].Balloon.AutoPace

control.Characters.this ["CharacterID"].Balloon.AutoPace = AutoPace

Visual C++

balloon->get_AutoPace (&AutoPace)

balloon->put_AutoPace (AutoPace)
Part Description
AutoPace A boolean value that indicates if the AutoPace style is enabled.

Remarks

When the AutoPace style is set, the word balloon paces its text output based on the current output rate, for example one word at a time. When output exceeds the size of the word balloon, the text is automatically scrolled. When it's not set, all text included in a Speak or Think request is displayed at once.
