Double Agent API

The Page Property

Description

Specifies the page displayed in the Advanced Character Options window.

Syntax

Visual Basic .NET

[Page = ]control.PropertySheet.Page

control.PropertySheet.Page = Page

Visual Basic 6

[Page = ]control.PropertySheet.Page

control.PropertySheet.Page = Page

VBScript

[Page = ]control.PropertySheet.Page

control.PropertySheet.Page = Page

Visual C#

[Page = ]control.PropertySheet.Page

control.PropertySheet.Page = Page

Visual C++

propertysheet->get_Page (&Page)

propertysheet->put_Page (Page)
Part Description
Page A string value indicating the visible page.
"Output" Selects the Output page.
"Speech" Selects the Speech Input page.
"Character" Selects the Gallery (default character selection) page.
"Copyright" Selects the Copyright page.

Remarks

If no speech engine is installed, setting Page to "Speech" has no effect. Also, the window's Visible property must be set to True for the user to see the page.

Note: When the user closes the window, this property is set to the last page shown.
