Double Agent API

The Page Property

Description

Returns or sets the page displayed in the Double Agent property sheet window.

Syntax

Visual Basic .NET

[Page = ]control.PropertySheet.Page

control.PropertySheet.Page = Page

Visual Basic 6

[Page = ]control.PropertySheet.Page

control.PropertySheet.Page = Page

VBScript

[Page = ]control.PropertySheet.Page

control.PropertySheet.Page = Page

Visual C#

[Page = ]control.PropertySheet.Page

control.PropertySheet.Page = Page

Visual C++

propertysheet->get_Page (&Page)

propertysheet->put_Page (Page)

Part Description
Page

Remarks

If no speech engine is installed, setting Page to "Speech" has no effect. Also, the window's Visible property must be set to True for the user to see the page.

Note: The user can override this property.
