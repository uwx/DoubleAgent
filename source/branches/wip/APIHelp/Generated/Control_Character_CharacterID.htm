Double Agent API

The CharacterID Property

Description

The unique string that identifies the Character Object in the Characters Collection.

Syntax

Visual Basic .NET

[CharacterID = ]character.CharacterID

Visual Basic 6

[CharacterID = ]character.CharacterID

VBScript

[CharacterID = ]character.CharacterID

Visual C#

[CharacterID = ]character.CharacterID

Visual C++

character.get_CharacterID (&CharacterID)

Part Description
CharacterID A string value that identifies the Character Object.

Remarks

When you keep a reference to a Character Object object, you can use this property to relate it back to it's owner Characters Collection.
