Double Agent API

The DefaultFileName Property

Description

Returns the name of the default character's definition file.

Syntax

Visual Basic .NET

[DefaultFileName = ]control.CharacterFiles.DefaultFileName

Visual Basic 6

[DefaultFileName = ]control.CharacterFiles.DefaultFileName

VBScript

[DefaultFileName = ]control.CharacterFiles.DefaultFileName

Visual C#

[DefaultFileName = ]control.CharacterFiles.DefaultFileName

Visual C++

characterfiles->get_DefaultFileName (&DefaultFileName)
Part Description
DefaultFileName A string value that contains the name of the default character's definition file.

Remarks

The default character is specified by the user in the Advanced Character Options window.
