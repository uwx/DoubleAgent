Double Agent API

The FindCharacterTTSEngines Method

Description

Retrieves a collection of Text-To-Speech Engine Objects that meet a specific character's requirements.

Syntax

Visual Basic .NET

[TTSEngines = ]control.FindCharacterTTSEngines (Provider, LanguageID)

Visual C#

[TTSEngines = ]control.FindCharacterTTSEngines (Provider, LanguageID)
Part Description
Provider A string value that specifies the character file, or an empty string for the default character.
LanguageID An optional numeric value that specifies the language to match, or zero to match any language.
TTSEngines A TTSEngines Collection that matches the character's text-to-speech requirements and the specified LanguageID (if any).

Remarks

This method allows you to choose a text-to-speech engine without loading the character.

For more information on specifying character file names, see the Load method.

Note: This method does not support the HTTP protocol for character files.
