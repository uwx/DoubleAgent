Double Agent API

The HasIcon Property

Description

Indicates if the character has a notification icon in its definition.

Syntax

Visual C++

character->get_HasIcon (&HasIcon)

Visual C#

[HasIcon = ]character.HasIcon

Visual Basic .NET

[HasIcon = ]character.HasIcon

Visual Basic 6

[HasIcon = ]character.HasIcon

VBScript

[HasIcon = ]character.HasIcon
Part Description
HasIcon A boolean value that indicates if the character definition includes a notification icon.

Remarks

If the character definition does not include an icon (or even if it does) you can use theGenerateIcon method to create an icon from the character's animations.
