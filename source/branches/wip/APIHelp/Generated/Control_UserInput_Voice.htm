Double Agent API

The Voice Property

Description

Returns the voice text the speech recognition engine matched for the best matching Command.

Syntax

Visual Basic .NET

[Name = ]userinput.Voice

Visual Basic 6

[Name = ]userinput.Voice

VBScript

[Name = ]userinput.Voice

Visual C#

[Name = ]userinput.Voice

Visual C++

userinput->get_Voice (&Voice)
Part Description
Voice The best matching voice text (or an empty string if the Command was chosen from the pop-up menu)

Remarks

Even if no command was matched, the text that the speech engine heard is returned.
