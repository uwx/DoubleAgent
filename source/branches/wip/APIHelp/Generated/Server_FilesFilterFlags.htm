Double Agent API

The FilesFilterFlags Enumeration

Description

Specifies the filter that is applied to the CharacterFiles.FilePaths property.

Members

0x00000001 = PathDoubleAgent Indicates if the Double Agent install location should be included in the DefaultSearchPath.
0x00000002 = PathMsAgent Indicates if the Microsoft Agent install location should be included in the DefaultSearchPath.
0x00000004 = PathMsOffice Indicates if the Microsoft Office install location should be included in the DefaultSearchPath.
0x00000010 = ExcludeNonSpeaking Indicates if the FilePaths property should exclude characters that are not designed to speak.
0x00000020 = ExcludeSpeaking Indicates if the FilePaths property should exclude characters that are designed to speak.
0x00010000 = NoValidateVersion Indicates if the the FilePaths property should include all character files, regardless of version.
