Double Agent API

The Manufacturer Property

Description

Returns the speech recognition engine's manufacturer's name.

Syntax

Visual Basic .NET

[Manufacturer = ]srengine.Manufacturer

Visual Basic 6

[Manufacturer = ]srengine.Manufacturer

VBScript

[Manufacturer = ]srengine.Manufacturer

Visual C#

[Manufacturer = ]srengine.Manufacturer

Visual C++

srengine->get_Manufacturer (&Manufacturer)
Part Description
Manufacturer A string value that supplies the name of the engine's manufacturer.

Remarks

Some speech recognition engines may not supply this property.
