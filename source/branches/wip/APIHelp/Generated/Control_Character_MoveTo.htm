Double Agent API

The MoveTo Method

Description

Moves a character to the specified location.

Syntax

Visual Basic .NET

[Request = ]control.Characters ("CharacterID").MoveTo (X, Y, Speed)

Visual Basic 6

[Set Request = ]control.Characters ("CharacterID").MoveTo X, Y[, Speed]

VBScript

[Set Request = ]control.Characters ("CharacterID").MoveTo X, Y[, Speed]

Visual C#

[Request = ]control.Characters ("CharacterID").MoveTo (X, Y, Speed)

Visual C++

character.MoveTo (X, Y, Speed, &Request)

Part Description
X The horizontal location, in pixels, relative to the left side of the screen.
Y The vertical location, in pixels, relative to the top of the screen.
Speed An optional numeric value specifying how quickly the character's image moves, in milliseconds. The default value is 1000. Specifying zero moves the image without playing an animation.
Request A Request object your application can used to track the method's status.

Remarks

The location of a character is based on the upper left corner of its image. Double Agent automatically plays the appropriate animation assigned to the Moving states.

Note:  If you call MoveTo with a non-zero Speed before the character is shown, it will return a failure status if you assigned a Request object, because the non-zero value indicates that you're attempting to play an animation when the character isn't visible.

Note:  The Speed parameter's actual effect may vary based on the computer's processor speed and the priority of other tasks running on the system.
