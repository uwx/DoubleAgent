Double Agent API

The CtlIdleComplete Event

Description

Signalled when Double Agent ends the Idling state of a character.

Syntax

Visual Basic .NET

Event control_CtlIdleComplete (ByVal Sender As Object, ByVal E As CtlIdleCompleteEvent)

Visual C#

delegate void control_CtlIdleCompleteEventHandler (object Sender, CtlIdleCompleteEvent E)
Part Description
Sender The source of the event.
E A CtlIdleCompleteEvent Object that contains the event data:
CharacterID
The unique identifier of the Idling character.

Remarks

Double Agent sends this event to all clients of the character.
