Double Agent API

The DefaultSearchPath Property

Description

Identifies the default search path used to fill the FilePaths property.

Syntax

Visual C++

characterfiles->get_DefaultSearchPath (&DefaultSearchPath)

Visual C#

[DefaultSearchPath = ]server.CharacterFiles.DefaultSearchPath

Visual Basic .NET

[DefaultSearchPath = ]server.CharacterFiles.DefaultSearchPath

Visual Basic 6

[DefaultSearchPath = ]server.CharacterFiles.DefaultSearchPath

VBScript

[DefaultSearchPath = ]server.CharacterFiles.DefaultSearchPath
Part Description
DefaultSearchPath A string value that identifies one or more UNC paths, separated by semicolons.

Remarks

This property returns the default search path, based on the selection criteria specified by the Filter property. The actual search path can be set with the SearchPath property.
