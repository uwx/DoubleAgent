Double Agent API

The CommandsWindow Object

Description

The CommandsWindow Object provides access to Double Agent's Voice Commands window. It allows you to show and hide the window, and retrieve the window's location and size.

Methods

GetPosition Returns the current position of the Voice Commands window.
GetSize Returns the current size of the Voice Commands window.

Properties

Height Returns the current height, in pixels, of the Voice Commands window.
Left Returns the horizontal location, in pixels, of the Voice Commands window.
Top Returns the vertical location, in pixels, of the Voice Commands window.
Visible Specifies if the Voice Commands window is visible.
Width Returns current width, in pixels, of the Voice Commands window.

Remarks

The Voice Commands window is shared among all Double Agent clients. Changes to the window's visibility should be made with this in mind.

This object can be accessed using the Server Object's CommandsWindow property.
