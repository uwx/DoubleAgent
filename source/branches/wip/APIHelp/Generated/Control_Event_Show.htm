Double Agent API

The Show Event

Description

Signalled when a character is displayed.

Syntax

Visual Basic .NET

Event control_Show (ByVal CharacterID As String, ByVal Cause As DoubleAgent.Control.VisibilityCauseType)

Visual Basic 6

Sub control_Show (ByVal CharacterID As String, ByVal Cause As VisibilityCauseType)

VBScript

Sub control_Show (ByVal CharacterID, ByVal Cause)

Visual C#

delegate void control_ShowEventHandler (string CharacterID, DoubleAgent.Control.VisibilityCauseType Cause)

Visual C++

void control_Show (LPCTSTR CharacterID, VisibilityCauseType Cause)

Part Description
CharacterID The unique identifier of the character that was shown.
Cause Indicates what caused the character to be hidden.
2The user showed the character (using the menu or voice command).
4Your client application showed the character.
6Another client application showed the character.

Remarks

Double Agent sends this event to all clients of the character. To query the current state of the character, use the Visible property.
