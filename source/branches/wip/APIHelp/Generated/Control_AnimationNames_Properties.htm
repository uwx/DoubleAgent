Double Agent API

The AnimationNames Object Properties

Description

The AnimationNames collection includes properties for accessing its items.

Syntax

Visual Basic .NET

[Count = ]control.Characters ("CharacterID").AnimationNames.Count

[Item = ]control.Characters ("CharacterID").AnimationNames.Item (Index) or

[Item = ]control.Characters ("CharacterID").AnimationNames (Index)

Visual Basic 6

[Count = ]control.Characters ("CharacterID").AnimationNames.Count

[Item = ]control.Characters ("CharacterID").AnimationNames.Item (Index) or

[Item = ]control.Characters ("CharacterID").AnimationNames (Index)

VBScript

[Count = ]control.Characters ("CharacterID").AnimationNames.Count

[Item = ]control.Characters ("CharacterID").AnimationNames.Item (Index) or

[Item = ]control.Characters ("CharacterID").AnimationNames (Index)

Visual C#

[Count = ]control.Characters ("CharacterID").AnimationNames.Count

[Item = ]control.Characters ("CharacterID").AnimationNames.this [Index]

Visual C++

animationnames->get_Count (&Count)

animationnames->get_Item (Index, &Item)

Part Description
Count The number of items in the collection
Item An animation name string
