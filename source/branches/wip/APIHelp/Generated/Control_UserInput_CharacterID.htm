Double Agent API

The CharacterID Property

Description

Returns the unique identifier of the Character that is the object of the Command event.

Syntax

Visual Basic .NET

[CharacterID = ]userinput.CharacterID

Visual Basic 6

[CharacterID = ]userinput.CharacterID

VBScript

[CharacterID = ]userinput.CharacterID

Visual C#

[CharacterID = ]userinput.CharacterID

Visual C++

userinput->get_CharacterID (&CharacterID)
Part Description
CharacterID Identifies the Character whose Command was chosen.
