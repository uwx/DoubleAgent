Double Agent API

The Register Method

Description

Registers a notification sink for the client application.

Syntax

Visual C++

server->Register (NotifySink, &Cookie)
Part Description
NotifySink The IUnknown interface of an event notification sink object that implements IDaSvrNotifySink2.
Cookie A long value that identifies the connection.

Remarks

The Server Object also implements the standard COM IConnectionPoint protocol for notification sinks.

For languages that support ActiveX® controls, you can use the language's default mechanism for receiving events (eg. WithEvents). For managed languages, the control's Interop Assembly supports event delegates.
