Double Agent API

The Visible Property

Description

Returns a boolean indicating whether or not the character is visible.

Syntax

Visual Basic .NET

[Visible = ]control.Characters ("CharacterID").Visible

Visual Basic 6

[Visible = ]control.Characters ("CharacterID").Visible

VBScript

[Visible = ]control.Characters ("CharacterID").Visible

Visual C#

[Visible = ]control.Characters ("CharacterID").Visible

Visual C++

character.get_Visible (&Visible)

Part Description
Visible True if the character is displayed, false if it's hidden

Remarks

This property indicates whether the character's animationa are being displayed. It does not necessarily mean that there is an image on the screen. For example, this property returns True even when the character is positioned off the visible display area or when the current character animation contains no images. This property's setting applies to all clients of the character.

This property is read-only. To make a character visible or hidden, use the Show or Hide methods.
