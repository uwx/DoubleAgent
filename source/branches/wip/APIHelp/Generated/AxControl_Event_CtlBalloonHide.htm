Double Agent API

The CtlBalloonHide Event

Description

Signalled when a character's word balloon is hidden.

Syntax

Visual Basic .NET

Event control_CtlBalloonHide (ByVal Sender As Object, ByVal E As CtlBalloonHideEvent)

Visual C#

delegate void control_CtlBalloonHideEventHandler (object Sender, CtlBalloonHideEvent E)
Part Description
Sender The source of the event.
E A CtlBalloonHideEvent Object that contains the event data:
CharacterID
The unique identifier of the character who's balloon was hidden.

Remarks

Double Agent sends this event only to the clients of the character (applications that have loaded the character) that use the word balloon.
