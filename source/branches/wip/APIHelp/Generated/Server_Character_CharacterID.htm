Double Agent API

The CharacterID Property

Description

The unique value that identifies a loaded the Character Object.

Syntax

Visual C++

character->get_CharacterID (&CharacterID)

Visual C#

[CharacterID = ]character.CharacterID

Visual Basic .NET

[CharacterID = ]character.CharacterID

Visual Basic 6

[CharacterID = ]character.CharacterID

VBScript

[CharacterID = ]character.CharacterID
Part Description
CharacterID A numeric value that identifies the Character Object.

Remarks

The Server assigns a unique CharacterID to each Character Object instance it creates.
