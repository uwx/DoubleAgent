Double Agent API

The FilePath Property

Description

Returns the full path (UNC or HTTP) of the character's definition file.

Syntax

Visual Basic .NET

[FilePath = ]control.Characters ("CharacterID").FilePath

Visual Basic 6

[FilePath = ]control.Characters ("CharacterID").FilePath

VBScript

[FilePath = ]control.Characters ("CharacterID").FilePath

Visual C#

[FilePath = ]control.Characters.this ["CharacterID"].FilePath

Visual C++

character->get_FilePath (&FilePath)
Part Description
FilePath A string value that contains the character's definition file path.

Remarks

This is the full path of the file that was used to Load the character.
