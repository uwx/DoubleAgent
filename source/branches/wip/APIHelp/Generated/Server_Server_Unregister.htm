Double Agent API

The Unregister Method

Description

Unregisters a notification sink for the client application.

Syntax

Visual C++

server->Unregister (Cookie)
Part Description
Cookie A long value that identifies the connection. Use the Cookie returned by the Register Method

Remarks

If you registered a notification sink, use this method before you release the Server Object.

The Server Object also implements the standard COM IConnectionPoint protocol for notification sinks.

For languages that support ActiveX® controls, you can use the language's default mechanism for receiving events (eg. WithEvents). For managed languages, the control's Interop Assembly supports event delegates.
