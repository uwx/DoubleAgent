Double Agent API

The Height Property

Description

Returns a numeric value specifying the current height, in pixels, of the Double Agent property sheet window.

Syntax

Visual Basic .NET

[Height = ]control.PropertySheet.Height

Visual Basic 6

[Height = ]control.PropertySheet.Height

VBScript

[Height = ]control.PropertySheet.Height

Visual C#

[Height = ]control.PropertySheet.Height

Visual C++

propertysheet->get_Height (&Height)

Part Description
Height

Remarks

Double Agent displays the window based on the location set by the user.
