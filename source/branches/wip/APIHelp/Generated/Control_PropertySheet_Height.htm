Double Agent API

The Height Property

Description

Returns the current height, in pixels, of the Advanced Character Options window.

Syntax

Visual Basic .NET

[Height = ]control.PropertySheet.Height

Visual Basic 6

[Height = ]control.PropertySheet.Height

VBScript

[Height = ]control.PropertySheet.Height

Visual C#

[Height = ]control.PropertySheet.Height

Visual C++

propertysheet->get_Height (&Height)

Part Description
Height A numeric value that indicates the height, in pixels, of the Advanced Character Options window.
