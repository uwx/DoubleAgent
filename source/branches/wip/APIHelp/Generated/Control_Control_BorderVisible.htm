Double Agent API

The BorderVisible Property

Description

Specifies if the ActiveX® control's has a visible border.

Syntax

Visual Basic .NET

[BorderVisible = ]control.BorderVisible

control.BorderVisible = BorderVisible

Visual Basic 6

[BorderVisible = ]control.BorderVisible

control.BorderVisible = BorderVisible

VBScript

[BorderVisible = ]control.BorderVisible

control.BorderVisible = BorderVisible

Visual C#

[BorderVisible = ]control.BorderVisible

control.BorderVisible = BorderVisible

Visual C++

control->get_BorderVisible (&BorderVisible)

control->put_BorderVisible (BorderVisible)
Part Description
BorderVisible An boolean value that indicates if the border is visible.

Remarks

Note: This property is relevant only when the control is visible (i.e. when it's used in contained mode).
