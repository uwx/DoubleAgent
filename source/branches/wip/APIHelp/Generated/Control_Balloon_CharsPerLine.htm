Double Agent API

The CharsPerLine Property

Description

Returns or sets the characters per line used for a character's word balloon.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Balloon.CharsPerLine[ = Integer]
Visual Basic 6
control.Characters ("CharacterID").Balloon.CharsPerLine[ = Integer]
VBScript
control.Characters ("CharacterID").Balloon.CharsPerLine[ = Integer]
Visual C#
control.Characters ("CharacterID").Balloon.CharsPerLine[ = int];
Visual C++

balloon->get_CharsPerLine (&long);

balloon->put_CharsPerLine (long);

Remarks

The CharsPerLine property is the average number of characters (letters) displayed in the word balloon. For compatibility with Microsoft Agent, you can also set the value using the Style property.
