Double Agent API

The CharsPerLine Property

Description

Returns or sets the characters per line used for a character's word balloon.

Syntax

Visual Basic .NET

[CharsPerLine = ]control.Characters ("CharacterID").Balloon.CharsPerLine

control.Characters ("CharacterID").Balloon.CharsPerLine = CharsPerLine

Visual Basic 6

[CharsPerLine = ]control.Characters ("CharacterID").Balloon.CharsPerLine

control.Characters ("CharacterID").Balloon.CharsPerLine = CharsPerLine

VBScript

[CharsPerLine = ]control.Characters ("CharacterID").Balloon.CharsPerLine

control.Characters ("CharacterID").Balloon.CharsPerLine = CharsPerLine

Visual C#

[CharsPerLine = ]control.Characters ("CharacterID").Balloon.CharsPerLine

control.Characters ("CharacterID").Balloon.CharsPerLine = CharsPerLine

Visual C++

balloon->get_CharsPerLine (&CharsPerLine)

balloon->put_CharsPerLine (CharsPerLine)

Part Description
CharsPerLine A numeric value that specifies the outline color.

Remarks

The CharsPerLine property is the average number of characters (letters) displayed in the word balloon. For backward compatibility, you can also set the value using the Style property.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
