Double Agent API

The Left Property

Description

Specifies the horizontal location, in pixels, of the Advanced Character Options window.

Syntax

Visual Basic .NET

[Left = ]control.PropertySheet.Left

control.PropertySheet.Left = Left

Visual Basic 6

[Left = ]control.PropertySheet.Left

control.PropertySheet.Left = Left

VBScript

[Left = ]control.PropertySheet.Left

control.PropertySheet.Left = Left

Visual C#

[Left = ]control.PropertySheet.Left

control.PropertySheet.Left = Left

Visual C++

propertysheet->get_Left (&Left)

propertysheet->put_Left (Left)
Part Description
Left A numeric value that specifies the left edge, in pixels, of the Advanced Character Options window.

Remarks

Double Agent remembers and reuses the window location set by the user.
