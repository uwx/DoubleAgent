Double Agent API

The Left Property

Description

Returns an integer value specifying the current left edge, in pixels, of the Double Agent property sheet window.

Syntax

Visual Basic .NET

[Left = ]control.PropertySheet.Left

control.PropertySheet.Left = Left

Visual Basic 6

[Left = ]control.PropertySheet.Left

control.PropertySheet.Left = Left

VBScript

[Left = ]control.PropertySheet.Left

control.PropertySheet.Left = Left

Visual C#

[Left = ]control.PropertySheet.Left

control.PropertySheet.Left = Left

Visual C++

propertysheet->get_Left (&Left)

propertysheet->put_Left (Left)

Part Description
Left

Remarks

Double Agent displays the window based on the location set by the user.
