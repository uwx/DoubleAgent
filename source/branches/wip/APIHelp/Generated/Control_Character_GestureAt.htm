Double Agent API

The GestureAt Method

Description

Plays a character's gesturing animation towards the specified location.

Syntax

Visual Basic .NET

[Request = ]control.Characters ("CharacterID").GestureAt (X, Y)

Visual Basic 6

[Set Request = ]control.Characters ("CharacterID").GestureAt X, Y

VBScript

[Set Request = ]control.Characters ("CharacterID").GestureAt X, Y

Visual C#

[Request = ]control.Characters ("CharacterID").GestureAt (X, Y)

Visual C++

character.GestureAt (X, Y, &Request)

Part Description
X A numeric value that indicates the horizontal screen coordinate, in pixels, to which the character will gesture.
Y A numeric value that indicates the vertical screen coordinate, in pixels, to which the character will gesture.
Request A Request object your application can used to track the method's status.

Remarks

Double Agent automatically plays the appropriate animation to gesture toward the specified location. The coordinates are always relative to the upper left corner of the screen.
