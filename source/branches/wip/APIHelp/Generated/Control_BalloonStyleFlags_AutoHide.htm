Double Agent API

The AutoHide Member

Description

The word balloon is hidden automatically after a short delay.

Value

&H00000004

Remarks

When the AutoHide style is set, the word balloon automatically hides when spoken output completes. When it's not set, the balloon remains displayed until the next Speak or Think call, the character is hidden, or the user clicks or drags the character.
