Double Agent API

The SmoothEdges Property

Description

Indicates if image smoothing is being applied to the edges of the character's animations.

Syntax

Visual Basic .NET

[SmoothEdges = ]control.Characters ("CharacterID").SmoothEdges

Visual Basic 6

[SmoothEdges = ]control.Characters ("CharacterID").SmoothEdges

VBScript

[SmoothEdges = ]control.Characters ("CharacterID").SmoothEdges

Visual C#

[SmoothEdges = ]control.Characters.this ["CharacterID"].SmoothEdges

Visual C++

character.get_SmoothEdges (&SmoothEdges)

Part Description
SmoothEdges A boolean value that indicates if animation frames have smoothed edges.

Remarks

Agent character animations are defined with an 8-bit color depth (256 color palette), which can result in low image quality and visible pixelation. Edge smoothing reduces pixelation by applying a slight blur to the edges of each animation frame.

Note: This is a read-only property. Its value is initialized from the Control.CharacterStyle property.

Note: This property applies to all clients of the character.
