Double Agent API

The StopAll Method

Description

Stops all animation requests or specified types of requests.

Syntax

Visual C++

character->StopAll (Types)

Visual C#

character.StopAll (Types)

Visual Basic .NET

character.StopAll (Types)

Visual Basic 6

character.StopAll[ Types]

VBScript

character.StopAll[ Types]
Part Description
Types A numeric value specifying the types of actions to stop. You can use any combination of the following members of the StopAllFlags Enumeration:
0x00000001 = Play Stop any pending or running animations.
0x00000002 = Move Stop any pending or running move requests.
0x00000004 = Speak Stop any pending or running speech requests.
0x00000008 = QueuedPrepare Stop any queued animation or file downloads (Prepare method with Queue parameter set to True).
0x00000010 = ImmediatePrepare Stop any non-queued animation or file downloads (Prepare method with Queue parameter set to False)
0x00000020 = Visibility Stop any pending show or hide requests.
0xFFFFFFFF = Everything Stop all pending actions.
