Double Agent API

The RaiseRequestErrors Property

Description

Returns or sets whether an error condition is raised (an exception thrown) for failed requests.

Syntax

Visual Basic .NET

[RaiseErrors = ]control.RaiseRequestErrors

control.RaiseRequestErrors = RaiseErrors

Visual Basic 6

[RaiseErrors = ]control.RaiseRequestErrors

control.RaiseRequestErrors = RaiseErrors

VBScript

[RaiseErrors = ]control.RaiseRequestErrors

control.RaiseRequestErrors = RaiseErrors

Visual C#

[RaiseErrors = ]control.RaiseRequestErrors

control.RaiseRequestErrors = RaiseErrors

Visual C++

control->get_RaiseRequestErrors (&RaiseErrors)

control->put_RaiseRequestErrors (RaiseErrors)

Part Description
RaiseErrors A boolean value that determines whether errors are raised for failed requests. The default value is true.

Remarks

When a method that returns a Request object fails, there are two ways to detect the error. If RaiseRequestErrors is set, the Double Agent control will raise an error condition. If it's not set, you can check the Request object's Status property. An example of a request error is when you specify an animation name that doesn't exist for the Play method.
