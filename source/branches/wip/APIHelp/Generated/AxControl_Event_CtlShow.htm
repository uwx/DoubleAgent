Double Agent API

The CtlShow Event

Description

Signalled when a character is displayed.

Syntax

Visual Basic .NET

Event control_CtlShow (ByVal Sender As Object, ByVal E As CtlShowEvent)

Visual C#

delegate void control_CtlShowEventHandler (object Sender, CtlShowEvent E)
Part Description
Sender The source of the event.
E A CtlShowEvent Object that contains the event data:
CharacterID
The unique identifier of the character that was shown.
Cause
A VisibilityCauseType value that indicates what caused the character to be shown.
2 = UserShowed The user showed the character using the command on the character's taskbar icon Pop-up Menu or using speech input.
4 = ProgramShowed Your application showed the character.
6 = OtherProgramShowed Another client application showed the character.

Remarks

Double Agent sends this event to all clients of the character. To query the current state of the character, use the Visible property.
