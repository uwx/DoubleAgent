Double Agent API

The FileName Property

Description

Returns the name of the character's definition file.

Syntax

Visual C++

character->get_FileName (&FileName)

Visual C#

[FileName = ]character.FileName

Visual Basic .NET

[FileName = ]character.FileName

Visual Basic 6

[FileName = ]character.FileName

VBScript

[FileName = ]character.FileName
Part Description
FileName A string value that contains the character's definition file name.

Remarks

This is the name of the file that was used to Load the character.
