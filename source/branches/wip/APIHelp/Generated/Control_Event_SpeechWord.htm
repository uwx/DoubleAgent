Double Agent API

The SpeechWord Event

Description

Signalled when the character reaches a new word for the SpeakFormatted method.

Syntax

Part Description
CharacterID The unique identifier of the speaking character.
FormattedText The FormattedText that's being spoken.
WordIndex A numeric value that identifies the word that is currently being spoken.

Remarks

Because events are signalled asynchronously, the synchronization between the speech output and the receipt of this event cannot be exact. However, WordIndex will generally identify the next word to be spoken.

Note: The FormattedText parameter may not contain the same reference pointer as the FormattedText parameter passed to the SpeakFormatted method, but the underlying data is the same.
