Double Agent API

The Confidence Property

Description

Returns the speech recognition confidence scoring for the best matching Command.

Syntax

Visual Basic .NET

[Confidence = ]userinput.Confidence

Visual Basic 6

[Confidence = ]userinput.Confidence

VBScript

[Confidence = ]userinput.Confidence

Visual C#

[Confidence = ]userinput.Confidence

Visual C++

userinput->get_Confidence (&Confidence)
Part Description
Confidence The relative confidence scoring of best match, from -100 to 100 (100 if theCommand was chosen from the pop-up menu)
