The Interrupt Method

Description

Interrupts an animation for another character.

Syntax

Visual Basic .NET

[Request = ]control.Characters ("CharacterID").Interrupt (InterruptRequest)

Visual Basic 6

[Set Request = ]control.Characters ("CharacterID").Interrupt InterruptRequest

VBScript

[Set Request = ]control.Characters ("CharacterID").Interrupt InterruptRequest

Visual C#

[Request = ]control.Characters.this ["CharacterID"].Interrupt (InterruptRequest)

Visual C++

character->Interrupt (InterruptRequest, &Request)
Part Description
InterruptRequest The Request Object for the animation call you want to interrupt.
Request A Request Object your application can used to track the method's status.

Remarks

You can use this to synchronize animation between characters. For example, if another character is in a looping animation, this method will stop the loop and move to the next animation in the character's queue.

Example

To specify the request parameter, create a variable for the animation request you want to interrupt: 

Dim GenieRequest as Object
Dim RobbyRequest as Object
Dim Genie as Object
Dim Robby as Object

Sub form_Load()
    control.Characters.Load "Genie", "Genie.acs"
    control.Characters.Load "Robby", "Robby.acs"

    Set Genie = control.Characters ("Genie")
    Set Robby = control.Characters ("Robby")

    Genie.Show
    Genie.Speak "Just a moment"
    Set GenieRequest = Genie.Play ("Processing")

    Robby.Show
    Robby.Play "confused"
    Robby.Speak "Hey, Genie. What are you doing?"
    Robby.Interrupt GenieRequest

    Genie.Speak "I was just checking on something."
End Sub

Details

You can only use the Interrupt method to interrupt another character. To stop an animation for a single character, use the Stop method.

Note: The Interrupt method does not flush the character's queue; it halts the specified animation and moves on to the next animation in the character's queue. To halt and flush a character's queue, use that character's StopAll method.

Note: You cannot interrupt a character animation that you didn't start (that was started by another client).
