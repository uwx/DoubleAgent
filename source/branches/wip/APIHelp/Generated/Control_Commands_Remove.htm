Double Agent API

The Remove Method

Description

Removes a Command Object from the Commands Collection.

Syntax

Visual Basic .NET

control.Characters ("CharacterID").Commands.Remove ("Name")

Visual Basic 6

control.Characters ("CharacterID").Commands.Remove "Name"

VBScript

control.Characters ("CharacterID").Commands.Remove "Name"

Visual C#

control.Characters.this ["CharacterID"].Commands.Remove ("Name")

Visual C++

commands->Remove (Name)
Part Description
Name The name of the Command Object to remove.

Remarks

When a Command Object is removed from the collection, it no longer appears when the character's Pop-up Menu is displayed, nor in the Voice Commands window when your client application is input-active.
