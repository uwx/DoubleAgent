Double Agent API

The Remove Method

Description

Removes a Command object from the Commands collection.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Commands.Remove ("Name")
Visual Basic 6
control.Characters ("CharacterID").Commands.Remove "Name"
VBScript
control.Characters ("CharacterID").Commands.Remove "Name"
Visual C#
control.Characters.this ["CharacterID"].Commands.Remove ("Name")
Visual C++
commands.Remove (Name)
Part Description
Name The name of the Command object to remove.

Remarks

When a Command object is removed from the collection, it no longer appears when the character's pop-up menu is displayed, nor in the CommandsWindow when your client application is input-active.
