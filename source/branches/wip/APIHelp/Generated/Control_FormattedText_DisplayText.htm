Double Agent API

The DisplayText Property

Description

Returns the text displayed in the word balloon.

Syntax

Visual Basic .NET

[DisplayText = ]formattedtext.DisplayText

Visual Basic 6

[DisplayText = ]formattedtext.DisplayText

VBScript

[DisplayText = ]formattedtext.DisplayText

Visual C#

[DisplayText = ]formattedtext.DisplayText

Visual C++

formattedtext->get_DisplayText (&DisplayText)
Part Description
DisplayText A string value that contains the ParsedText up to and including the current WordIndex.

Remarks

The value of this property does not depend on the Balloon's Style, or even if a word balloon is shown. It reflects what would be displayed in a word balloon with the SizeToText and AutoPace styles.
