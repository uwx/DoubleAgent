Double Agent API

The BalloonFont Property

Description

Returns the font that will be used for all word balloons, if any.

Syntax

Visual C++

settings->get_BalloonFont (&BalloonFont)

Visual C#

[BalloonFont = ]server.Settings.BalloonFont

Visual Basic .NET

[BalloonFont = ]server.Settings.BalloonFont

Visual Basic 6

[BalloonFont = ]server.Settings.BalloonFont

VBScript

[BalloonFont = ]server.Settings.BalloonFont
Part Description
BalloonFont An optional font object that specifies the font the user has chosen to override all word balloon fonts.

Remarks

This property reflects the "A single font for all characters" option on the Output page of the Advanced Character Options window.

Note: The property setting affects all characters and is read-only; only the user can set this property value.
