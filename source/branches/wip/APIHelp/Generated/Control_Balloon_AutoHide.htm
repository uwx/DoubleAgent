Double Agent API

The AutoHide Property

Description

Indicates if the word balloon will be automatically hidden after all of its text has been shown.

Syntax

Visual Basic .NET

[AutoHide = ]control.Characters ("CharacterID").Balloon.AutoHide

control.Characters ("CharacterID").Balloon.AutoHide = AutoHide

Visual Basic 6

[AutoHide = ]control.Characters ("CharacterID").Balloon.AutoHide

control.Characters ("CharacterID").Balloon.AutoHide = AutoHide

VBScript

[AutoHide = ]control.Characters ("CharacterID").Balloon.AutoHide

control.Characters ("CharacterID").Balloon.AutoHide = AutoHide

Visual C#

[AutoHide = ]control.Characters.this ["CharacterID"].Balloon.AutoHide

control.Characters.this ["CharacterID"].Balloon.AutoHide = AutoHide

Visual C++

balloon->get_AutoHide (&AutoHide)

balloon->put_AutoHide (AutoHide)
Part Description
AutoHide A boolean value that indicates if the AutoHide style is enabled.

Remarks

When the AutoHide style is set, the word balloon automatically hides when spoken output completes. When it's not set, the word balloon remains displayed until the next Speak or Think call, the character is hidden, or the user clicks or drags the character.
