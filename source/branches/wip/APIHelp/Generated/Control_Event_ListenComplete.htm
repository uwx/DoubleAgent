Double Agent API

The ListenComplete Event

Description

Signalled when Listening mode (speech recognition) has ended.

Syntax

Visual Basic .NET

Event control_ListenComplete (ByVal CharacterID As String, ByVal Cause As DoubleAgent.Control.ListenCompleteType)

Visual Basic 6

Sub control_ListenComplete (ByVal CharacterID As String, ByVal Cause As ListenCompleteType)

VBScript

Sub control_ListenComplete (ByVal CharacterID, ByVal Cause)

Visual C#

delegate void control_ListenCompleteEventHandler (string CharacterID, DoubleAgent.Control.ListenCompleteType Cause)

Visual C++

void control_ListenComplete (LPCTSTR CharacterID, ListenCompleteType Cause)

Part Description
CharacterID The unique identifier of the listening character.
Cause The cause of the ListenComplete event.
1Listening mode was turned off by program code.
2Listening mode (turned on by program code) timed out.
3 Listening mode (turned on by the Listening key) timed out.
4 Listening mode was turned off because the user released the Listening key.
5Listening mode ended because the user finished speaking.
6Listening mode ended because the input-active client was deactivated.
7Listening mode ended because the default character was changed.
8Listening mode ended because the user disabled speech input.

Remarks

This event is sent to all clients when the Listening mode time-out ends, after the user releases the Listening key, when the input-active client calls the Listen method with False, or the user has finished speaking. You can use this event to determine when to resume spoken output.

If you turn on Listening mode using the Listen method and then the user presses the Listening key, the Listening mode resets and continues until the Listening key time-out completes, the Listening key is released, or the user finishes speaking, whichever is later. In this situation, you will not receive a ListenComplete event until the Listening key's mode completes.

The event returns the CharacterID to the clients that currently have this character loaded. All other clients receive a null CharacterID (empty string).
