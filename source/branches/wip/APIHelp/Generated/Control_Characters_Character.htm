Double Agent API

The Character Method

Description

Returns a Character Object in the Characters Collection.

Syntax

Visual Basic .NET

[Character = ]control.Characters.Character ("CharacterID")

Visual Basic 6

[Set Character = ]control.Characters.Character "CharacterID"

VBScript

[Set Character = ]control.Characters.Character "CharacterID"

Visual C#

[Character = ]control.Characters.Character ("CharacterID")

Visual C++

characters->Character ("CharacterID", &Character)
Part Description
CharacterID The character's unique identifier specified when the character was loaded.
Character A loaded Character Object.

Remarks

You can use this method to access a Character Object's methods and properties.

Note: This method may be required for some programming languages that don't support collections. For languages that do support collections, you can use the language's collection indexing syntax.
