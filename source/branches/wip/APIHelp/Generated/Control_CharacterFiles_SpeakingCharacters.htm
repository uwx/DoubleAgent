The SpeakingCharacters Property

Description

Indicates if the FilePaths property should include characters that are designed to speak.

Syntax

Visual Basic .NET

[SpeakingCharacters = ]control.CharacterFiles.SpeakingCharacters

control.CharacterFiles.SpeakingCharacters = SpeakingCharacters

Visual Basic 6

[SpeakingCharacters = ]control.CharacterFiles.SpeakingCharacters

control.CharacterFiles.SpeakingCharacters = SpeakingCharacters

VBScript

[SpeakingCharacters = ]control.CharacterFiles.SpeakingCharacters

control.CharacterFiles.SpeakingCharacters = SpeakingCharacters

Visual C#

[SpeakingCharacters = ]control.CharacterFiles.SpeakingCharacters

control.CharacterFiles.SpeakingCharacters = SpeakingCharacters

Visual C++

characterfiles.get_SpeakingCharacters (&SpeakingCharacters)

characterfiles.put_SpeakingCharacters (SpeakingCharacters)

Part Description
SpeakingCharacters A boolean value that indicates if speaking characters should be included.
