Double Agent API

The ListenCompleteType Enumeration
1 = ProgramDisabled This program turned off the Listening mode with the Listen method.
2 = ProgramTimedOut The Listening mode was started by this program with the Listen method, and it timed out.
3 = UserTimedOut The Listening mode was started with the Listening key, and it timed out.
4 = UserReleasedKey The Listening mode was started with the Listening key, and the key was released.
5 = UserSpeechEnded Speech recognition has produced a result.
6 = CharacterClientDeactivated The character is no longer active or it's active client has changed.
7 = DefaultCharacterChanged The default character was changed in the Advanced Character Options window.
8 = UserDisabled Speech recognition was disabled in the Advanced Character Options window.
