Double Agent API

The Show Method

Description

Shows the character and optionally plays its associated Showing animation.

Syntax

Visual Basic .NET

[Request = ]control.Characters ("CharacterID").Show (Fast)

Visual Basic 6

[Set Request = ]control.Characters ("CharacterID").Show[ Fast]

VBScript

[Set Request = ]control.Characters ("CharacterID").Show[ Fast]

Visual C#

[Request = ]control.Characters.this ["CharacterID"].Show (Fast)

Visual C++

character->Show (Fast, &Request)
Part Description
Fast A boolean value specifying whether or not the Showing state animation is played.
Request A Request Object your application can use to track the method's status.

Remarks

Avoid setting the Fast parameter to True without first playing an animation; otherwise, the character may be shown with no image. In particular, note that if you call MoveTo when the character is not visible, it does not play any animation. Therefore, if you call the Show method with Fast set to True, no image will display. Similarly, if you call Hide, then Show with Fast set to True, there will be no visible image.
