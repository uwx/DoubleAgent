Double Agent API

The CommandsWindow Object Properties

Description

The CommandsWindow object provides access to Double Agent's Voice Commands window.

Syntax

[Left = ]control.CommandsWindow.Left

control.CommandsWindow.Left = Left

[Top = ]control.CommandsWindow.Top

control.CommandsWindow.Top = Top

[Width = ]control.CommandsWindow.Width

[Height = ]control.CommandsWindow.Height

[Visible = ]control.CommandsWindow.Visible

control.CommandsWindow.Visible = Visible

commandswindow->put_Visible (Visible)

Part Description
Left A numeric value that specifies the left edge, in pixels, of the Voice Commands window.
Top A numeric value that specifies the top edge, in pixels, of the Voice Commands window.
Width A numeric value that indicates the width, in pixels, of the Voice Commands window.
Height A numeric value that indicates the height, in pixels, of the Voice Commands window.
Visible A boolean value indicating whether or not the Voice Commands window is visible.

Remarks

The Voice Commands window is shared among all Double Agent clients. Changes to the window's location and visibility should be made with this in mind.
