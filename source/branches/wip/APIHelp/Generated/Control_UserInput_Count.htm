Double Agent API

The Count Property

Description

Returns the number of possible matching commands.

Syntax

Visual Basic .NET

[Count = ]userinput.Count

Visual Basic 6

[Count = ]userinput.Count

VBScript

[Count = ]userinput.Count

Visual C#

[Count = ]userinput.Count

Visual C++

userinput->get_Count (&Count)

Part Description
Count The number of possible matching commands, from 0 to 3.

Remarks

Even if the Count is zero (no commands were matched), the Voice property will still return the spoken text.
