Double Agent API

The SRListeningTip Property

Description

Returns a Boolean indicating the current user setting for the Listening Tip.

Syntax

Visual Basic .NET

[SRListeningTip = ]control.Settings.SRListeningTip

Visual Basic 6

[SRListeningTip = ]control.Settings.SRListeningTip

VBScript

[SRListeningTip = ]control.Settings.SRListeningTip

Visual C#

[SRListeningTip = ]control.Settings.SRListeningTip

Visual C++

settings->get_SRListeningTip (&SRListeningTip)

Part Description
SRListeningTip

Remarks

The ListeningTip property indicates whether the Display Listening Tip option in the Microsoft Agent property sheet (Advanced Character Options) is enabled. When ListeningTip returns True and speech input is enabled, Double Agent displays the tip window when the user presses the Listening key.
