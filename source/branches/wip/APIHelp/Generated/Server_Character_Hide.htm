Double Agent API

The Hide Method

Description

Hides the character.

Syntax

Visual C++

character->Hide (Fast, &RequestID)

Visual C#

character.Hide (Fast, out RequestID)

Visual Basic .NET

character.Hide (Fast, RequestID)

Visual Basic 6

character.Hide [Fast][, RequestID]

VBScript

character.Hide [Fast][, RequestID]
Part Description
Fast A boolean value that indicates whether to skip the character's Hiding state animation.
RequestID An optional unique numeric value your application can use to track the method's status.

Remarks

Double Agent queues the actions of the Hide method in the character's queue, so you can use it to hide the character after a sequence of other animations. You can play the action immediately by using the Stop or StopAll method before calling this method.

Note: Hiding a character can trigger the ActivateInputState event of another client.

Note: Hidden characters cannot access the screen or the audio channel. Double Agent will pass a failure status in the RequestComplete event if you attempt to animate a hidden character.
