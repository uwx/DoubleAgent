The IconVisible Property

Description

Indicates if the character has an icon in the Windows notification area.

Syntax

Visual Basic .NET

[IconVisible = ]control.Characters ("CharacterID").IconVisible

Visual Basic 6

[IconVisible = ]control.Characters ("CharacterID").IconVisible

VBScript

[IconVisible = ]control.Characters ("CharacterID").IconVisible

Visual C#

[IconVisible = ]control.Characters.this ["CharacterID"].IconVisible

Visual C++

character.get_IconVisible (&IconVisible)

Part Description
IconVisible A boolean value that indicates if the character is showing a notification icon.

Remarks

This property only indicates if the notification icon has been created. The user decides if the icon is actually visible or hidden.

The IconShown property only applies to your client application, so it is only in effect when you application is the character's active client. When your application is not the character's active client, you can use this property to determine if another client has shown or hidden the notification icon.

Note: This property applies to all clients of the character.
