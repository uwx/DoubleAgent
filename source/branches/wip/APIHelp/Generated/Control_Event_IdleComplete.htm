Double Agent API

The IdleComplete Event

Description

Signalled when Double Agent ends the Idling state of a character.

Syntax

Visual Basic .NET

Event control_IdleComplete (ByVal CharacterID As String)

Visual Basic 6

Sub control_IdleComplete (ByVal CharacterID As String)

VBScript

Sub control_IdleComplete (ByVal CharacterID)

Visual C#

delegate void control_IdleCompleteEventHandler (string CharacterID)

Visual C++

void control_IdleComplete (LPCTSTR CharacterID)
Part Description
CharacterID The unique identifier of the Idling character.

Remarks

Double Agent sends this event to all clients of the character.
