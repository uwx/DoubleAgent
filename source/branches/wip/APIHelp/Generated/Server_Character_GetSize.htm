Double Agent API

The GetSize Method

Description

Retrieves the character's dimensions on the screen.

Syntax

Visual C++

character->GetSize (&Width, &Height)

Visual C#

character.GetSize (out Width, out Height)

Visual Basic .NET

character.GetSize (Width, Height)

Visual Basic 6

character.GetSize Width, Height

VBScript

character.GetSize Width, Height
Part Description
Width A numeric value that specifies the width of the character's image.
Height A numeric value that specifies the height of the character's image.

Remarks

The character's dimensions are always expressed in pixels.

Even though the character appears in an irregularly shaped region window, the dimensions of the character are based on the external dimensions of the rectangular animation frame used when the character was compiled with the Microsoft Agent Character Editor.

Note: This property applies to all clients of the character.
