Double Agent API

The Balloon Object

Description

Double Agent supports textual captioning of the Speak method using a cartoon word balloon. The Think method lets you display text without audio output in a thought balloon.

The Balloon Object allows you to customize the appearance and behaviour of a character's word balloon.

Properties

AutoHide Indicates if the word balloon will be automatically hidden after all of its text has been shown.
AutoPace Indicates if the word balloon paces its text display to the speech output.
BackColor Specifies the background color of a character's word balloon.
BorderColor Specifies the border color of a character's word balloon.
CharsPerLine Specifies the characters per line used for a character's word balloon.
Enabled Indicates if the word balloon is enabled for a character.
FontBold Specifies the font style of a character's the word balloon.
FontCharSet Specifies the character set for the font used in a character's word balloon.
FontItalic Specifies the font style of a character's the word balloon.
FontName Specifies the font used in the word balloon for the specified character.
FontSize Specifies the font size supported for the word balloon for the specified character.
FontStrikethru Specifies the font style of a character's the word balloon.
FontUnderline Specifies the font style of a character's word balloon.
NumberOfLines Specifies the number of lines used for a character's word balloon.
ShowPartialLines Indicates if the word balloon shows partial lines when it scrolls.
SizeToText Indicates if the word balloon is automatically resized to fit it's contents.
Style Specifies the character's word balloon output style.
TextColor Returns the text color displayed in a character's word balloon.
Visible Specifies the visiblity of a character's word balloon.

Remarks

A character's initial word balloon defaults are defined and compiled in the Microsoft Agent Character Editor. Once running, the balloon's Enabled and Font properties may be overridden by the user. If a user changes the word balloon's properties, they affect all characters. Both the Speak and Think word balloons use the same property settings for size. You can access the properties for a character's word balloon through the Balloon Object, which is a child of the Character Object.

This object can be accessed using the Character Object's Balloon property.
