Double Agent API

The AutoSize Property

Description

Specifies if the ActiveX® control is automatically sized to fit the ControlCharacter.

Syntax

Visual Basic .NET

[AutoSize = ]control.AutoSize

control.AutoSize = AutoSize

Visual Basic 6

[AutoSize = ]control.AutoSize

control.AutoSize = AutoSize

VBScript

[AutoSize = ]control.AutoSize

control.AutoSize = AutoSize

Visual C#

[AutoSize = ]control.AutoSize

control.AutoSize = AutoSize

Visual C++

control->get_AutoSize (&AutoSize)

control->put_AutoSize (AutoSize)
Part Description
AutoSize A boolean value that specifies if the control's size is automatically adjusted to exactly fit the contained character.

Remarks

Note: This property is relevant only when the control is visible (i.e. when it's used in contained mode).
