Double Agent API

The ExtraData Property

Description

Returns a string that specifies additional data stored as part of the character.

Syntax

Visual Basic .NET

[ExtraData = ]control.Characters ("CharacterID").ExtraData

Visual Basic 6

[ExtraData = ]control.Characters ("CharacterID").ExtraData

VBScript

[ExtraData = ]control.Characters ("CharacterID").ExtraData

Visual C#

[ExtraData = ]control.Characters.this ["CharacterID"].ExtraData

Visual C++

character->get_ExtraData (&ExtraData)
Part Description
ExtraData A string that contains any additional data stored as part of the character.

Remarks

The default value for a character's ExtraData property is defined when the character is compiled with the Microsoft Agent Character Editor. It cannot be changed or specified at run time.

Note: The ExtraData property setting is optional and may not be supplied for all characters.
