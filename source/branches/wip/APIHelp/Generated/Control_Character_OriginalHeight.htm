Double Agent API

The OriginalHeight Property

Description

Returns an integer value that specifies the character's original height.

Syntax

Visual Basic .NET

[OriginalHeight = ]control.Characters ("CharacterID").OriginalHeight

Visual Basic 6

[OriginalHeight = ]control.Characters ("CharacterID").OriginalHeight

VBScript

[OriginalHeight = ]control.Characters ("CharacterID").OriginalHeight

Visual C#

[OriginalHeight = ]control.Characters ("CharacterID").OriginalHeight

Visual C++

character.get_OriginalHeight (&OriginalHeight)

Part Description
OriginalHeight This property returns the character's frame height as built with the Microsoft Agent Character Editor.

Remarks

This property returns the character's frame height as built with the Microsoft Agent Character Editor.
