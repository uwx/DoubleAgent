Double Agent API

The OriginalHeight Property

Description

Returns the character's default height.

Syntax

Visual Basic .NET

[OriginalHeight = ]control.Characters ("CharacterID").OriginalHeight

Visual Basic 6

[OriginalHeight = ]control.Characters ("CharacterID").OriginalHeight

VBScript

[OriginalHeight = ]control.Characters ("CharacterID").OriginalHeight

Visual C#

[OriginalHeight = ]control.Characters.this ["CharacterID"].OriginalHeight

Visual C++

character->get_OriginalHeight (&OriginalHeight)
Part Description
OriginalHeight The character's image height as defined with the Microsoft Agent Character Editor.
