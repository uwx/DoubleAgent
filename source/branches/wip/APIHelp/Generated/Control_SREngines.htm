The SREngines Collection

The SREngines collection contains a selected set of SREngine objects. You can use it to help choose a character's speech recognition engine.

  • The SREngines object has the following properties:

    Count, Item

Remarks

The contents of the collection depends on how it was retrieved.

When the SREngines Collection is retrieved using matching criteria (for example using the FindSREngines method), it is an ordered collection. It's items are in order from best to worst match.
