Double Agent API

The LanguageIDs Property

Description

Returns the speech recognition engine's supported language identifiers.

Syntax

Visual Basic .NET

[LanguageIDs = ]srengine.LanguageIDs

Visual Basic 6

[LanguageIDs = ]srengine.LanguageIDs

VBScript

[LanguageIDs = ]srengine.LanguageIDs

Visual C#

[LanguageIDs = ]srengine.LanguageIDs

Visual C++

srengine.get_LanguageIDs (&LanguageIDs)

Part Description
LanguageIDs An array of numeric values that identify all languages supported by the engine.
