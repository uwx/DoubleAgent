Double Agent API

The CharacterFiles Object

Description

The CharacterFiles Object provides information that allows you to locate locally installed character definition files. It includes a number of filtering properties that specify where to look for character files, and what kind of files to find. The results are returned in its FilePaths property.

Properties

DefaultFileName Returns the name of the default character's definition file.
DefaultFilePath Returns the full path of the default character's definition file.
DefaultSearchPath Identifies the default search path used to fill the FilePaths property.
DoubleAgentFiles Indicates if the Double Agent install location should be included in the DefaultSearchPath.
FilePaths Returns a list of character file paths that satisfy the filtering criteria defined by other properties in the CharacterFiles Object.
MsAgentFiles Indicates if the Microsoft Agent install location should be included in the DefaultSearchPath.
MsOfficeFiles Indicates if the Microsoft Office install location should be included in the DefaultSearchPath.
NonSpeakingCharacters Indicates if the FilePaths property should include characters that are not designed to speak.
SearchPath Identifies the actual search path used to fill the FilePaths property. Unless you have special requirements, this is usually the same as the DefaultSearchPath property.
SpeakingCharacters Indicates if the FilePaths property should include characters that are designed to speak.
VerifyVersion Indicates if the the FilePaths property should be limited to version 2.x characters.

Remarks

This object can be accessed using the Control Object's CharacterFiles property.
