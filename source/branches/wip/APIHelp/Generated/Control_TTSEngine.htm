The TTSEngine Object

Description

The TTSEngine Object provides information about an installed text-to-speech engine. You can use it to help choose a character's text-to-speech engine.

Methods

GetVersion Returns the text-to-speech engine's version.

Properties

DisplayName Returns the text-to-speech engine's human-readable name.
Gender Returns the text-to-speech engine's gender.
LanguageID Returns the text-to-speech engine's language identifier.
LanguageName Returns the text-to-speech engine's human-readable language name.
Manufacturer Returns the text-to-speech engine's manufacturer's name.
TTSModeID Returns the unique identifier used to select this text-to-speech engine.

Remarks

Some text-to-speech engines support several modes, generally one for each supported language or language variant. In that case, this object represents only one mode, and there will one TTSEngine Object for each mode supported by the speech engine, each with a unique TTSModeID property.
