Double Agent API

The Count Property

Description

Returns the count of Command objects in the Commands collection.

Syntax

Visual Basic .NET
[Count = ]control.Characters ("CharacterID").Commands.Count
Visual Basic 6
[Count = ]control.Characters ("CharacterID").Commands.Count
VBScript
[Count = ]control.Characters ("CharacterID").Commands.Count
Visual C#
[Count = ]control.Characters ("CharacterID").Commands.Count
Visual C++
command.get_Count (&Count)
Part Description
Count A numeric value that indicates the number of Command objects in the collection.

Remarks

Count includes only the number of Command objects you define in yourCommands collection. Pre-defined entries or entries defined by other clients are not included.
