Double Agent API

The Count Property

Description

Returns the count of Command Objects in the Commands Collection.

Syntax

Visual Basic .NET

[Count = ]control.Characters ("CharacterID").Commands.Count

Visual Basic 6

[Count = ]control.Characters ("CharacterID").Commands.Count

VBScript

[Count = ]control.Characters ("CharacterID").Commands.Count

Visual C#

[Count = ]control.Characters.this ["CharacterID"].Commands.Count

Visual C++

command->get_Count (&Count)
Part Description
Count A numeric value that indicates the number of Command Objects in the collection.

Remarks

Count includes only the number of Command Objects you define in your Commands Collection. Pre-defined entries or entries defined by other clients are not included.
