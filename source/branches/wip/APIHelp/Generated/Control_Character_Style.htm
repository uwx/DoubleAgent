Double Agent API

The Style Property

Description

Specifies a group of settings that affect the character's appearence.

Syntax

Visual Basic .NET

[Style = ]control.Characters ("CharacterID").Style

control.Characters ("CharacterID").Style = Style

Visual Basic 6

[Style = ]control.Characters ("CharacterID").Style

control.Characters ("CharacterID").Style = Style

VBScript

[Style = ]control.Characters ("CharacterID").Style

control.Characters ("CharacterID").Style = Style

Visual C#

[Style = ]control.Characters.this ["CharacterID"].Style

control.Characters.this ["CharacterID"].Style = Style

Visual C++

character->get_Style (&Style)

character->put_Style (Style)
Part Description
Style A numeric bit-field value that has one or more of the following:
&H00000001 = SoundEffects The animation sound effects are played.
&H00000002 = IdleEnabled The character enters idle mode and shows it's idling animations when it's not doing anything else.
&H00000004 = AutoPopupMenu A right-click on the character or it's notification icon automatically shows a pop-up menu.
&H00000008 = IconShown The character has an icon in the taskbar notification area.
&H00000020 = SmoothEdges Image smoothing (antialiasing) is applied to the edges of the character's animations.
&H00000030 = Smoothed Image smoothing (antialiasing) is applied to the character's animations.

Remarks

Each style bit corresponds to a specific Character Object property. You can use this property to query or modify all of the style settings at once.

The Smoothed and SmoothEdges bits are read-only. They can only be set in the Control.CharacterStyle property before the Character is loaded.
