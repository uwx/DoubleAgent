Double Agent API

The SREngines Property

Description

Returns a collection of Speech Recognition Engine Objects for all available speech recognition engines.

Syntax

Visual Basic .NET

[SREngines = ]control.SREngines

Visual C#

[SREngines = ]control.SREngines
Part Description
SREngines A SREngines Collection that includes all available speech recognition engines.

Remarks

This property allows you to query available speech recognition engines without loading a character.
