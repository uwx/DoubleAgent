Double Agent API

The Commands Collection

Description

The Commands Collection defines the Commands that the user can apply to a Character.

Methods

Add Adds a Command Object to the Commands Collection.
Command Returns a Command Object in a Commands Collection.
Insert Inserts a Command Object in the Commands Collection.
Remove Removes a Command Object from the Commands Collection.
RemoveAll Removes all Command Objects from the Commands Collection.

Properties

Caption Specifies the text displayed for the Commands Collection in the character's Pop-up Menu.
Count Returns the count of Command Objects in the Commands Collection.
DefaultCommand Specifies the default command of the Commands Collection.
FontName Specifies the font used in the Voice Commands window.
FontSize Specifies the font size used in the Voice Commands window.
Index Returns a Command Object in the collection.
Item Returns a Command Object in the collection.
Visible Specifies if the Commands Collection appears in the character's Pop-up Menu.
VoiceGrammar Specifies the text that is passed to the speech engine for recognition.
VoiceCaption Specifies the text displayed for the Commands Collection in the Voice Commands window.

Remarks

The Commands Collection can appear in both the Pop-up Menu and the Voice Commands window for a character. To include commands in the Pop-up Menu, set its Caption property. To include commands in the Voice Commands window, set its VoiceCaption property (for backward compatibility, if there is no VoiceCaption, the Caption setting is used).

This object can be accessed using the Character's Commands property.
