Double Agent API

The Commands Collection
The Commands collection defines the commands that the user can apply to a Character.
  • The Commands collection has the following methods:

    Add, Command, Insert, Remove, RemoveAll

  • The Commands collection has the following properties:

    Caption, Count, DefaultCommand, FontName, FontSize, HelpContextID, Visible, VoiceGrammar, VoiceCaption

Remarks

The Commands collection can appear in both the pop-up menu and the Voice Commands Window for a character. To include commands in the pop-up menu, set its Caption property. To include commands in the Voice Commands Window, set its VoiceCaption property (for backward compatibility, if there is no VoiceCaption, the Caption setting is used).
