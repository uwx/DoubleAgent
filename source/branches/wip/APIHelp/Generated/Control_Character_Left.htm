Double Agent API

The Left Property

Description

Specifies the left edge of the character's image.

Syntax

Visual Basic .NET

[Left = ]control.Characters ("CharacterID").Left

control.Characters ("CharacterID").Left = Left

Visual Basic 6

[Left = ]control.Characters ("CharacterID").Left

control.Characters ("CharacterID").Left = Left

VBScript

[Left = ]control.Characters ("CharacterID").Left

control.Characters ("CharacterID").Left = Left

Visual C#

[Left = ]control.Characters.this ["CharacterID"].Left

control.Characters.this ["CharacterID"].Left = Left

Visual C++

character->get_Left (&Left)

character->put_Left (Left)
Part Description
Left A numeric value that specifies the left edge of the character's image.

Remarks

The Left property is always expressed in pixels, relative to screen origin (upper left).

Even though the character appears in an irregularly shaped region window, the location of the character is based on the external dimensions of the rectangular animation frame used when the character was compiled with the Microsoft Agent Character Editor.

Note: This property applies to all clients of the character.
