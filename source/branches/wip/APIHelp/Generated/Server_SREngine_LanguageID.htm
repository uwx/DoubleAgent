Double Agent API

The LanguageID Property

Description

Returns the speech recognition engine's primary language identifier.

Syntax

Visual C++

srengine->get_LanguageID (&LanguageID)

Visual C#

[LanguageID = ]srengine.LanguageID

Visual Basic .NET

[LanguageID = ]srengine.LanguageID

Visual Basic 6

[LanguageID = ]srengine.LanguageID

VBScript

[LanguageID = ]srengine.LanguageID
Part Description
LanguageID A numeric value that identifies the primary language supported by the engine.
