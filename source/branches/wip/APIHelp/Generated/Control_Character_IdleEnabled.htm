Double Agent API

The IdleEnabled Property

Description

Specifies if the character will show its Idling animations when it's not busy.

Syntax

Visual Basic .NET

[IdleEnabled = ]control.Characters ("CharacterID").IdleEnabled

control.Characters ("CharacterID").IdleEnabled = IdleEnabled

Visual Basic 6

[IdleEnabled = ]control.Characters ("CharacterID").IdleEnabled

control.Characters ("CharacterID").IdleEnabled = IdleEnabled

VBScript

[IdleEnabled = ]control.Characters ("CharacterID").IdleEnabled

control.Characters ("CharacterID").IdleEnabled = IdleEnabled

Visual C#

[IdleEnabled =] control.Characters.this ["CharacterID"].IdleEnabled

control.Characters.this ["CharacterID"].IdleEnabled = IdleEnabled

Visual C++

character->get_IdleEnabled (&IdleEnabled)

character->put_IdleEnabled (IdleEnabled)
Part Description
IdleEnabled A boolean value specifying whether Double Agent manages idle mode.

Remarks

Double Agent automatically sets a time-out after the last animation played for a character. When this timer's interval is complete, Double Agent begins the Idling state for a character, playing its associated Idling animations at regular intervals. If you want to disable Double Agent from automatically playing the Idling state animations, set the property to False and play an animation or call the Stop method. Setting this value does not affect the current animation state of the character.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
