Double Agent API

The ShowPopupMenu Method

Description

Displays the character's pop-up menu at the specified location.

Syntax

Visual C++

character->ShowPopupMenu (X, Y)

Visual C#

character.ShowPopupMenu (X, Y)

Visual Basic .NET

character.ShowPopupMenu (X, Y)

Visual Basic 6

character.ShowPopupMenu X, Y

VBScript

character.ShowPopupMenu X, Y
Part Description
X The horizontal screen coordinate to display the menu, in pixels.
Y The vertical screen coordinate to display the menu, in pixels.

Remarks

If the character's Style property includes AutoPopupMenu, Double Agent automatically displays the character's pop-up menu when the user right-clicks the character. If you set the Style property without AutoPopupMenu, you can use ShowPopupMenu to display the menu.

The menu remains displayed until the user selects a command or displays another menu. Only one pop-up menu can be displayed at a time; therefore, calls to this method will cancel (remove) the former menu.

This method should be called only when your client application is the active client of the character; otherwise it fails.
