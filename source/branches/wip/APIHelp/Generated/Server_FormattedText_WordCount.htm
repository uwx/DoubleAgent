Double Agent API

The WordCount Property

Description

Returns the number of words in the ParsedText and SpeechText properties.

Syntax

Visual C++

formattedtext->get_WordCount (&WordCount)

Visual C#

[WordCount = ]formattedtext.WordCount

Visual Basic .NET

[WordCount = ]formattedtext.WordCount

Visual Basic 6

[WordCount = ]formattedtext.WordCount

VBScript

[WordCount = ]formattedtext.WordCount
Part Description
WordCount A numeric value that indicates how many words are in the parsed text.

Remarks

The WordCount reflects how text is displayed in the word balloon and how the DisplayText is synchronized with the speech output.
