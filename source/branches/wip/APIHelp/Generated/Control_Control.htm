Double Agent API

The Control Object

Description

The Double Agent control has its own methods, properties and events, and provides access to most other objects supported by Double Agent.

Methods

FindCharacterSREngines Retrieves a collection of Speech Recognition Engine Objects that meet a specific character's requirements.
FindCharacterTTSEngines Retrieves a collection of Text-To-Speech Engine Objects that meet a specific character's requirements.
FindSREngines Retrieves a collection of Speech Recognition Engine Objects for a specific language.
FindTTSEngines Retrieves a collection of Text-To-Speech Engine Objects for a specific language and gender.
GetCharacterSREngine Retrieves a Speech Recognition Engine Object thats meet a specific character's requirements.
GetCharacterTTSEngine Retrieves a Text-To-Speech Engine Object that meets a specific character's requirements.
ShowDefaultCharacterProperties Shows the Gallery page of the Advanced Character Options window.

Properties

AutoConnect Specifies if the Double Agent control will automatically connect to the Double Agent server.
CharacterStyle Specifies the initial style to use when a Character is loaded.
Connected Specifies if the control is currently connected to the Double Agent server.
ControlCharacter Specifies the control's contained Character.
RaiseRequestErrors Specifies if an error condition is raised (an exception thrown) for failed requests.

The Control Object also includes properties to retrieve other Double Agent objects:

CharacterFiles Returns the control's CharacterFiles Object.
Characters Returns the control's Characters Collection.
CommandsWindow Returns the control's CommandsWindow Object.
PropertySheet Returns the control's PropertySheet Object.
Settings Returns the control's Settings Object.
SREngines Returns a collection of Speech Recognition Engine Objects for all available speech recognition engines.
TTSEngines Returns a collection of Text-To-Speech Engine Objects for all available text-to-speech engines.

The Control Object supports following ActiveX® appearence properties when the control is used in contained mode:

AutoSize Specifies if the ActiveX® control is automatically sized to fit the ControlCharacter.
BackColor Specifies the ActiveX® control's background color.
BorderColor Specifies the ActiveX® control's border color.
BorderStyle Specifies the style of the ActiveX® control's border.
BorderVisible Specifies if the ActiveX® control's has a visible border.
BorderWidth Specifies the width of the ActiveX® control's border.
MousePointer Specifies the pointer to show when the mouse moves over the ActiveX® control.

Events

ActivateInput Signalled when a client becomes input-active.
ActiveClientChange Signalled when the active client of the character changes.
AgentPropertyChange Signalled when the user changes a property in the Advanced Character Options window.
BalloonHide Signalled when a character's word balloon is hidden.
BalloonShow Signalled when a character's word balloon is shown.
Bookmark Signalled when a bookmark defined by your application in a speech text string is reached.
Click Signalled when the user clicks a character, or the character's icon.
Command Signalled when the user chooses a (client's) command.
DblClick Signalled when the user double-clicks a character.
DeactivateInput Signalled when a client becomes non-input-active.
DefaultCharacterChange Signalled when the user changes the default character.
DragComplete Signalled when the user stops dragging a character.
DragStart Signalled when the user starts dragging a character.
Hide Signalled when a character is hidden.
IdleComplete Signalled when Double Agent ends the Idling state of a character.
IdleStart Signalled when Double Agent sets a character to the Idling state.
ListenComplete Signalled when Listening mode (speech recognition) has ended.
ListenStart Signalled when Listening mode (speech recognition) begins.
Move Signalled when a character has moved.
RequestComplete Signalled when Double Agent completes a queued request.
RequestStart Signalled when Double Agent begins a queued request.
Show Signalled when a character is displayed.
Size Signalled when a character's size changes.

Note: Double Agent does not support the Restart, Shutdown or HelpComplete events.

Remarks

Some programming environments may assign additional design-time or run-time properties. For example, Visual Basic adds a Name property that identifies the control, and also Left and Top properties that define the location of the control on a form, even if the control doesn't appear on the form's page at run time.
