Double Agent API

The Visible Property

Description

Returns or sets whether the Double Agent property sheet window is visible (open).

Syntax

Visual Basic .NET

[Visible = ]control.PropertySheet.Visible

control.PropertySheet.Visible = Visible

Visual Basic 6

[Visible = ]control.PropertySheet.Visible

control.PropertySheet.Visible = Visible

VBScript

[Visible = ]control.PropertySheet.Visible

control.PropertySheet.Visible = Visible

Visual C#

[Visible = ]control.PropertySheet.Visible

control.PropertySheet.Visible = Visible

Visual C++

propertysheet->get_Visible (&Visible)

propertysheet->put_Visible (Visible)

Part Description
Visible

Remarks

Double Agent displays the window based on the location and size set by the user.

Note: The user can override this property.
