Double Agent API

The Version Property

Description

Returns the character's version.

Syntax

Visual Basic .NET

[Version = ]control.Characters ("CharacterID").Version

Visual Basic 6

[Version = ]control.Characters ("CharacterID").Version

VBScript

[Version = ]control.Characters ("CharacterID").Version

Visual C#

[Version = ]control.Characters.this ["CharacterID"].Version

Visual C++

character->get_Version (&Version)
Part Description
Version A string that identifies the Microsoft Agent version for which the character was built.

Remarks

The Version property returns a string that corresponds to the version of the standard animation set definition for which the character was compiled. The character's version number is automatically set when you build it with the Microsoft Agent Character Editor.
