Double Agent API

The Version Property

Description

Returns version of the character.

Syntax

Visual Basic .NET

[Version = ]control.Characters ("CharacterID").Version

Visual Basic 6

[Version = ]control.Characters ("CharacterID").Version

VBScript

[Version = ]control.Characters ("CharacterID").Version

Visual C#

[Version = ]control.Characters ("CharacterID").Version

Visual C++

character.get_Version (&Version)

Part Description
Version The Version property returns a string that corresponds to the version of the standard animation set definition for which the character was compiled. The character's version number is automatically set when you build it with the Microsoft Agent Character Editor.

Remarks

The Version property returns a string that corresponds to the version of the standard animation set definition for which the character was compiled. The character's version number is automatically set when you build it with the Microsoft Agent Character Editor.
