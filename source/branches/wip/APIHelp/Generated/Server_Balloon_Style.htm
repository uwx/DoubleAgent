Double Agent API

The Style Property

Description

Specifies the character's word balloon output style.

Syntax

Visual C++

balloon->get_Style (&Style)

balloon->put_Style (Style)

Visual C#

[Style = ]character.Balloon.Style

character.Balloon.Style = Style

Visual Basic .NET

[Style = ]character.Balloon.Style

character.Balloon.Style = Style

Visual Basic 6

[Style = ]character.Balloon.Style

character.Balloon.Style = Style

VBScript

[Style = ]character.Balloon.Style

character.Balloon.Style = Style
Part Description
An numeric value that represents the balloon's output style. The style value is a bit field that contains values from the BalloonStyleFlags Enumeration:

Members

0x00000001 = Enabled The word balloon is enabled.
0x00000002 = SizeToText The word balloon is automatically resized to fit the speech or thought text.
0x00000004 = AutoHide The word balloon is hidden automatically after a short delay.
0x00000008 = AutoPace The text in the word balloon is shown one word at a time.
0x00000010 = ShowPartialLines Partial lines are shown when the balloon text scrolls (for a word balloon that's not SizeToText).

Remarks

The defaults for the Style property are set when the character is compiled with the Microsoft Agent Character Editor.

Note: The Style property can be set even if the user has disabled word balloons in the Advanced Character Options window.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
