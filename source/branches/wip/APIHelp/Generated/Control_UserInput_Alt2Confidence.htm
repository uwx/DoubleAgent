Double Agent API

The Alt2Confidence Property

Description

Returns the speech recognition confidence scoring for the third-best matching Command.

Syntax

Visual Basic .NET

[Alt2Confidence = ]userinput.Alt2Confidence

Visual Basic 6

[Alt2Confidence = ]userinput.Alt2Confidence

VBScript

[Alt2Confidence = ]userinput.Alt2Confidence

Visual C#

[Alt2Confidence = ]userinput.Alt2Confidence

Visual C++

userinput->get_Alt2Confidence (&Alt2Confidence)

Part Description
Alt2Confidence The relative confidence scoring of third-best match, from -100 to 100 (0 if the Command was chosen from the pop-up menu)
