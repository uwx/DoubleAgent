Double Agent API

The BalloonEnabled Property

Description

Indicates if the word balloon for the Speak and Think methods is enabled.

Syntax

Visual C++

settings->get_BalloonEnabled (&BalloonEnabled)

Visual C#

[BalloonEnabled = ]server.Settings.BalloonEnabled

Visual Basic .NET

[BalloonEnabled = ]server.Settings.BalloonEnabled

Visual Basic 6

[BalloonEnabled = ]server.Settings.BalloonEnabled

VBScript

[BalloonEnabled = ]server.Settings.BalloonEnabled
Part Description
BalloonEnabled A boolean value that indicates if the word balloon is enabled.

Remarks

This property reflects the "Display spoken output in word balloon …" option on the Output page of the Advanced Character Options window.

Note: The property setting affects all characters and is read-only; only the user can set this property value.
