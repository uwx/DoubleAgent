Double Agent API

The TTSSpeed Property

Description

Returns the value of the text-to-speech rate configured by the user.

Syntax

Visual Basic .NET

[TTSSpeed = ]control.Settings.TTSSpeed

Visual Basic 6

[TTSSpeed = ]control.Settings.TTSSpeed

VBScript

[TTSSpeed = ]control.Settings.TTSSpeed

Visual C#

[TTSSpeed = ]control.Settings.TTSSpeed

Visual C++

settings->get_TTSSpeed (&TTSSpeed)

Part Description
TTSSpeed A numeric value that indicates the relative rate applied to text-to-speech. It ranges from 0 (slow) to 10 (fast). A value of 5 indicates no change to the speech engine's normal speaking speed.

Remarks

This property reflects the Speaking speed option on the Output page of the Advanced Character Options window.

Note: The property setting affects all characters and is read-only; only the user can set this property value.
