The ShowPartialLines Property

Description

Indicates if the word balloon shows partial lines when it scrolls.

Syntax

Visual Basic .NET

[ShowPartialLines = ]control.Characters ("CharacterID").Balloon.ShowPartialLines

control.Characters ("CharacterID").Balloon.ShowPartialLines = ShowPartialLines

Visual Basic 6

[ShowPartialLines = ]control.Characters ("CharacterID").Balloon.ShowPartialLines

control.Characters ("CharacterID").Balloon.ShowPartialLines = ShowPartialLines

VBScript

[ShowPartialLines = ]control.Characters ("CharacterID").Balloon.ShowPartialLines

control.Characters ("CharacterID").Balloon.ShowPartialLines = ShowPartialLines

Visual C#

[ShowPartialLines = ]control.Characters.this ["CharacterID"].Balloon.ShowPartialLines

control.Characters.this ["CharacterID"].Balloon.ShowPartialLines = ShowPartialLines

Visual C++

balloon->get_ShowPartialLines (&ShowPartialLines)

balloon->put_ShowPartialLines (ShowPartialLines)

Part Description
ShowPartialLines A boolean value that indicates if the ShowPartialLines style is enabled.

Remarks

When the ShowPartialLines style is set, the word balloon will show partial lines at the bottom as it scrolls its text. When it's not set, only full lines are shown. This style only applies when the SizeToText style is not set.
