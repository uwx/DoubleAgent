Double Agent API

The OriginalWidth Property

Description

Returns the character's default width.

Syntax

Visual C++

character->get_OriginalWidth (&OriginalWidth)

Visual C#

[OriginalWidth = ]character.OriginalWidth

Visual Basic .NET

[OriginalWidth = ]character.OriginalWidth

Visual Basic 6

[OriginalWidth = ]character.OriginalWidth

VBScript

[OriginalWidth = ]character.OriginalWidth
Part Description
OriginalWidth The character's image width as defined with the Microsoft Agent Character Editor.
