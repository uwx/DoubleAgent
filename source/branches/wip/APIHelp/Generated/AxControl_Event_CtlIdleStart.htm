Double Agent API

The CtlIdleStart Event

Description

Signalled when Double Agent sets a character to the Idling state.

Syntax

Visual Basic .NET

Event control_CtlIdleStart (ByVal Sender As Object, ByVal E As CtlIdleStartEvent)

Visual C#

delegate void control_CtlIdleStartEventHandler (object Sender, CtlIdleStartEvent E)
Part Description
Sender The source of the event.
E A CtlIdleStartEvent Object that contains the event data:
CharacterID
The unique identifier of the Idling character.

Remarks

Double Agent sends this event to all clients of the character.
