Double Agent API

The NumberOfLines Property

Description

Returns or sets the number of lines used for a character's word balloon.

Syntax

Visual Basic .NET

[NumberOfLines = ]control.Characters ("CharacterID").Balloon.NumberOfLines

control.Characters ("CharacterID").Balloon.NumberOfLines = NumberOfLines

Visual Basic 6

[NumberOfLines = ]control.Characters ("CharacterID").Balloon.NumberOfLines

control.Characters ("CharacterID").Balloon.NumberOfLines = NumberOfLines

VBScript

[NumberOfLines = ]control.Characters ("CharacterID").Balloon.NumberOfLines

control.Characters ("CharacterID").Balloon.NumberOfLines = NumberOfLines

Visual C#

[NumberOfLines = ]control.Characters ("CharacterID").Balloon.NumberOfLines

control.Characters ("CharacterID").Balloon.NumberOfLines = NumberOfLines

Visual C++

balloon->get_NumberOfLines (&NumberOfLines)

balloon->put_NumberOfLines (NumberOfLines)

Part Description
NumberOfLines A numeric value that specifies the number of lines in a fixed-size word balloon.

Remarks

The NumberOfLines property is only used if the word balloon doesn't have the AutoSize style.
