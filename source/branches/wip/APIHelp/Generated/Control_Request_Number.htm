Double Agent API

The Number Property

Description

Returns the error or cause of the Status code.

Syntax

Visual Basic .NET

[Number = ]request.Number

Visual Basic 6

[Number = ]request.Number

VBScript

[Number = ]request.Number

Visual C#

[Number = ]request.Number

Visual C++

request->get_Number (&Number)

Part Description
Number The error or cause of the Status code. If there is no error, this value is zero.

Remarks

For the values and meanings of the Number property, see Microsoft Agent Error Codes.
