Double Agent API

The IdleStart Event

Description

Signalled when Double Agent sets a character to the Idling state.

Syntax

Visual Basic .NET

Event control_IdleStart (ByVal CharacterID As String)

Visual Basic 6

Sub control_IdleStart (ByVal CharacterID As String)

VBScript

Sub control_IdleStart (ByVal CharacterID)

Visual C#

delegate void control_IdleStartEventHandler (string CharacterID)

Visual C++

void control_IdleStart (LPCTSTR CharacterID)

Part Description
CharacterID The unique identifier of the Idling character.

Remarks

Double Agent sends this event to all clients of the character.
