Double Agent API

The DefaultCommand Property

Description

Returns or sets the default command of the Commands collection.

Syntax

Visual Basic .NET

[DefaultCommand = ]control.Characters ("CharacterID").Commands.DefaultCommand

control.Characters ("CharacterID").Commands.DefaultCommand = DefaultCommand

Visual Basic 6

[DefaultCommand = ]control.Characters ("CharacterID").Commands.DefaultCommand

control.Characters ("CharacterID").Commands.DefaultCommand = DefaultCommand

VBScript

[DefaultCommand = ]control.Characters ("CharacterID").Commands.DefaultCommand

control.Characters ("CharacterID").Commands.DefaultCommand = DefaultCommand

Visual C#

[DefaultCommand = ]control.Characters.this ["CharacterID"].Commands.DefaultCommand

control.Characters.this ["CharacterID"].Commands.DefaultCommand = DefaultCommand

Visual C++

commands.get_DefaultCommand (&DefaultCommand)

commands.put_DefaultCommand (DefaultCommand)

Part Description
DefaultCommand The name of the Command that's shown bold in the pop-up menu (an empty string if none).

Remarks

This property allows you to set a Command in your Commands collection as the default command, rendering it bold. This does not actually change command handling or double-click events.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
