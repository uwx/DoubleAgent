Double Agent API

The DefaultCommand Property

Description

Returns or sets the default command of the Commands object.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Commands.DefaultCommand[ = String]
Visual Basic 6
control.Characters ("CharacterID").Commands.DefaultCommand[ = String]
VBScript
control.Characters ("CharacterID").Commands.DefaultCommand[ = String]
Visual C#
control.Characters ("CharacterID").Commands.DefaultCommand[ = string]
Visual C++

commands.get_DefaultCommand (&BSTR)

commands.put_DefaultCommand (BSTR)

Remarks

This property enables you to set a Command in your Commands collection as the default command, rendering it bold. This does not actually change command handling or double-click events.

This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
