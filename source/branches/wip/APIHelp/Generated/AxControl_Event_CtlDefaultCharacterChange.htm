Double Agent API

The CtlDefaultCharacterChange Event

Description

Signalled when the user changes the default character.

Syntax

Visual Basic .NET

Event control_CtlDefaultCharacterChange (ByVal Sender As Object, ByVal E As CtlDefaultCharacterChangeEvent)

Visual C#

delegate void control_CtlDefaultCharacterChangeEventHandler (object Sender, CtlDefaultCharacterChangeEvent E)
Part Description
Sender The source of the event.
E A CtlDefaultCharacterChangeEvent Object that contains the event data:
CharGUID
The unique identifier of the new default character.

Remarks

This event indicates that the user has changed the character assigned as the user's default character. Double Agent sends this only to clients that have loaded the default character.

When the new character appears, it assumes the same size as any already loaded instance of the character, or the previous default character (in that order).
