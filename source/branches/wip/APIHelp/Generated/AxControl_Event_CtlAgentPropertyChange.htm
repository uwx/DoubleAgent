Double Agent API

The CtlAgentPropertyChange Event

Description

Signalled when the user changes a property in the Advanced Character Options window.

Syntax

Visual Basic .NET

Event control_CtlAgentPropertyChange (ByVal Sender As Object, ByVal E As EventArgs)

Visual C#

delegate void control_CtlAgentPropertyChangeEventHandler (object Sender, EventArgs E)
Part Description
Sender The source of the event.
E An EventArgs object that contains no event data.

Remarks

This event indicates that the user has changed and applied any property in the Advanced Character Options window. You can use the Settings Object to query the specific property settings.
