Double Agent API

The TTSEngines Property

Description

Returns a collection of Text-To-Speech Engine Objects for all available text-to-speech engines.

Syntax

Visual Basic .NET

[TTSEngines = ]control.TTSEngines

Visual C#

[TTSEngines = ]control.TTSEngines
Part Description
TTSEngines A TTSEngines Collection that includes all available text-to-speech engines.

Remarks

This property allows you to query available text-to-speech engines without loading a character.
