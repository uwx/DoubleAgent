Double Agent API

The Width Property

Description

Returns or sets the width of the the specified character's image.

Syntax

Visual Basic .NET

[Width = ]control.Characters ("CharacterID").Width

control.Characters ("CharacterID").Width = Width

Visual Basic 6

[Width = ]control.Characters ("CharacterID").Width

control.Characters ("CharacterID").Width = Width

VBScript

[Width = ]control.Characters ("CharacterID").Width

control.Characters ("CharacterID").Width = Width

Visual C#

[Width = ]control.Characters ("CharacterID").Width

control.Characters ("CharacterID").Width = Width

Visual C++

character.get_Width (&Width)

character.put_Width (Width)

Part Description
Width A numeric value that specifies the character's image width.

Remarks

The Width property is always expressed in pixels. This property's setting applies to all clients of the character.

Even though the character appears in an irregularly shaped region window, the location of the character is based on the external dimensions of the rectangular animation frame used when the character was compiled with the Microsoft Agent Character Editor.
