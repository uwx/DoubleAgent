Double Agent API

The Width Property

Description

Specifies the width of the character's image.

Syntax

Visual Basic .NET

[Width = ]control.Characters ("CharacterID").Width

control.Characters ("CharacterID").Width = Width

Visual Basic 6

[Width = ]control.Characters ("CharacterID").Width

control.Characters ("CharacterID").Width = Width

VBScript

[Width = ]control.Characters ("CharacterID").Width

control.Characters ("CharacterID").Width = Width

Visual C#

[Width = ]control.Characters.this ["CharacterID"].Width

control.Characters.this ["CharacterID"].Width = Width

Visual C++

character->get_Width (&Width)

character->put_Width (Width)
Part Description
Width A numeric value that specifies the width of the character's image.

Remarks

The Width property is always expressed in pixels.

Even though the character appears in an irregularly shaped region window, the dimensions of the character are based on the external dimensions of the rectangular animation frame used when the character was compiled with the Microsoft Agent Character Editor.

Note: This property applies to all clients of the character.
