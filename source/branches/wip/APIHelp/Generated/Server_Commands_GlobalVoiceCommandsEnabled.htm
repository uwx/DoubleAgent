Double Agent API

The GlobalVoiceCommandsEnabled Property

Description

Specifies if pre-defined commands appear in the Voice Commands window.

Syntax

Visual C++

commands->get_GlobalVoiceCommandsEnabled (&Enabled)

commands->put_GlobalVoiceCommandsEnabled (Enabled)

Visual C#

[Enabled = ]character.Commands.GlobalVoiceCommandsEnabled

character.Commands.GlobalVoiceCommandsEnabled = Enabled

Visual Basic .NET

[Enabled = ]character.Commands.GlobalVoiceCommandsEnabled

character.Commands.GlobalVoiceCommandsEnabled = Enabled

Visual Basic 6

[Enabled = ]character.Commands.GlobalVoiceCommandsEnabled

character.Commands.GlobalVoiceCommandsEnabled = Enabled

VBScript

[Enabled = ]character.Commands.GlobalVoiceCommandsEnabled

character.Commands.GlobalVoiceCommandsEnabled = Enabled
Part Description
Enabled A boolean value that indicates if global voice commands are enabled.

Remarks

Double Agent automatically adds voice commands (grammar) for opening and closing the Voice Commands window and for showing and hiding the character. If you set GlobalVoiceCommandsEnabled to False, Agent disables any voice parameters for these voice commands as well as the voice commands for the Caption of other clients' Commands Collection. This allows you to eliminate these from your client's current active grammar. However, because this potentially blocks voice access to other clients, reset this property to True after processing the user's voice input.

Disabling the property does not affect the character's Pop-up Menu. The global commands added by Double Agent will still appear; you cannot remove them from the Pop-up Menu.
