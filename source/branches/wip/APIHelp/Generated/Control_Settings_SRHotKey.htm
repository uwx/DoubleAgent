Double Agent API

The SRHotKey Property

Description

Returns the user's current setting for the Listening key.

Syntax

Visual Basic .NET

[SRHotKey = ]control.Settings.SRHotKey

Visual Basic 6

[SRHotKey = ]control.Settings.SRHotKey

VBScript

[SRHotKey = ]control.Settings.SRHotKey

Visual C#

[SRHotKey = ]control.Settings.SRHotKey

Visual C++

settings->get_SRHotKey (&SRHotKey)
Part Description
SRHotKey A string value that describes the current Listening key.

Remarks

This property reflects the "When you press the … key" option on the Speech Input page of the Advanced Character Options window.

Note: The property setting affects all characters and is read-only; only the user can set this property value.
