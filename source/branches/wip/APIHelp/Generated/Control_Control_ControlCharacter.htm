Double Agent API

The ControlCharacter Property

Description

Specifies the control's contained Character.

Syntax

Visual Basic .NET

[ControlCharacter = ]control.ControlCharacter

control.ControlCharacter = ControlCharacter|Nothing

Visual Basic 6

[Set ControlCharacter = ]control.ControlCharacter

Set control.ControlCharacter = ControlCharacter|Nothing

VBScript

[Set ControlCharacter = ]control.ControlCharacter

Set control.ControlCharacter = ControlCharacter|Nothing

Visual C#

[ControlCharacter = ]control.ControlCharacter

control.ControlCharacter = ControlCharacter|null

Visual C++

control->get_ControlCharacter (&ControlCharacter)

control->put_ControlCharacter (ControlCharacter|NULL)
Part Description
ControlCharacter The Character Object object that's contained within the visible area of the control, or an empty reference (Nothing).

Remarks

Setting the ControlCharacter property to a Character Object changes the control from stand-alone mode to contained mode. Setting ControlCharacter to an empty reference changes the control from contained mode back to stand-alone mode.

Normally, the control isn't visible at runtime, but when it has a ControlCharacter it will become visible as a child window of its containing form. This is the only time the ActiveX® appearence properties, such as BackColor and BorderColor, are used.

Note: You cannot set the ControlCharacter property if the control is Connected to the server.
