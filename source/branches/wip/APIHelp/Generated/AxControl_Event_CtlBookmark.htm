Double Agent API

The CtlBookmark Event

Description

Signalled when a bookmark defined by your application in a speech text string is reached.

Syntax

Visual Basic .NET

Event control_CtlBookmark (ByVal Sender As Object, ByVal E As CtlBookmarkEvent)

Visual C#

delegate void control_CtlBookmarkEventHandler (object Sender, CtlBookmarkEvent E)
Part Description
Sender The source of the event.
E A CtlBookmarkEvent Object that contains the event data:
BookmarkID
The bookmark's unique identifier.

Remarks

To specify a bookmark event, use the Speak method with a Mrk tag in your supplied text. For more information about tags, see Speech Output Tags.
