Double Agent API

The Description Property

Description

Specifies a string that specifies the description for the specified character.

Syntax

Visual C++

character->get_Description (&Description)

character->put_Description (Description)

Visual C#

[Description = ]character.Description

character.Description = "Description"

Visual Basic .NET

[Description = ]character.Description

character.Description = "Description"

Visual Basic 6

[Description = ]character.Description

character.Description = "Description"

VBScript

[Description = ]character.Description

character.Description = "Description"
Part Description
Description A string value corresponding to the character's description (in the current language setting).

Remarks

A character's Description may depend on the character's LanguageID setting. A character's description in one language may be different or use different characters than in another. The character's default Description for a specific language is defined when the character is compiled with the Microsoft Agent Character Editor.

Note: The Description property is optional and may not be supplied for all characters.

Note: This property applies to all clients of the character.
