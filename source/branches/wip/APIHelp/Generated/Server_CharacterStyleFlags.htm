Double Agent API

The CharacterStyleFlags Enumeration

Description

Specifies settings that affect a character's appearence and behaviour.

Members

0x00000001 = SoundEffects The animation sound effects are played.
0x00000002 = IdleEnabled The character enters idle mode and shows it's idling animations when it's not doing anything else.
0x00000004 = AutoPopupMenu A right-click on the character or it's notification icon automatically shows a pop-up menu.
0x00000008 = IconShown The character has an icon in the taskbar notification area.
0x00000020 = SmoothEdges Image smoothing (antialiasing) is applied to the edges of the character's animations.
0x00000030 = Smoothed Image smoothing (antialiasing) is applied to the character's animations.
