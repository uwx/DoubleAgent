Double Agent API

The Item Property

Description

Returns a Command Object in the collection.

Syntax

Visual Basic .NET

[Command = ]control.Characters ("CharacterID").Commands (Name)

Visual Basic 6

[Set Command = ]control.Characters ("CharacterID").Commands (Name)

VBScript

[Set Command = ]control.Characters ("CharacterID").Commands (Name)

Visual C#

[Command = ]control.Characters.this ["CharacterID"].Commands.this [Name]

Visual C++

commands->get_Item (Name, &Command)
Part Description
Name A string value that specifies the Name given to the command when it was defined.
Command The Command Object with the specified Name.
