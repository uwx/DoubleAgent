Double Agent API

The GetSize Method

Description

Returns the current size of the Advanced Character Options window.

Syntax

Visual C++

propertysheet->GetSize (&Width, &Height)

Visual C#

server.PropertySheet.GetSize (out Width, out Height)

Visual Basic .NET

server.PropertySheet.GetSize (Width, Height)

Visual Basic 6

server.PropertySheet.GetSize Width, Height

VBScript

server.PropertySheet.GetSize Width, Height
Part Description
Width A numeric value that indicates the width, in pixels, of the Advanced Character Options window.
Height A numeric value that indicates the height, in pixels, of the Advanced Character Options window.
