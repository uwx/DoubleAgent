Double Agent API

The IconShown Property

Description

Specifies if the character's notification icon is shown in the Windows notification area.

Syntax

Visual Basic .NET

[IconShown = ]control.Characters ("CharacterID").IconShown

control.Characters ("CharacterID").IconShown = IconShown

Visual Basic 6

[IconShown = ]control.Characters ("CharacterID").IconShown

control.Characters ("CharacterID").IconShown = IconShown

VBScript

[IconShown = ]control.Characters ("CharacterID").IconShown

control.Characters ("CharacterID").IconShown = IconShown

Visual C#

[IconShown =] control.Characters.this ["CharacterID"].IconShown

control.Characters.this ["CharacterID"].IconShown = IconShown

Visual C++

character->get_IconShown (&IconShown)

character->put_IconShown (IconShown)
Part Description
IconShown A boolean value that indicates if the icon is shown.

Remarks

This property only specifies if the notification icon is added to the Windows notification area. The user decides whether it's visible or hidden. See the IconIdentity property for more details.

Note: This property has no effect if the character doesn't have a notification icon.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
