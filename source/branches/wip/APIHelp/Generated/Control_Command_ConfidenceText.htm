Double Agent API

The ConfidenceText Property

Description

Returns or sets the client's ConfidenceText that appears in the Listening Tip.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Commands("Name").ConfidenceText[ = String]
Visual Basic 6
control.Characters ("CharacterID").Commands("Name").ConfidenceText[ = String]
VBScript
control.Characters ("CharacterID").Commands("Name").ConfidenceText[ = String]
Visual C#
control.Characters ("CharacterID").Commands("Name").ConfidenceText[ = string]
Visual C++

command.get_ConfidenceText (&BSTR)

command.put_ConfidenceText (BSTR)

Remarks

When the returned confidence value of the best match (UserInput.Confidence) does not exceed the Confidence setting, Double Agent displays the text supplied in ConfidenceText in the Listening Tip.
