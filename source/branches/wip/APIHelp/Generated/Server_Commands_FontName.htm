Double Agent API

The FontName Property

Description

Specifies the font used in the Voice Commands window.

Syntax

Visual C++

commands->get_FontName (&FontName)

commands->put_FontName (FontName)

Visual C#

[FontName = ]character.Commands.FontName

character.Commands.FontName = FontName

Visual Basic .NET

[FontName = ]character.Commands.FontName

character.Commands.FontName = FontName

Visual Basic 6

[FontName = ]character.Commands.FontName

character.Commands.FontName = FontName

VBScript

[FontName = ]character.Commands.FontName

character.Commands.FontName = FontName
Part Description
FontName A string value specifying the font name.

Remarks

The FontName property defines the font used to display text in the Voice Commands window. The default value for the font setting is based on the menu font for the character's LanguageID or, if that's not set, the user default language.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
