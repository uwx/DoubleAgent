Double Agent API

The Filter Property

Description

Specifies the filtering criteria applied to the FilePaths and DefaultSearchPath properties.

Syntax

Visual C++

characterfiles->get_Filter (&Filter)

characterfiles->put_Filter (Filter)

Visual C#

[Filter = ]server.CharacterFiles.Filter

server.CharacterFiles.Filter = Filter

Visual Basic .NET

[Filter = ]server.CharacterFiles.Filter

server.CharacterFiles.Filter = Filter

Visual Basic 6

[Filter = ]server.CharacterFiles.Filter

server.CharacterFiles.Filter = Filter

VBScript

[Filter = ]server.CharacterFiles.Filter

server.CharacterFiles.Filter = Filter
Part Description
Filter A numeric bit-field that contains any of the following FilesFilterFlags values:
0x00000001 = PathDoubleAgent Indicates if the Double Agent install location should be included in the DefaultSearchPath.
0x00000002 = PathMsAgent Indicates if the Microsoft Agent install location should be included in the DefaultSearchPath.
0x00000004 = PathMsOffice Indicates if the Microsoft Office install location should be included in the DefaultSearchPath.
0x00000010 = ExcludeNonSpeaking Indicates if the FilePaths property should exclude characters that are not designed to speak.
0x00000020 = ExcludeSpeaking Indicates if the FilePaths property should exclude characters that are designed to speak.
0x00010000 = NoValidateVersion Indicates if the the FilePaths property should include all character files, regardless of version.

Remarks

The path for Double Agent characters is the Chars subdirectory of the Double Agent install location. Typically, this would be C:\Program Files\Double Agent\Chars.

The path for Microsoft Agent characters is the Chars subdirectory of the Microsoft Agent install location. Typically, this would be C:\Windows\MsAgent\Chars.

Typically, Microsoft Office characters would be in C:\Program Files\Microsoft Office\Office10.

Note: Only older versions of Microsoft Office include Agent character files.
