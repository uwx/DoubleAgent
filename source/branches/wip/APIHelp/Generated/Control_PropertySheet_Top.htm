Double Agent API

The Top Property

Description

Returns an integer value specifying the current top edge, in pixels, of the Double Agent property sheet window.

Syntax

Visual Basic .NET

[Top = ]control.PropertySheet.Top

control.PropertySheet.Top = Top

Visual Basic 6

[Top = ]control.PropertySheet.Top

control.PropertySheet.Top = Top

VBScript

[Top = ]control.PropertySheet.Top

control.PropertySheet.Top = Top

Visual C#

[Top = ]control.PropertySheet.Top

control.PropertySheet.Top = Top

Visual C++

propertysheet->get_Top (&Top)

propertysheet->put_Top (Top)

Part Description
Top

Remarks

Double Agent displays the window based on the location set by the user.
