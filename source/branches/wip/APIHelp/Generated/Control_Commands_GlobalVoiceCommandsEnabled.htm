Double Agent API

The GlobalVoiceCommandsEnabled Property

Description

Returns or sets whether pre-defined commands appear in the Voice Commands window.

Syntax

Visual Basic .NET

[Enabled = ]control.Characters ("CharacterID").Commands.GlobalVoiceCommandsEnabled

control.Characters ("CharacterID").Commands.GlobalVoiceCommandsEnabled = Enabled

Visual Basic 6

[Enabled = ]control.Characters ("CharacterID").Commands.GlobalVoiceCommandsEnabled

control.Characters ("CharacterID").Commands.GlobalVoiceCommandsEnabled = Enabled

VBScript

[Enabled = ]control.Characters ("CharacterID").Commands.GlobalVoiceCommandsEnabled

control.Characters ("CharacterID").Commands.GlobalVoiceCommandsEnabled = Enabled

Visual C#

[Enabled = ]control.Characters ("CharacterID").Commands.GlobalVoiceCommandsEnabled

control.Characters ("CharacterID").Commands.GlobalVoiceCommandsEnabled = Enabled

Visual C++

commands.get_GlobalVoiceCommandsEnabled (&Enabled)

commands.put_GlobalVoiceCommandsEnabled (Enabled)

Part Description
Enabled A boolean value that indicates if global voice commands are enabled.

Remarks

Double Agent automatically adds voice commands (grammar) for opening and closing the Voice Commands window and for showing and hiding the character. If you set GlobalVoiceCommandsEnabled to False, Agent disables any voice parameters for these voide commands as well as the voice commands for the Caption of other clients' Commands objects. This allows you to eliminate these from your client's current active grammar. However, because this potentially blocks voice access to other clients, reset this property to True after processing the user's voice input.

Disabling the property does not affect the character's pop-up menu. The global commands added by Double Agent will still appear; you cannot remove them from the pop-up menu.
