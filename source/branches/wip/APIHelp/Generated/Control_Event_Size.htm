Double Agent API

The Size Event

Description

Signalled when a character's size changes.

Syntax

Visual Basic .NET

Event control_Size (ByVal CharacterID As String, ByVal Width As Short, ByVal Height As Short)

Visual Basic 6

Sub control_Size (ByVal CharacterID As String, ByVal Width As Integer, ByVal Height As Integer)

VBScript

Sub control_Size (ByVal CharacterID, ByVal Width, ByVal Height)

Visual C#

delegate void control_SizeEventHandler (string CharacterID, short Width, short Height)

Visual C++

void control_Size (LPCTSTR CharacterID, short Width, short Height)
Part Description
CharacterID The unique identifier of the character that was resized.
Width The character's new width, in pixels.
Height The character's new height, in pixels.

Remarks

This event is signalled when an application changes the size of a character. This event is sent only to the clients of the character (applications that have loaded the character).
