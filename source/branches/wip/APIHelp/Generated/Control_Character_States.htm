Double Agent API

The States Property

Description

Returns the names of the character's state animations.

Syntax

Visual Basic .NET

[States = ]control.Characters ("CharacterID").States

Visual Basic 6

[States = ]control.Characters ("CharacterID").States

VBScript

[States = ]control.Characters ("CharacterID").States

Visual C#

[States = ]control.Characters.this ["CharacterID"].States

Visual C++

character->get_States (&States)
Part Description
States An array of string values that name all of the state animations the character can play.

Remarks

For .ACF characters, the array contains all the states defined for the character, not just the ones that have been retrieved with the Prepare method.

Most characters include the following states:

  • "Gesturing" or "GesturingDown", "GesturingLeft", "GesturingRight" and "GesturingUp"
  • "Hiding"
  • "Hearing"
  • "Idling" or "IdlingLevel1", "IdlingLevel2" and "IdlingLevel3"
  • "Listening"
  • "Moving" or "MovingDown", "MovingLeft", "MovingRight" and "MovingUp"
  • "Showing"
  • "Speaking"
