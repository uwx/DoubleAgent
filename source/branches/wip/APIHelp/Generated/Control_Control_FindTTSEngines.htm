Double Agent API

The FindTTSEngines Method

Description

Retrieves a collection of Text-To-Speech Engine objects for a specific language and gender.

Syntax

Visual Basic .NET

[TTSEngines = ]control.FindTTSEngines (LanguageID, Gender)

Visual Basic 6

[Set TTSEngines = ]control.FindTTSEngines ([LanguageID][, Gender])

VBScript

[Set TTSEngines = ]control.FindTTSEngines ([LanguageID][, Gender])

Visual C#

[TTSEngines = ]control.FindTTSEngines (LanguageID, Gender)

Visual C++

control->FindTTSEngines (LanguageID, Gender, &TTSEngines)

Part Description
LanguageID A numeric value that specifies the language to match, or zero for any language.
Gender A numeric value, or a member of the SpeechGenderType enumeration, that specifies the gender to match:
1 = Female A female voice.
2 = Male A male voice.
0 = Neutral Either a female or male voice.
TTSEngines A TTSEngines Collection that matches the the specified LanguageID and Gender.

Remarks

This method allows you to query available text-to-speech engines without loading a character.
