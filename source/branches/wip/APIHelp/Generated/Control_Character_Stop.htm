Double Agent API

The Stop Method

Description

Stops an animation.

Syntax

Visual Basic .NET

control.Characters ("CharacterID").Stop (Request)

Visual Basic 6

control.Characters ("CharacterID").Stop Request

VBScript

control.Characters ("CharacterID").Stop Request

Visual C#

control.Characters.this ["CharacterID"].Stop (Request)

Visual C++

character->Stop (Request)
Part Description
Request A Request Object specifying a particular animation call, or a null reference to stop all animations.

Remarks

To specify the Request parameter, you must create a variable and assign the animation request you want to stop. If you don't set the Request parameter, Double Agent stops all animations for the character, including queued Prepare calls, and clears its animation queue (unless the character is currently playing its Hiding or Showing animation).

Note: This method is performed immediately and does not generate a Request Object.

Note: This method does not stop non-queued Prepare calls.

To stop a specific animation or Prepare call, declare an object variable and assign your animation request to that variable: 

Dim MyRequest
Dim Genie

control.Characters.Load "Genie", "Genie.acs"

Set Genie = control.Characters ("Genie")
Genie.Show

'This animation will never play
Set MyRequest = Genie.Play ("Greet")

Genie.Stop MyRequest
