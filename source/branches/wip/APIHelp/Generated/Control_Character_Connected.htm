Double Agent API

The Connected Property

Description

Indicates if the Character Object is connected to the Double Agent server.

Syntax

Visual Basic .NET

[Connected = ]control.Characters ("CharacterID").Connected

Visual Basic 6

[Connected = ]control.Characters ("CharacterID").Connected

VBScript

[Connected = ]control.Characters ("CharacterID").Connected

Visual C#

[Connected = ]control.Characters.this ["CharacterID"].Connected

Visual C++

character->get_Connected (&Connected)
Part Description
Connected A boolean value that indicates if the character is running on the server or locally.

Remarks

When the control is being used in stand-alone mode, any Character you load will be Connected. When the control is being used in stand-alone mode, any Character you load will not be Connected.
