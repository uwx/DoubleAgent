Double Agent API

The GetPosition Method

Description

Returns the current position of the Advanced Character Options window.

Syntax

Visual C++

propertysheet->GetPosition (&Left, &Top)

Visual C#

server.PropertySheet.GetPosition (out Left, out Top)

Visual Basic .NET

server.PropertySheet.GetPosition (Left, Top)

Visual Basic 6

server.PropertySheet.GetPosition Left, Top

VBScript

server.PropertySheet.GetPosition Left, Top
Part Description
Left A numeric value that specifies the left edge, in pixels, of the Advanced Character Options window.
Top A numeric value that specifies the top edge, in pixels, of the Advanced Character Options window.

Remarks

Double Agent remembers and reuses the window location set by the user.
