Double Agent API

The Enabled Property

Description

Indicates if the word balloon is enabled for a character.

Syntax

Visual C++

balloon->get_Enabled (&Enabled)

Visual C#

[Enabled = ]character.Balloon.Enabled

Visual Basic .NET

[Enabled = ]character.Balloon.Enabled

Visual Basic 6

[Enabled = ]character.Balloon.Enabled

VBScript

[Enabled = ]character.Balloon.Enabled
Part Description
Enabled A boolean value that indicates if the Enabled style is set.

Remarks

When the Enabled style is set, the word balloon appears when the Speak or Think method is used, unless the user overrides this setting in the Advanced Character Options window. When it's not set, the word balloon does not appear.

The default enabled style is set as part of a character's definition when the character is compiled in the Microsoft Agent Character Editor. If a character is defined to not support the word balloon, this property will always be False for the character.

The user can override the enabled style for all characters in the Advanced Character Options window.
