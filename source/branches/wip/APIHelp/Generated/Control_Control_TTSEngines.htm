Double Agent API

The TTSEngines Property

Description

Retrieves a collection of Text-To-Speech Engine objects for all available text-to-speech engines.

Syntax

Visual Basic .NET

[TTSEngines = ]control.TTSEngines

Visual Basic 6

[Set SREngines = ]control.TTSEngines

VBScript

[Set SREngines = ]control.TTSEngines

Visual C#

[TTSEngines = ]control.TTSEngines

Visual C++

control->get_TTSEngines (&TTSEngines)

Part Description
TTSEngines A TTSEngines Collection that includes all available text-to-speech engines.

Remarks

This property allows you to query available text-to-speech engines without loading a character.
