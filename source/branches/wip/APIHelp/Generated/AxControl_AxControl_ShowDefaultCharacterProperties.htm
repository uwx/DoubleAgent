Double Agent API

The ShowDefaultCharacterProperties Method

Description

Shows the Gallery page of the Advanced Character Options window.

Syntax

Visual Basic .NET

control.ShowDefaultCharacterProperties (X, Y)

Visual C#

control.ShowDefaultCharacterProperties (X, Y)
Part Description
X The horizontal position, in pixels, of the Advanced Character Options window. This parameter is optional. For programming languages that don't support optional parameters, a negative value will be ignored.
Y The vertical position, in pixels, of the Advanced Character Options window. This parameter is optional. For programming languages that don't support optional parameters, a negative value will be ignored.

Remarks

This method is obsolete, but is still supported for backward compatibility. Calling it shows the Advanced Character Options window. New applications should use the PropertySheet Object instead.

If you don't specify the X and Y coordinates (or use a negative value for programming languages that don't support optional parameters), the window appears at the last location it was displayed.
