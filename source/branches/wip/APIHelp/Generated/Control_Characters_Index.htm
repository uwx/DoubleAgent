Double Agent API

The Index Property

Description

Returns a Character Object in the collection.

Syntax

Visual Basic .NET

[Character = ]control.Characters.Index (Index)

Visual Basic 6

[Set Character = ]control.Characters.Index (Index)

VBScript

[Set Character = ]control.Characters.Index (Index)

Visual C#

[Character = ]control.Characters.get_Index (Index)

Visual C++

characters->get_Index (Index, &Character)
Part Description
Index A numeric value that specifies the zero-based index of the item to retrieve.
Character The Character Object at the spedified Index.

Remarks

When you want to use the Characters Collection as an ordered collection, you can use the Index property instead of the Item property.
