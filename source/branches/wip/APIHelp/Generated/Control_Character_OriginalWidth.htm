Double Agent API

The OriginalWidth Property

Description

Returns an integer value that specifies the character's original width.

Syntax

Visual Basic .NET

[OriginalWidth = ]control.Characters ("CharacterID").OriginalWidth

Visual Basic 6

[OriginalWidth = ]control.Characters ("CharacterID").OriginalWidth

VBScript

[OriginalWidth = ]control.Characters ("CharacterID").OriginalWidth

Visual C#

[OriginalWidth = ]control.Characters ("CharacterID").OriginalWidth

Visual C++

character.get_OriginalWidth (&OriginalWidth)

Part Description
OriginalWidth This property returns the character's frame width as built with the Microsoft Agent Character Editor.

Remarks

This property returns the character's frame width as built with the Microsoft Agent Character Editor.
