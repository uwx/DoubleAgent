Double Agent API

The Command Event

Description

Signalled when the user chooses a (client's) command.

Syntax

Visual Basic .NET

Event control_Command (ByVal UserInput As DoubleAgent.Control.UserInput)

Visual Basic 6

Sub control_Command (ByVal UserInput As DaCtlUserInput)

VBScript

Sub control_Command (ByVal UserInput)

Visual C#

delegate void control_CommandEventHandler (DoubleAgent.Control.UserInput UserInput)

Visual C++

void control_Command (IDaCtlUserInput * UserInput)
Part Description
UserInput A UserInput Object that identifies the chosen command.

Remarks

Double Agent sends this event when your application is input-active and the user chooses a command from the character's pop-up menu, or by spoken input.
