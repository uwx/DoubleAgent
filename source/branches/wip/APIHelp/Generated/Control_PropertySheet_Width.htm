Double Agent API

The Width Property

Description

Returns current width, in pixels, of the Advanced Character Options window.

Syntax

Visual Basic .NET

[Width = ]control.PropertySheet.Width

Visual Basic 6

[Width = ]control.PropertySheet.Width

VBScript

[Width = ]control.PropertySheet.Width

Visual C#

[Width = ]control.PropertySheet.Width

Visual C++

propertysheet->get_Width (&Width)

Part Description
Width A numeric value that indicates the width, in pixels, of the Advanced Character Options window.
