Double Agent API

The Width Property

Description

Returns a numeric value specifying the current width, in pixels, of the Double Agent property sheet window.

Syntax

Visual Basic .NET

[Width = ]control.PropertySheet.Width

Visual Basic 6

[Width = ]control.PropertySheet.Width

VBScript

[Width = ]control.PropertySheet.Width

Visual C#

[Width = ]control.PropertySheet.Width

Visual C++

propertysheet->get_Width (&Width)

Part Description
Width

Remarks

Double Agent displays the window based on the location set by the user.
