Double Agent API

The Visible Property

Description

Specifies if the Command is visible in the character's pop-up menu.

Syntax

Visual Basic .NET

[Visible] = control.Characters ("CharacterID").Commands ("Name").Visible

control.Characters ("CharacterID").Commands ("Name").Visible[ = Visible]

Visual Basic 6

[Visible] = control.Characters ("CharacterID").Commands ("Name").Visible

control.Characters ("CharacterID").Commands ("Name").Visible[ = Visible]

VBScript

[Visible] = control.Characters ("CharacterID").Commands ("Name").Visible

control.Characters ("CharacterID").Commands ("Name").Visible[ = Visible]

Visual C#

[Visible] = control.Characters.this ["CharacterID"].Commands.this ["Name"].Visible

control.Characters.this ["CharacterID"].Commands.this ["Name"].Visible[ = Visible]

Visual C++

command->get_Visible (&Visible)

command->put_Visible (Visible)
Part Description
Visible A boolean value that indicates if the command is visible.

Remarks

Set this property to False when you want to include voice input for your command without having it appear in the pop-up menu for the character. If you set a Command Object's Caption property to the empty string (""), the caption text will not appear in the pop-up menu (for example, as a blank line), regardless of its Visible property setting.

The Visible property of a Command Object's parent Commands Collection does not affect the Visible property of the Command.
