Double Agent API

The Visible Property

Description

Specifies if the Commands Collection appears in the character's pop-up menu.

Syntax

Visual C++

commands->get_Visible (&Visible)

commands->put_Visible (Visible)

Visual C#

[Visible = ]character.Commands.Visible

character.Commands.Visible = Visible

Visual Basic .NET

[Visible = ]character.Commands.Visible

character.Commands.Visible = Visible

Visual Basic 6

[Visible = ]character.Commands.Visible

character.Commands.Visible = Visible

VBScript

[Visible = ]character.Commands.Visible

character.Commands.Visible = Visible
Part Description
Visible A boolean value that indiciates if the Commands Caption is visible.

Remarks

For the caption to appear in the character's pop-up menu when your application is not the input-active client, this property must be set to True and the Caption property set for your Commands Collection. In addition, this property must be set to True for commands in your collection to appear in the pop-up menu when your application is input-active.
