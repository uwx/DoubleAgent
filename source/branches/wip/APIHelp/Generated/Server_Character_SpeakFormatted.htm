The SpeakFormatted Method

Description

Speaks the specified text.

Syntax

Visual C++

character->SpeakFormatted (FormattedText, &RequestID)

Visual C#

character.SpeakFormatted (FormattedText, out RequestID)

Visual Basic .NET

character.SpeakFormatted (FormattedText, RequestID)

Visual Basic 6

character.SpeakFormatted FormattedText[, RequestID]

VBScript

character.SpeakFormatted FormattedText[, RequestID]
Part Description
FormattedText A FormattedText Object object containing the text to be spoken.
RequestID An optional unique numeric value your application can use to track the method's status.

Remarks

This method is the same as the Speak method, except that the input text is already parsed and formatted, and using this method triggers the SpeechStart, SpeechEnd, and SpeechWord events.

Calling the Speak method with a specific text string is equivalent to:

  1. calling NewFormattedText to get a FormattedText object;
  2. calling the FormattedText object's Parse method with the same text string;
  3. calling SpeakFormatted with the FormattedText object.
