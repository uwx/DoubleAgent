Double Agent API

The AutoPopupMenu Property

Description

Returns or sets whether right-clicking the character or its taskbar icon automatically displays the character's pop-up menu.

Syntax

Visual Basic .NET

[Enabled = ]control.Characters ("CharacterID").AutoPopupMenu

control.Characters ("CharacterID").AutoPopupMenu = Enabled

Visual Basic 6

[Enabled = ]control.Characters ("CharacterID").AutoPopupMenu

control.Characters ("CharacterID").AutoPopupMenu = Enabled

VBScript

[Enabled = ]control.Characters ("CharacterID").AutoPopupMenu

control.Characters ("CharacterID").AutoPopupMenu = Enabled

Visual C#

[Enabled = ]control.Characters ("CharacterID").AutoPopupMenu

control.Characters ("CharacterID").AutoPopupMenu = Enabled

Visual C++

character.get_AutoPopupMenu (&Enabled)

character.put_AutoPopupMenu (Enabled)

Part Description
Enabled A boolean value specifying whether Double Agent automatically displays the character's pop-up menu on right-click.

Remarks

By setting this property to False, you can create your own menu-handling behavior. To display the menu after setting this property to False, use the ShowPopupMenu method.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
