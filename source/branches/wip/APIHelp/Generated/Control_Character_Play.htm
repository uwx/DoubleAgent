Double Agent API

The Play Method

Description

Plays the specified animation.

Syntax

Visual Basic .NET

[Request = ]control.Characters ("CharacterID").GestureAt ("Animation")

Visual Basic 6

[Set Request = ]control.Characters ("CharacterID").GestureAt "Animation"

VBScript

[Set Request = ]control.Characters ("CharacterID").GestureAt "Animation"

Visual C#

[Request = ]control.Characters.this ["CharacterID"].GestureAt ("Animation")

Visual C++

character->GestureAt ("Animation", &Request)
Part Description
Animation A string value that specifies the name of an animation sequence.
Request A Request Object your application can used to track the method's status.

Remarks

An animation's name is defined when the character is authored. Before playing the specified animation, Double Agent attempts to play the Return animation for the previous animation, if one has been assigned.

The Play method does not make the character visible. If the character is not visible, Double Agent plays the animation invisibly, and sets the Status property of the Request Object.

When accessing a character's animations using a conventional file protocol, you can simply use thePlay method specifying the name of the animation. However, if you're using the HTTP protocol to access character animation data, use the Prepare method to load the animation before calling the Play method.
