Double Agent API

The Visible Property

Description

Specifies if the Advanced Character Options window is visible (open).

Syntax

Visual C++

propertysheet->get_Visible (&Visible)

propertysheet->put_Visible (Visible)

Visual C#

[Visible = ]server.PropertySheet.Visible

server.PropertySheet.Visible = Visible

Visual Basic .NET

[Visible = ]server.PropertySheet.Visible

server.PropertySheet.Visible = Visible

Visual Basic 6

[Visible = ]server.PropertySheet.Visible

server.PropertySheet.Visible = Visible

VBScript

[Visible = ]server.PropertySheet.Visible

server.PropertySheet.Visible = Visible
Part Description
Visible A boolean value indicating whether or not the Advanced Character Options window is visible.
