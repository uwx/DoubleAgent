Double Agent API

The SRListeningPrompt Property

Description

Indicates if a sound is played when Listening mode is activated or deactivated.

Syntax

Visual Basic .NET

[SRListeningPrompt = ]control.Settings.SRListeningPrompt

Visual Basic 6

[SRListeningPrompt = ]control.Settings.SRListeningPrompt

VBScript

[SRListeningPrompt = ]control.Settings.SRListeningPrompt

Visual C#

[SRListeningPrompt = ]control.Settings.SRListeningPrompt

Visual C++

settings->get_SRListeningPrompt (&SRListeningPrompt)

Part Description
SRListeningPrompt A boolean value that indicates if the Listening prompt is enabled.

Remarks

This property reflects the Play tone when you can speak option on the Speech Input page of the Advanced Character Options window.

Note: The property setting affects all characters and is read-only; only the user can set this property value.
