Double Agent API

The DefaultSearchPath Property

Description

Identifies the default search path used to fill the FilePaths property.

Syntax

Visual Basic .NET

[DefaultSearchPath = ]control.CharacterFiles.DefaultSearchPath

Visual Basic 6

[DefaultSearchPath = ]control.CharacterFiles.DefaultSearchPath

VBScript

[DefaultSearchPath = ]control.CharacterFiles.DefaultSearchPath

Visual C#

[DefaultSearchPath = ]control.CharacterFiles.DefaultSearchPath

Visual C++

characterfiles.get_DefaultSearchPath (&DefaultSearchPath)

Part Description
DefaultFilePath A string value that identifies one or more UNC paths, separated by semicolons.

Remarks

This property returns the default search path, based on the selection criteria specified by the DoubleAgentFiles, MsAgentFiles and MsOfficeFiles properties. The actual search path can be set with the SearchPath property.
