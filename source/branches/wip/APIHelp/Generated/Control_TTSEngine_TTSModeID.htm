Double Agent API

The TTSModeID Property

Description

Returns the unique identifier used to select this text-to-speech engine.

Syntax

Visual Basic .NET

[TTSModeID = ]ttsengine.TTSModeID

Visual Basic 6

[TTSModeID = ]ttsengine.TTSModeID

VBScript

[TTSModeID = ]ttsengine.TTSModeID

Visual C#

[TTSModeID = ]ttsengine.TTSModeID

Visual C++

ttsengine->get_TTSModeID (&TTSModeID)
Part Description
TTSModeID A string value that uniquely identifies a specific speech mode of a specific text-to-speech engine.

Remarks

This property corresponds to the Character.TTSModeID property. To use this TTSEngine Object, set the character's TTSModeID property to this value.
