Double Agent API

The SRHotKeyTime Property

Description

Returns the current timeout delay for the Listening key.

Syntax

Visual Basic .NET

[SRHotKeyTime = ]control.Settings.SRHotKeyTime

Visual Basic 6

[SRHotKeyTime = ]control.Settings.SRHotKeyTime

VBScript

[SRHotKeyTime = ]control.Settings.SRHotKeyTime

Visual C#

[SRHotKeyTime = ]control.Settings.SRHotKeyTime

Visual C++

settings->get_SRHotKeyTime (&SRHotKeyTime)

Part Description
SRHotKeyTime A numeric value that indicates the Listening key timeout, in seconds.

Remarks

This property reflects the And continue for … seconds … option on the Speech Input page of the Advanced Character Options window.

Note: The property setting affects all characters and is read-only; only the user can set this property value.
