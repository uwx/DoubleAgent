Double Agent API

The FindCharacterSREngines Method

Description

Retrieves a collection of Speech Recognition Engine Objects that meet a specific character's requirements.

Syntax

Visual C++

server->FindCharacterSREngines (Provider, LanguageID, &SREngines)

Visual C#

[SREngines = ]server.FindCharacterSREngines (Provider, LanguageID)

Visual Basic .NET

[SREngines = ]server.FindCharacterSREngines (Provider, LanguageID)

Visual Basic 6

[Set SREngines = ]server.FindCharacterSREngines (Provider[, LanguageID])

VBScript

[Set SREngines = ]server.FindCharacterSREngines (Provider[, LanguageID])
Part Description
Provider A string value that specifies the character file, or an empty string for the default character.
LanguageID An optional numeric value that specifies the language to match, or zero to match any language.
SREngines A SREngines Collection that matches the character's text-to-speech requirements and the specified LanguageID (if any).

Remarks

This method allows you to choose a speech recognition engine without loading the character.

For more information on specifying character file names, see the Load method.

Note: This method does not support the HTTP protocol for character files.
