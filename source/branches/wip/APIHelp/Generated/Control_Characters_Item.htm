Double Agent API

The Item Property

Description

Returns a Character Object in the collection.

Syntax

Visual Basic .NET

[Command = ]control.Characters (CharacterID)

Visual Basic 6

[Set Command = ]control.Characters (CharacterID)

VBScript

[Set Command = ]control.Characters (CharacterID)

Visual C#

[Command = ]control.Characters.this [CharacterID]

Visual C++

control->get_Item (CharacterID, &Character)
Part Description
CharacterID A string value that specifies the CharacterID assigned to the character when it was loaded.
Character The Character Object with the specified CharacterID.
