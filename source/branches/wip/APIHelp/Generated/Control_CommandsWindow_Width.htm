Double Agent API

The Width Property

Description

Returns current width, in pixels, of the Voice Commands window.

Syntax

Visual Basic .NET

[Width = ]control.CommandsWindow.Width

Visual Basic 6

[Width = ]control.CommandsWindow.Width

VBScript

[Width = ]control.CommandsWindow.Width

Visual C#

[Width = ]control.CommandsWindow.Width

Visual C++

commandswindow->get_Width (&Width)

Part Description
Width A numeric value that indicates the width, in pixels, of the Voice Commands window.

Remarks

Double Agent remembers and reuses the window width set by the user.
