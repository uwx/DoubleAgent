Double Agent API

The Height Property

Description

Returns or sets the height of the specified character's image.

Syntax

Visual Basic .NET

[Height = ]control.Characters ("CharacterID").Height

control.Characters ("CharacterID").Height = Height

Visual Basic 6

[Height = ]control.Characters ("CharacterID").Height

control.Characters ("CharacterID").Height = Height

VBScript

[Height = ]control.Characters ("CharacterID").Height

control.Characters ("CharacterID").Height = Height

Visual C#

[Height = ]control.Characters ("CharacterID").Height

control.Characters ("CharacterID").Height = Height

Visual C++

character.get_Height (&Height)

character.put_Height (Height)

Part Description
Height A numeric value that specifies the character's image height.

Remarks

The Height property is always expressed in pixels. This property's setting applies to all clients of the character.

Even though the character appears in an irregularly shaped region window, the height of the character is based on the external dimensions of the rectangular animation frame used when the character was compiled with the Microsoft Agent Character Editor.
