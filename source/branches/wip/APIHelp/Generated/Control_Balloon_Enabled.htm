Double Agent API

The Enabled Property

Description

Returns whether the word balloon is enabled for a character.

Syntax

Visual Basic .NET

[Enabled = ]control.Characters ("CharacterID").Balloon.Enabled

Visual Basic 6

[Enabled = ]control.Characters ("CharacterID").Balloon.Enabled

VBScript

[Enabled = ]control.Characters ("CharacterID").Balloon.Enabled

Visual C#

[Enabled = ]control.Characters ("CharacterID").Balloon.Enabled

Visual C++

balloon->get_Enabled (&Enabled)

Part Description
Enabled A value that indicates if the character can have a word balloon.

Remarks

The default enabled state is set as part of a character's definition when the character is compiled in the Microsoft Agent Character Editor. If a character is defined to not support the word balloon, this property will always be false for the character.

The user can override the enabled state for all characters in the Advanced Character Options window.
