Double Agent API

The DisplayName Property

Description

Returns the text-to-speech engine's human-readable name.

Syntax

Visual Basic .NET

[DisplayName = ]ttsengine.DisplayName

Visual Basic 6

[DisplayName = ]ttsengine.DisplayName

VBScript

[DisplayName = ]ttsengine.DisplayName

Visual C#

[DisplayName = ]ttsengine.DisplayName

Visual C++

ttsengine.get_DisplayName (&DisplayName)

Part Description
DisplayName A string value gives the engine's name in a form suitable for the user.
