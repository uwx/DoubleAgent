Double Agent API

The Alt1Voice Property

Description

Returns the voice text the speech recognition engine matched for the second-best matching Command.

Syntax

[Alt1Voice = ]userinput.Alt1Voice

Visual Basic 6

[Alt1Voice = ]userinput.Alt1Voice

VBScript

[Alt1Voice = ]userinput.Alt1Voice

Visual C#

[Alt1Voice = ]userinput.Alt1Voice

Visual C++

userinput->get_Alt1Voice (&Alt1Voice)

Part Description
Alt1Voice The second-best matching voice text
