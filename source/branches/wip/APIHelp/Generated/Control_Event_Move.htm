Double Agent API

The Move Event

Description

Signalled when a character has moved.

Syntax

Visual Basic .NET

Event control_Move (ByVal CharacterID As String, ByVal X As Short, ByVal Y As Short, ByVal Cause As DoubleAgent.Control.MoveCauseType)

Visual Basic 6

Sub control_Move (ByVal CharacterID As String, ByVal X As Integer, ByVal Y As Integer, ByVal Cause As MoveCauseType)

VBScript

Sub control_Move (ByVal CharacterID, ByVal X, ByVal Y, ByVal Cause)

Visual C#

delegate void control_MoveEventHandler (string CharacterID, short X, short Y, DoubleAgent.Control.MoveCauseType Cause)

Visual C++

void control_Move (LPCTSTR CharacterID, short X, short Y, MoveCauseType Cause)
Part Description
CharacterID The unique identifier of the character that was moved.
X The character's new horizontal location, in pixels, relative to the left side of the screen.
Y The character's new vertical location, in pixels, relative to the top of the screen.
Cause The reason the character moved:
1 = UserMoved The user dragged the character.
2 = ProgramMoved Your client application moved the character.
3 = OtherProgramMoved Another client application moved the character.

Remarks

This event is signalled when the user or an application changes the character's position. Coordinates are relevant to the upper left corner of the screen. This event is sent only to the clients of the character (applications that have loaded the character).
