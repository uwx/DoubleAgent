Double Agent API

The FormattedText Object

Description

The FormattedText object is used to pre-format text that is to be spoken, or shown in a thought word balloon.

Methods

Append Appends one FormattedText Object to another FormattedText Object.
Parse Parses text containing speech tags.

Properties

DisplayText Returns the text displayed in the word balloon.
DisplayWord Returns the ParsedText for a specific WordIndex.
ParsedText Returns the text formatted for display.
RawText Returns the unformatted text.
SpeechText Returns the text formatted for the Text-to-Speech engine.
WordCount Returns the number of words in the ParsedText and SpeechText properties.
WordIndex Returns the index of the word currently being spoken and/or displayed.

Remarks

If your application uses the same text for multiple calls to the Speak or Think method, it can be more efficient to pre-format the text in a FormattedText object and use the SpeakFormatted or ThinkFormatted method instead.

Also, if you want to receive the SpeechStart, SpeechEnd, and SpeechWord events, you must use the SpeakFormatted method.
