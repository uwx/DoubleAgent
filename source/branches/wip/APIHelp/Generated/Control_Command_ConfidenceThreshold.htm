Double Agent API

The ConfidenceThreshold Property

Description

Specifies if the command's ConfidenceText appears in the Listening tip.

Syntax

Visual Basic .NET

[ConfidenceThreshold = ]control.Characters ("CharacterID").Commands ("Name").ConfidenceThreshold

control.Characters ("CharacterID").Commands ("Name").ConfidenceThreshold = ConfidenceThreshold

Visual Basic 6

[ConfidenceThreshold = ]control.Characters ("CharacterID").Commands ("Name").ConfidenceThreshold

control.Characters ("CharacterID").Commands ("Name").ConfidenceThreshold = ConfidenceThreshold

VBScript

[ConfidenceThreshold = ]control.Characters ("CharacterID").Commands ("Name").ConfidenceThreshold

control.Characters ("CharacterID").Commands ("Name").ConfidenceThreshold = ConfidenceThreshold

Visual C#

[ConfidenceThreshold = ]control.Characters.this ["CharacterID"].Commands.this ["Name"].ConfidenceThreshold

control.Characters.this ["CharacterID"].Commands.this ["Name"].ConfidenceThreshold = ConfidenceThreshold

Visual C++

command->get_ConfidenceThreshold (&ConfidenceThreshold)

command->put_ConfidenceThreshold (ConfidenceThreshold)
Part Description
ConfidenceThreshold A numeric value that specifies the confidence threshold for the command.

Remarks

If the Confidence value of the best match does not exceed the value you set for the ConfidenceThreshold property, the text supplied in ConfidenceText is displayed in the Listening tip.
