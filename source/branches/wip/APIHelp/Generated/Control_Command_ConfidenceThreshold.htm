Double Agent API

The ConfidenceThreshold Property

Description

Returns or sets whether the client's ConfidenceText appears in the Listening Tip.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Commands("Name").Confidence[ = Integer]
Visual Basic 6
control.Characters ("CharacterID").Commands("Name").Confidence[ = Long]
VBScript
control.Characters ("CharacterID").Commands("Name").Confidence[ = Long]
Visual C#
control.Characters ("CharacterID").Commands("Name").Confidence[ = int]
Visual C++

command.get_Confidence (&long)

command.put_Confidence (long)

Remarks

If the returned confidence value of the best match (UserInput.Confidence) does not exceed value you set for the Confidence property, the text supplied in ConfidenceText is displayed in the Listening Tip.
