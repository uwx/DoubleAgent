Double Agent API

The ConfidenceThreshold Property

Description

Returns or sets whether the commands's ConfidenceText appears in the Listening Tip.

Syntax

Visual Basic .NET
control.Characters ("CharacterID").Commands("Name").Confidence[ = Long]
Visual Basic 6
control.Characters ("CharacterID").Commands("Name").Confidence[ = Long]
VBScript
control.Characters ("CharacterID").Commands("Name").Confidence[ = Long]
Visual C#
[ConfidenceThreshold = ]control.Characters ("CharacterID").Commands("Name").ConfidenceThreshold control.Characters ("CharacterID").Commands("Name").ConfidenceThreshold = ConfidenceThreshold
Visual C++

command.get_ConfidenceThreshold (&ConfidenceThreshold)

command.put_ConfidenceThreshold (ConfidenceThreshold)

Part Description
ConfidenceThreshold A numeric value that specifies the confidence threshold for the command.

Remarks

If the Confidence value of the best match does not exceed the value you set for the ConfidenceThreshold property, the text supplied in ConfidenceText is displayed in the Listening Tip.
