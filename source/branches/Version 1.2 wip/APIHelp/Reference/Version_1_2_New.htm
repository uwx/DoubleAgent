Double Agent SDK

What's New for Version 1.2

This is a summary of what has changed between version 1.1 and version 1.2 of Double Agent.

The Double Agent control for web pages

Double Agent 1.2 includes a new build of its ActiveX Control component specifically intended for use in web pages.  It's packaged as a downloadable ActiveX Control compatible with the <OBJECT ... CODEBASE=...> HTML construct.  See Using the Control in Web Pages for details.

Version 1.5 Character Files

Double Agent can now use character files in Microsoft® Agent 1.5 format.  Previous versions of Double Agent could only read Microsoft® Agent 2.0 character files.

The Programming Interfaces

The .NET Primary Interop Assemblies for the Server or Control interfaces have been modified to support .NET 4.0.

  • There are now two versions of each Primary Interop Assembly, one for .NET 3.5 or earlier, and one for .NET 4.0 or later.
  • Although .NET 4.0 is supposed to be able to use COM objects without any Primary Interop Assembly, there are problems when the COM objects use named enumerations.  This is why Double Agent still uses Primary Interop Assemblies, even for .NET 4.0.

The Control interface has some minor adjustments.

  • The Character.SetSize method has been added.
  • The Control.AutoConnect property has been adjusted slightly for strict Microsoft® Agent compatibility.
