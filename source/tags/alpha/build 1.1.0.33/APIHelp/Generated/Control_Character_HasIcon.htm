Double Agent API

The HasIcon Property

Description

Indicates if the character has a notification icon in its definition.

Syntax

Visual Basic .NET

[HasIcon = ]control.Characters ("CharacterID").HasIcon

Visual Basic 6

[HasIcon = ]control.Characters ("CharacterID").HasIcon

VBScript

[HasIcon = ]control.Characters ("CharacterID").HasIcon

Visual C#

[HasIcon = ]control.Characters.this ["CharacterID"].HasIcon

Visual C++

character.get_HasIcon (&HasIcon)

Part Description
HasIcon A boolean value that indicates if the character definition includes a notification icon.

Remarks

If the character definition does not include an icon (or even if it does) you can use the GenerateIcon method to create an icon from the character's animations.
