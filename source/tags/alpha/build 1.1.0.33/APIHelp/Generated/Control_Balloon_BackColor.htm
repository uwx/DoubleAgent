Double Agent API

The BackColor Property

Description

Returns or sets the background color of a character's word balloon.

Syntax

Visual Basic .NET

[BackColor = ]control.Characters ("CharacterID").Balloon.BackColor

control.Characters ("CharacterID").Balloon.BackColor = BackColor

Visual Basic 6

[BackColor = ]control.Characters ("CharacterID").Balloon.BackColor

control.Characters ("CharacterID").Balloon.BackColor = BackColor

VBScript

[BackColor = ]control.Characters ("CharacterID").Balloon.BackColor

control.Characters ("CharacterID").Balloon.BackColor = BackColor

Visual C#

[BackColor = ]control.Characters.this ["CharacterID"].Balloon.BackColor

control.Characters.this ["CharacterID"].Balloon.BackColor = BackColor

Visual C++

balloon->get_BackColor (&BackColor)

balloon->put_BackColor (BackColor)

Part Description
BackColor A numeric value that specifies the background color.

Remarks

The valid range for a normal RGB color is 0 to 16,777,215 (&HFFFFFF). The high byte of a number in this range equals 0; the lower 3 bytes, from least to most significant byte, determine the amount of red, green, and blue, respectively. The red, green, and blue components are each represented by a number between 0 and 255 (&HFF).

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
