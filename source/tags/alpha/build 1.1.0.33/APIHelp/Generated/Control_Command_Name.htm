Double Agent API

The Name Property

Description

The unique name that identifies the Command Object in the Commands Collection.

Syntax

Visual Basic .NET

[Name = ]command.Name

Visual Basic 6

[Name = ]command.Name

VBScript

[Name = ]command.Name

Visual C#

[Name = ]command.Name

Visual C++

command.get_Name (&Name)

Part Description
Name A string value that identifies the Command Object.

Remarks

When you keep a reference to a Command Object object, you can use this property to relate it back to it's owner Commands Collection.
