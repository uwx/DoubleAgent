Double Agent API

The AgentPropertyChange Event

Description

Signalled when the user changes a property in the Advanced Character Options window.

Syntax

Visual Basic .NET

Event control_AgentPropertyChange ()

Visual Basic 6

Sub control_AgentPropertyChange ()

VBScript

Sub control_AgentPropertyChange ()

Visual C#

delegate void control_AgentPropertyChangeEventHandler ()

Visual C++

void control_AgentPropertyChange ()

Remarks

This event indicates that the user has changed and applied any property in the PropertySheet window. You can use the Settings object to query the specific property settings.
