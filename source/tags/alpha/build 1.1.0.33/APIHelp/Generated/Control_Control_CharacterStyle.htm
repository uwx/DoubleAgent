Double Agent API

The CharacterStyle Property

Description

Specifies the initial style to use when a Character is loaded.

Syntax

Visual Basic .NET

[CharacterStyle = ]control.CharacterStyle

control.CharacterStyle = CharacterStyle

Visual Basic 6

[CharacterStyle = ]control.CharacterStyle

control.CharacterStyle = CharacterStyle

VBScript

[CharacterStyle = ]control.CharacterStyle

control.CharacterStyle = CharacterStyle

Visual C#

[CharacterStyle = ]control.CharacterStyle

control.CharacterStyle = CharacterStyle

Visual C++

control->get_CharacterStyle (&CharacterStyle)

control->put_CharacterStyle (CharacterStyle)

Part Description
CharacterStyle A numeric bit-field value that specifies the Style to apply when a Character is loaded.
&H00000001 = SoundEffects The animation sound effects are played.
&H00000002 = IdleEnabled The character enters idle mode and shows it's idling animations when it's not doing anything else.
&H00000004 = AutoPopupMenu A right-click on the character or it's notification icon automatically shows a pop-up menu.
&H00000008 = IconShown The character has an icon in the taskbar notification area.
&H00000020 = SmoothEdges Image smoothing (antialiasing) is applied to the edges of the character's animations.
&H00000030 = Smoothed Image smoothing (antialiasing) is applied to the character's animations.

Remarks

Setting this property allows you to establish a default Character. Style for you application.

The Smoothed and SmoothEdges bits can only be changed here. Once a Character is loaded, these bits cannot be changed in it's Style property. Also, these two settings are only used the first time the character is loaded. If another client has already loaded the character, it's smoothing settings will not change. If your client application is the first client to load the character, then your smoothing settings will apply to all clients.

For compatibility with Microsoft Agent, the IconShown bit is set to 1 by default. To load a Character without showing it's notification icon, set the IconShown bit to 0.
