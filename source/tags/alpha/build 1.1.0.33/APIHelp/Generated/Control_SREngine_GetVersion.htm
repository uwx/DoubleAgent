Double Agent API

The GetVersion Method

Description

Returns the speech recognition engine's version.

Syntax

Visual Basic .NET

srengine.GetVersion (MajorVersion, MinorVersion)

Visual Basic 6

Call srengine.GetVersion (MajorVersion, MinorVersion)

VBScript

Call srengine.GetVersion (MajorVersion, MinorVersion)

Visual C#

srengine.GetVersion (out MajorVersion, out MinorVersion)

Visual C++

srengine.GetVersion (&MajorVersion, &MinorVersion)

Part Description
MajorVersion A numeric values that identifies the engine's major version (always 5 at this time).
MinorVersion A numeric values that identifies the engine's minor version (always 0 at this time).
