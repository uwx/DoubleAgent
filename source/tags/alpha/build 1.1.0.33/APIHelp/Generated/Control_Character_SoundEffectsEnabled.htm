Double Agent API

The SoundEffectsEnabled Property

Description

Returns or sets whether sound effects are enabled for your character.

Syntax

Visual Basic .NET

[SoundEffectsEnabled = ]control.Characters ("CharacterID").SoundEffectsEnabled

control.Characters ("CharacterID").SoundEffectsEnabled = SoundEffectsEnabled

Visual Basic 6

[SoundEffectsEnabled = ]control.Characters ("CharacterID").SoundEffectsEnabled

control.Characters ("CharacterID").SoundEffectsEnabled = SoundEffectsEnabled

VBScript

[SoundEffectsEnabled = ]control.Characters ("CharacterID").SoundEffectsEnabled

control.Characters ("CharacterID").SoundEffectsEnabled = SoundEffectsEnabled

Visual C#

[SoundEffectsEnabled = ]control.Characters.this ["CharacterID"].SoundEffectsEnabled

control.Characters.this ["CharacterID"].SoundEffectsEnabled = SoundEffectsEnabled

Visual C++

character.get_SoundEffectsEnabled (&SoundEffectsEnabled)

character.put_SoundEffectsEnabled (SoundEffectsEnabled)

Part Description
SoundEffectsEnabled A boolean value specifying whether sound effects are enabled.

Remarks

This property determines whether the sound effects included as a part of a character's animations will play when an animation plays.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.

Note: This settings can be overridden by the user in the Advanced Character Options window.
