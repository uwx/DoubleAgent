Double Agent API

The TTSEnabled Property

Description

Indicates if spoken audio output is enabled.

Syntax

Visual Basic .NET

[TTSEnabled = ]control.Settings.TTSEnabled

Visual Basic 6

[TTSEnabled = ]control.Settings.TTSEnabled

VBScript

[TTSEnabled = ]control.Settings.TTSEnabled

Visual C#

[TTSEnabled = ]control.Settings.TTSEnabled

Visual C++

settings->get_TTSEnabled (&TTSEnabled)

Part Description
TTSEnabled A boolean value indicating if spoken output is enabled.

Remarks

This property reflects the Play spoken audio option on the Output page of the Advanced Character Options window.

Note: The property setting affects all characters and is read-only; only the user can set this property value.
