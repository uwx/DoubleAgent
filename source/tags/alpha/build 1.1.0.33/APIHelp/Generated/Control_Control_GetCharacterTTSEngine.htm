Double Agent API

The GetCharacterTTSEngine Method

Description

Retrieves a Text-To-Speech Engine object that meets a specific character's requirements.

Syntax

Visual Basic .NET

[TTSEngine = ]control.GetCharacterTTSEngine (Provider)

Visual Basic 6

[Set TTSEngine = ]control.GetCharacterTTSEngine (Provider)

VBScript

[Set TTSEngine = ]control.GetCharacterTTSEngine (Provider)

Visual C#

[TTSEngine = ]control.GetCharacterTTSEngine (Provider)

Visual C++

control->GetCharacterTTSEngine (Provider, &TTSEngine)

Part Description
Provider A string value that specifies the character file, or an empty string for the default character.
TTSEngine The TTSEngine Object that matches the character's text-to-speech requirements (if any).

Remarks

This method allows you to query a character's default text-to-speech engine without loading the character.

For more information on specifying character file names, see the Load method.

Note: This method does not support the HTTP protocol for character files.
