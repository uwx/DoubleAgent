Double Agent API

The OriginalWidth Property

Description

Returns a numeric value that specifies the character's original width.

Syntax

Visual Basic .NET

[OriginalWidth = ]control.Characters ("CharacterID").OriginalWidth

Visual Basic 6

[OriginalWidth = ]control.Characters ("CharacterID").OriginalWidth

VBScript

[OriginalWidth = ]control.Characters ("CharacterID").OriginalWidth

Visual C#

[OriginalWidth = ]control.Characters.this ["CharacterID"].OriginalWidth

Visual C++

character.get_OriginalWidth (&OriginalWidth)

Part Description
OriginalWidth The character's image width as defined with the Microsoft Agent Character Editor.
