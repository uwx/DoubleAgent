Double Agent API

The CharacterFiles Object

The CharacterFiles object provides information that allows you to locate locally installed character definition files. It includes a number of filtering properties that specify where to look for character files, and what kind of files to find. The results are returned in its FilePaths property.

  • The CharacterFiles object has the following properties:

    DefaultFileName, DefaultFilePath, DefaultSearchPath, DoubleAgentFiles, FilePaths, MsAgentFiles, MsOfficeFiles, NonSpeakingCharacters, SearchPath, SpeakingCharacters, VerifyVersion
