Double Agent API

The Speed Property

Description

Returns the character's speech output (text-to-speech) speed setting.

Syntax

Visual Basic .NET

[Speed = ]control.Characters ("CharacterID").Speed

Visual Basic 6

[Speed = ]control.Characters ("CharacterID").Speed

VBScript

[Speed = ]control.Characters ("CharacterID").Speed

Visual C#

[Speed = ]control.Characters.this ["CharacterID"].Speed

Visual C++

character.get_Speed (&Speed)

Part Description
Speed A numeric value that specifies the speed (in words-per-minute) of the character's default text-to-speech engine.

Remarks

This property is of limited value since it only supplies the default speed from the character's definition, scaled by the TTSSpeed property. It's supported for compatibility.

Although your application cannot write this value, you can include Spd (speed) tags in your speech output text that will temporarily change the output speed for a particular utterance. However, using the Spd tag to change the character's spoken output does not affect the Speed property value. For further information, see Speech Output Tags.
