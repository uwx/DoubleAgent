Double Agent API

The Enabled Property

Description

Returns or sets whether the Command is enabled in the character's pop-up menu.

Syntax

Visual Basic .NET

[Enabled = ]control.Characters ("CharacterID").Commands("Name").Enabled

control.Characters ("CharacterID").Commands("Name").Enabled = Enabled

Visual Basic 6

[Enabled = ]control.Characters ("CharacterID").Commands("Name").Enabled

control.Characters ("CharacterID").Commands("Name").Enabled = Enabled

VBScript

[Enabled = ]control.Characters ("CharacterID").Commands("Name").Enabled

control.Characters ("CharacterID").Commands("Name").Enabled = Enabled

Visual C#

[Enabled = ]control.Characters.this ["CharacterID"].Commands("Name").Enabled

control.Characters.this ["CharacterID"].Commands("Name").Enabled = Enabled

Visual C++

command.get_Enabled (&Enabled)

command.put_Enabled (Enabled)

Part Description
Enabled A boolean value that indicates if the command is enabled.

Remarks

If the Enabled property is set to True, the Command object's Caption appears as normal text in the character's pop-up menu when the client application is input-active. If the Enabled property is False, the Caption appears as unavailable (disabled) text. A disabled Command is also not accessible for voice input.
