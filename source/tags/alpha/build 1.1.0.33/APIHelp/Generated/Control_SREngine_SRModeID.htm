Double Agent API

The SRModeID Property

Description

Returns the unique identifier used to select this speech recognition engine.

Syntax

Visual Basic .NET

[Count = ]srengine.SRModeID

Visual Basic 6

[Count = ]srengine.SRModeID

VBScript

[Count = ]srengine.SRModeID

Visual C#

[Count = ]srengine.SRModeID

Visual C++

srengine.get_SRModeID (&SRModeID)

Part Description
SRModeID A string value that uniquely identifies the speech recognition engine.

Remarks

This property corresponds to the Character.SRModeID property. To use this SREngine Object, set the character's SRModeID property to this value.
