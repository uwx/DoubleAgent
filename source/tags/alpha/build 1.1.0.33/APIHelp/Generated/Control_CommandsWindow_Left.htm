Double Agent API

The Left Property

Description

Returns or sets the horizontal location, in pixels, of the Voice Commands window.

Syntax

Visual Basic .NET

[Left = ]control.CommandsWindow.Left

control.CommandsWindow.Left = Left

Visual Basic 6

[Left = ]control.CommandsWindow.Left

control.CommandsWindow.Left = Left

VBScript

[Left = ]control.CommandsWindow.Left

control.CommandsWindow.Left = Left

Visual C#

[Left = ]control.CommandsWindow.Left

control.CommandsWindow.Left = Left

Visual C++

commandswindow->get_Left (&Left)

commandswindow->put_Left (Left)

Part Description
Left A numeric value that specifies the left edge, in pixels, of the Voice Commands window.

Remarks

Double Agent remembers and reuses the window location set by the user.
