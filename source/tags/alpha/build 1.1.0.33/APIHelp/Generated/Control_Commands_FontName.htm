Double Agent API

The FontName Property

Description

Returns or sets the font used in the Voice Commands window.

Syntax

Visual Basic .NET

[FontName = ]control.Characters ("CharacterID").Commands.FontName

control.Characters ("CharacterID").Commands.FontName = FontName

Visual Basic 6

[FontName = ]control.Characters ("CharacterID").Commands.FontName

control.Characters ("CharacterID").Commands.FontName = FontName

VBScript

[FontName = ]control.Characters ("CharacterID").Commands.FontName

control.Characters ("CharacterID").Commands.FontName = FontName

Visual C#

[FontName = ]control.Characters.this ["CharacterID"].Commands.FontName

control.Characters.this ["CharacterID"].Commands.FontName = FontName

Visual C++

commands->get_FontName (&FontName)

commands->put_FontName (FontName)

Part Description
FontName A string value specifying the font name.

Remarks

The FontName property defines the font used to display text in the CommandsWindow. The default value for the font setting is based on the menu font for the character's LanguageID or, if that's not set, the user default language.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
