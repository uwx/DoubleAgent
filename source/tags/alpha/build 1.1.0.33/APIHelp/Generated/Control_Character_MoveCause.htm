Double Agent API

The MoveCause Property

Description

Returns a numeric value that specifies what caused the character's last move.

Syntax

Visual Basic .NET

[MoveCause = ]control.Characters ("CharacterID").MoveCause

Visual Basic 6

[MoveCause = ]control.Characters ("CharacterID").MoveCause

VBScript

[MoveCause = ]control.Characters ("CharacterID").MoveCause

Visual C#

[MoveCause = ]control.Characters.this ["CharacterID"].MoveCause

Visual C++

character.get_MoveCause (&MoveCause)

Part Description
MoveCause One of the following members of the MoveCauseType enumeration:
0 = NeverMoved The character hasn't moved.
1 = UserMoved The user dragged the character.
2 = ProgramMoved Your client application moved the character.
3 = OtherProgramMoved Another client application moved the character.

Remarks

You can use this property to determine what caused the character to move, when more than one application is sharing (has loaded) the same character. These values are the same as those returned by the Move event.
