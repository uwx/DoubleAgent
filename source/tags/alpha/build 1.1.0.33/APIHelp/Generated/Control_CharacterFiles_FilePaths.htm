Double Agent API

The FilePaths Property

Description

A list of character file paths that satisfy the filtering criteria defined by other properties in the CharacterFiles Object.

Syntax

Visual Basic .NET

[FilePaths = ]control.CharacterFiles.FilePaths

Visual Basic 6

[FilePaths = ]control.CharacterFiles.FilePaths

VBScript

[FilePaths = ]control.CharacterFiles.FilePaths

Visual C#

[FilePaths = ]control.CharacterFiles.FilePaths

Visual C++

characterfiles.get_FilePaths (&FilePaths)

Part Description
DefaultFilePath An array of string values that identify locally available character definition files.

Remarks

This property is the raison d'être of the CharacterFiles Object. As you change the other properties in this object, the contents of this array will change accordingly.
