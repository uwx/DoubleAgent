Double Agent API

The VoiceCaption Property

Description

Sets or returns the text displayed for the Command object in the Voice Commands window.

Syntax

Visual Basic .NET

[VoiceCaption = ]control.Characters ("CharacterID").Commands("Name").VoiceCaption

control.Characters ("CharacterID").Commands("Name").VoiceCaption = VoiceCaption

Visual Basic 6

[VoiceCaption = ]control.Characters ("CharacterID").Commands("Name").VoiceCaption

control.Characters ("CharacterID").Commands("Name").VoiceCaption = VoiceCaption

VBScript

[VoiceCaption = ]control.Characters ("CharacterID").Commands("Name").VoiceCaption

control.Characters ("CharacterID").Commands("Name").VoiceCaption = VoiceCaption

Visual C#

[VoiceCaption = ]control.Characters.this ["CharacterID"].Commands("Name").VoiceCaption

control.Characters.this ["CharacterID"].Commands("Name").VoiceCaption = VoiceCaption

Visual C++

command.get_VoiceCaption (&VoiceCaption)

command.put_VoiceCaption (VoiceCaption)

Remarks

This text will appear in the Voice Commands window when your client application is input-active and the character is visible. If this property is not set, the Caption property is used instead. When neither the VoiceCaption nor Caption property is set, the command does not appear in the Voice Commands window.

If you define a Command object in a Commands collection with a non-blank VoiceCaption property, you will typically also set the Command's VoiceCaption property.
