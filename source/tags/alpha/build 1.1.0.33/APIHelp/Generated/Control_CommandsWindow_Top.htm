Double Agent API

The Top Property

Description

Returns or sets the vertical location, in pixels, of the Voice Commands window.

Syntax

Visual Basic .NET

[Top = ]control.CommandsWindow.Top

control.CommandsWindow.Top = Top

Visual Basic 6

[Top = ]control.CommandsWindow.Top

control.CommandsWindow.Top = Top

VBScript

[Top = ]control.CommandsWindow.Top

control.CommandsWindow.Top = Top

Visual C#

[Top = ]control.CommandsWindow.Top

control.CommandsWindow.Top = Top

Visual C++

commandswindow->get_Top (&Top)

commandswindow->put_Top (Top)

Part Description
Top A numeric value that specifies the top edge, in pixels, of the Voice Commands window.

Remarks

Double Agent remembers and reuses the window location set by the user.
