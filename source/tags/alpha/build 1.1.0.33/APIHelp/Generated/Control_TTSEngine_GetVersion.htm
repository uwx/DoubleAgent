Double Agent API

The GetVersion Method

Description

Returns the text-to-speech engine's version.

Syntax

Visual Basic .NET

ttsengine.GetVersion (MajorVersion, MinorVersion)

Visual Basic 6

Call ttsengine.GetVersion (MajorVersion, MinorVersion)

VBScript

Call ttsengine.GetVersion (MajorVersion, MinorVersion)

Visual C#

ttsengine.GetVersion (out MajorVersion, out MinorVersion)

Visual C++

ttsengine.GetVersion (&MajorVersion, &MinorVersion)

Part Description
MajorVersion A numeric values that identifies the engine's major version (always 4 or 5 at this time).
MinorVersion A numeric values that identifies the engine's minor version (always 0 at this time).

Remarks

You can use this property to determine if the text-to-speech engine is a SAPI4 or SAPI5 engine.
