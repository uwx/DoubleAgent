Double Agent API

The Request Object Properties

Description

Double Agent processes some methods, such as Load, Play, and Speak, asynchronously. This allows your application code to continue while the method is completing. When a client application calls one of these methods, the control queues the request and returns a Request object. The Request object allows you to check the status of your request by keeping a reference to the object.

Syntax

Visual Basic .NET

[Status = ]request.Status

[Number = ]request.Number

[Description = ]request.Description

Visual Basic 6

[Status = ]request.Status

[Number = ]request.Number

[Description = ]request.Description

VBScript

[Status = ]request.Status

[Number = ]request.Number

[Description = ]request.Description

Visual C#

[Status = ]request.Status

[Number = ]request.Number

[Description = ]request.Description

Visual C++

request->get_Status (&Status)

request->get_Number (&Number)

request->get_Description (&Description)

Part Description
Status The current status of the request:
0 = Success The request was successfully completed.
1 = Failed The request failed and an error code is available in the Request object's Number property.
2 = Pending The request is on the queue.
3 = Interrupted The request was interrupted and an error code is available in the Request object's Number property.
4 = InProgress The request is currently in progress.
Number The error or cause of the Status code. If there is no error, this value is zero.
Description A string value that corresponds to the error Number. If no description is found for an error, Description contains "Application-defined or object-defined error" .

Remarks

For the values and meanings of the Number property, see Microsoft Agent Error Codes .

Example

Dim MyRequest

Set MyRequest = AgentControl.Characters("Genie").Play ("GetAttention")

If (MyRequest.Status = 2) then
    'do something
Else If (MyRequest.Status = 0) then
    'do something right away
End If
