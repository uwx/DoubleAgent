Double Agent API

The LanguageNames Property

Description

Returns the speech recognition engine's human-readable supported language names.

Syntax

Visual Basic .NET

[LanguageNames = ]srengine.LanguageNames[ [EnglishNames]]

Visual Basic 6

[LanguageNames = ]srengine.LanguageNames[ [EnglishNames]]

VBScript

[LanguageNames = ]srengine.LanguageNames[ [EnglishNames]]

Visual C#

[LanguageNames = ]srengine.get_LanguageNames (EnglishNames)

Visual C++

srengine.get_LanguageNames (EnglishNames, &LanguageNames)

Part Description
LanguageNames An array of string values that identify all of the engine's supported languages in a form suitable for the user.
EnglishNames An optional boolean value that specifies if the language names should be in English, or in the user's language. The default value is True.
