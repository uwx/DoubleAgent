Double Agent API

The FontUnderline Property

Description

Returns or sets the font style of a character's word balloon.

Syntax

Visual Basic .NET

[FontUnderline = ]control.Characters ("CharacterID").Balloon.FontUnderline

control.Characters ("CharacterID").Balloon.FontUnderline = FontUnderline

Visual Basic 6

[FontUnderline = ]control.Characters ("CharacterID").Balloon.FontUnderline

control.Characters ("CharacterID").Balloon.FontUnderline = FontUnderline

VBScript

[FontUnderline = ]control.Characters ("CharacterID").Balloon.FontUnderline

control.Characters ("CharacterID").Balloon.FontUnderline = FontUnderline

Visual C#

[FontUnderline = ]control.Characters.this ["CharacterID"].Balloon.FontUnderline

control.Characters.this ["CharacterID"].Balloon.FontUnderline = FontUnderline

Visual C++

balloon->get_FontUnderline (&FontUnderline)

balloon->put_FontUnderline (FontUnderline)

Part Description
FontUnderline A boolean value that indicates if the font is underlined.

Remarks

The default values for the font of a character's word balloon are set in the Microsoft Agent Character Editor. In addition, the user can override font settings for all characters in the Double Agent Property Sheet.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
