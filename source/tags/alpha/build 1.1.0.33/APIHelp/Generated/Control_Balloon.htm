Double Agent API

The Balloon Object

Double Agent supports textual captioning of the Speak method using a cartoon word balloon. The Think method lets you display text without audio output in a thought balloon.

The Balloon object allows you to customize the appearance and behaviour of a character's word balloon. It can be accessed using the Character's Balloon property.

  • The Balloon object has the following properties:

    AutoHide, AutoPace, BackColor, BorderColor, CharsPerLine, Enabled, FontBold, FontCharSet, FontItalic, FontName, FontSize, FontStrikethru, FontUnderline, NumberOfLines, ShowPartialLines, SizeToText, Style, TextColor, Visible

Remarks

A character's initial word balloon window defaults are defined and compiled in the Microsoft Agent Character Editor. Once running, the balloon's Enabled and Font properties may be overridden by the user. If a user changes the word balloon's properties, they affect all characters. Both the Speak and Think word balloons use the same property settings for size. You can access the properties for a character's word balloon through the Balloon object, which is a child of the Character object.
