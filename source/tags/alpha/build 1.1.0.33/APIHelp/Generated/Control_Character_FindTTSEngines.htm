Double Agent API

The FindTTSEngines Method

Description

Retrieves a collection of Text-To-Speech Engine objects that meet the character's requirements.

Syntax

Visual Basic .NET

[TTSEngines = ]control.Characters ("CharacterID").FindTTSEngines (LanguageID)

Visual Basic 6

[Set TTSEngines = ]control.Characters ("CharacterID").FindTTSEngines (LanguageID)

VBScript

[Set TTSEngines = ]control.Characters ("CharacterID").FindTTSEngines (LanguageID)

Visual C#

[TTSEngines = ]control.Characters.this ["CharacterID"].FindTTSEngines (LanguageID)

Visual C++

character->FindTTSEngines (LanguageID, &TTSEngines)

Part Description
LanguageID A numeric value that specifies the language to match, or zero for any language.
TTSEngines A TTSEngines Collection that matches the character's text-to-speech requirements and the specified LanguageID.
