Double Agent API

The Style Property

Description

Returns or sets the character's word balloon output style.

Syntax

Visual Basic .NET

[Style = ]control.Characters ("CharacterID").Balloon.Style

control.Characters ("CharacterID").Balloon.Style = Style

Visual Basic 6

[Style = ]control.Characters ("CharacterID").Balloon.Style

control.Characters ("CharacterID").Balloon.Style = Style

VBScript

[Style = ]control.Characters ("CharacterID").Balloon.Style

control.Characters ("CharacterID").Balloon.Style = Style

Visual C#

[Style = ]control.Characters.this ["CharacterID"].Balloon.Style

control.Characters.this ["CharacterID"].Balloon.Style = Style

Visual C++

balloon->get_Style (&Style)

balloon->put_Style (Style)

Part Description
Style An numeric value that represents the balloon's output style. The style value is a bit field with bits defined as follows:
bit 0 (&H00000001)Enabled
bit 1 (&H00000002)SizeToText
bit 2 (&H00000004)AutoHide
bit 3 (&H00000008)AutoPace
bit 4 (&H00000010)ShowPartialLines
bits 16-23CharsPerLine
bits 24-31NumberOfLines

Remarks

All of the settings included in the Style property can be accessed using specific Balloon object properties. The Style property lets you retrieve or set all of these settings at once.

  • When the Enabled style is set, the word balloon appears when the Speak or Think method is used, unless the user overrides this setting in the Advanced Character Options window. When it's not set, the word balloon does not appear.
  • When the SizeToText style is set, the word balloon automatically adjusts the height of the balloon to the current size of the text for the Speak or Think request. When it's not set, the balloon's height is based on the NumberOfLines property. If this style is set and you attempt to set the NumberOfLines property, Double Agent raises an error.
  • When the AutoHide style is set, the word balloon automatically hides when spoken output completes. When it's not set, the balloon remains displayed until the next Speak or Think call, the character is hidden, or the user clicks or drags the character.
  • When the AutoPace style is set, the word balloon paces its text output based on the current output rate, for example one word at a time. When output exceeds the size of the balloon, the text is automatically scrolled. When it's not set, all text included in a Speak or Think request is displayed at once.
  • When the ShowPartialLines style is set, the word balloon will show partial lines at the bottom as it scrolls its text. When it's not set, only full lines are shown. This style only applies when the SizeToText style is not set.
  • The Style property also contains the CharsPerLine in the lower byte of the upper word and the NumberOfLines in the high byte of the upper word. The value for the CharsPerLine must be between 8 and 255, and the NumberOfLines must be between 1 and 128. If you provide an invalid value for the Style property, Double Agent will raise an error.

The defaults for the Style property are set when the character is compiled with the Microsoft Agent Character Editor.

Note: The Style property can be set even if the user has disabled word balloons in the Advanced Character Options window.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
