Double Agent API

The CommandsWindow Object

The CommandsWindow object provides access to Double Agent's Voice Commands window. It allows you to show and hide the window, and retrieve and update the window's location. It can be accessed using the Control's CommandsWindow property.

  • The CommandsWindow object has the following properties:

    Left, Height, Top, Visible, Width

Remarks

The Voice Commands window is shared among all Double Agent clients. Changes to the window's location and visibility should be made with this in mind.
