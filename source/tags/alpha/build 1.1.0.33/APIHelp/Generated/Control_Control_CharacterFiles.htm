The CharacterFiles Property

Syntax

Visual Basic 6

[Set CharacterFiles = ]control.CharacterFiles

VBScript

Visual C#

[CharacterFiles = ]control.CharacterFiles

Visual C++

control->get_CharacterFiles (&CharacterFiles)
