Double Agent API

The ConfidenceText Property

Description

Returns or sets the text that appears in the Listening tip.

Syntax

Visual Basic .NET

[ConfidenceText = ]control.Characters ("CharacterID").Commands("Name").ConfidenceText

control.Characters ("CharacterID").Commands("Name").ConfidenceText = ConfidenceText

Visual Basic 6

[ConfidenceText = ]control.Characters ("CharacterID").Commands("Name").ConfidenceText

control.Characters ("CharacterID").Commands("Name").ConfidenceText = ConfidenceText

VBScript

[ConfidenceText = ]control.Characters ("CharacterID").Commands("Name").ConfidenceText

control.Characters ("CharacterID").Commands("Name").ConfidenceText = ConfidenceText

Visual C#

[ConfidenceText = ]control.Characters.this ["CharacterID"].Commands("Name").ConfidenceText

control.Characters.this ["CharacterID"].Commands("Name").ConfidenceText = ConfidenceText

Visual C++

command.get_ConfidenceText (&ConfidenceText)

command.put_ConfidenceText (ConfidenceText)

Part Description
ConfidenceText A string value that specifies the text displayed in the Listening Tip .

Remarks

When the Confidence value of the best match does not exceed the command's ConfidenceThreshold property, Double Agent displays the text specified in ConfidenceText in the Listening Tip .
