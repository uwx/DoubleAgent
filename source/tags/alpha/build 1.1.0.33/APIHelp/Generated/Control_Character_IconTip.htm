The IconTip Property

Description

Specifies the tip text associated with the character's notification icon.

Syntax

Visual Basic .NET

[IconTip = ]control.Characters ("CharacterID").IconTip

control.Characters ("CharacterID").IconTip = IconTip

Visual Basic 6

[IconTip = ]control.Characters ("CharacterID").IconTip

control.Characters ("CharacterID").IconTip = IconTip

VBScript

[IconTip = ]control.Characters ("CharacterID").IconTip

control.Characters ("CharacterID").IconTip = IconTip

Visual C#

[IconTip =] control.Characters.this ["CharacterID"].IconTip

control.Characters.this ["CharacterID"].IconTip = IconTip

Visual C++

character.get_IconTip (&IconTip)

character.put_IconTip (IconTip)

Part Description
IconTip A string value that specifies the icon's tip text. An empty string means that the default tip (the character's name) is used.

Remarks

By default, the character's name is shown in the notification icon tip. You can use this property to show your own tip.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
