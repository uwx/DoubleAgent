Double Agent API

The LanguageName Property

Description

Returns the text-to-speech engine's human-readable language name.

Syntax

Visual Basic .NET

[LanguageName = ]ttsengine.LanguageName[ [EnglishName]]

Visual Basic 6

[LanguageName = ]ttsengine.LanguageName[ [EnglishName]]

VBScript

[LanguageName = ]ttsengine.LanguageName[ [EnglishName]]

Visual C#

[LanguageName = ]ttsengine.get_LanguageName (EnglishName)

Visual C++

ttsengine.get_LanguageName (EnglishName, &LanguageName)

Part Description
LanguageName A string value that identifies the engine's language in a form suitable for the user.
EnglishName An optional boolean value that specifies if the language name should be in English, or in the user's language. The default value is True.
