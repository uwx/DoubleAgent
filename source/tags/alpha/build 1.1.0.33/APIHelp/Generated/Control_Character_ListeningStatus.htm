Double Agent API

The ListeningStatus Property

Description

Returns whether speech input can be started for the character.

Syntax

Visual Basic .NET

[SRStatus = ]control.Characters ("CharacterID").SRStatus

Visual Basic 6

[SRStatus = ]control.Characters ("CharacterID").SRStatus

VBScript

[SRStatus = ]control.Characters ("CharacterID").SRStatus

Visual C#

[SRStatus = ]control.Characters.this ["CharacterID"].SRStatus

Visual C++

character.get_SRStatus (&SRStatus)

Part Description
ListeningStatus One of the following members of the ListeningStatusType enumeration:
0 = Available Conditions support speech input.
2 = CharacterInactive Speech recognition is unavailable because another client is the active client of this character, or the current character is not input-active.
4 = InitializeFailed Speech recognition is unavailable because it could not be initialized. This includes the possibility that there is no speech engine available matching the character's language setting.
5 = InputDisabled The user has disabled speech input in the Advanced Character Options window.
6 = Error The speech recognition status could not be determined.

Remarks

This property checks for the conditions necessary to support speech input. You can check this property before you call the Listen method to better ensure its success.

Details

When speech input is enabled in the Advanced Character Options window, querying this property will load the associated engine, if it is not already loaded, and start speech services. That is, the Listening key is available, and the Listening tip is automatically displayable. (The Listening key and Listening tip are only enabled if they are also enabled in the Advanced Character Options window.) However, if you query the property when speech is disabled, Double Agent does not start speech services.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
