Double Agent API

The SizeToText Property

Description

Indicates if the word balloon is automatically resized to fit it's contents.

Syntax

Visual Basic .NET

[SizeToText = ]control.Characters ("CharacterID").Balloon.SizeToText

control.Characters ("CharacterID").Balloon.SizeToText = SizeToText

Visual Basic 6

[SizeToText = ]control.Characters ("CharacterID").Balloon.SizeToText

control.Characters ("CharacterID").Balloon.SizeToText = SizeToText

VBScript

[SizeToText = ]control.Characters ("CharacterID").Balloon.SizeToText

control.Characters ("CharacterID").Balloon.SizeToText = SizeToText

Visual C#

[SizeToText = ]control.Characters.this ["CharacterID"].Balloon.SizeToText

control.Characters.this ["CharacterID"].Balloon.SizeToText = SizeToText

Visual C++

balloon->get_SizeToText (&SizeToText)

balloon->put_SizeToText (SizeToText)

Part Description
SizeToText A boolean value that indicates if the SizeToText style is enabled.

Remarks

When the SizeToText style is set, the word balloon automatically adjusts the height of the balloon to the current size of the text for the Speak or Think request. When it's not set, the balloon's height is based on the NumberOfLines property. If this style is set and you attempt to set the NumberOfLines property, Double Agent raises an error.
