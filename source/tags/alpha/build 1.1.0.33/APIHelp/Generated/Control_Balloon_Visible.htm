Double Agent API

The Visible Property

Description

Returns or sets the visiblity of a character's word balloon.

Syntax

Visual Basic .NET

[Visible = ]control.Characters ("CharacterID").Balloon.Visible

control.Characters ("CharacterID").Balloon.Visible = Visible

Visual Basic 6

[Visible = ]control.Characters ("CharacterID").Balloon.Visible

control.Characters ("CharacterID").Balloon.Visible = Visible

VBScript

[Visible = ]control.Characters ("CharacterID").Balloon.Visible

control.Characters ("CharacterID").Balloon.Visible = Visible

Visual C#

[Visible = ]control.Characters.this ["CharacterID"].Balloon.Visible

control.Characters.this ["CharacterID"].Balloon.Visible = Visible

Visual C++

balloon->get_Visible (&Visible)

balloon->put_Visible (Visible)

Part Description
Visible A boolean value that indicates if the balloon is visible.

Remarks

If you follow a Speak or Think call with a statement that changes this property, it may not affect the balloon's Visible state because the Speak or Think call gets queued, but the call to set this property does not. Therefore, only set this value when no Speak or Think calls are in the character's queue.

If you attempt to set this property while the character is speaking, moving, or being dragged, the property setting does not take effect until the preceding operation is completed.

Calling the Speak and Think methods automatically makes the balloon visible, setting the Visible property to True. If the character's balloon AutoHide property is enabled, the balloon is automatically hidden after the text output is complete. Clicking or dragging a character that is not currently speaking also automatically hides the balloon even if its AutoHide setting is disabled.
