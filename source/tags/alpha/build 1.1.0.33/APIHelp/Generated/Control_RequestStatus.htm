The RequestStatus Enumeration
Specifies the Status of a Request object.
0 = Success The request was successfully completed.
1 = Failed The request failed and an error code is available in the Request object's Number property.
2 = Pending The request is on the queue.
3 = Interrupted The request was interrupted and an error code is available in the Request object's Number property.
4 = InProgress The request is currently in progress.
