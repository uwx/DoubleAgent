The TTSEngines Object

The TTSEngines collection contains a selected set of TTSEngine objects. You can use it to help choose a character's text-to-speech engine.

  • The TTSEngines object has the following properties:

    Count, Item

Remarks

The contents of the collection depends on how it was retrieved.

When the TTSEngines Collection is retrieved using matching criteria (for example using the FindTTSEngines method), it is an ordered collection. It's items are in order from best to worst match.
