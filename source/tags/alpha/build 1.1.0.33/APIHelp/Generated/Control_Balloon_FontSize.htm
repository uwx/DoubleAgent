Double Agent API

The FontSize Property

Description

Returns or sets the font size supported for the word balloon for the specified character.

Syntax

Visual Basic .NET

[FontSize = ]control.Characters ("CharacterID").Balloon.FontSize

control.Characters ("CharacterID").Balloon.FontSize = FontSize

Visual Basic 6

[FontSize = ]control.Characters ("CharacterID").Balloon.FontSize

control.Characters ("CharacterID").Balloon.FontSize = FontSize

VBScript

[FontSize = ]control.Characters ("CharacterID").Balloon.FontSize

control.Characters ("CharacterID").Balloon.FontSize = FontSize

Visual C#

[FontSize = ]control.Characters.this ["CharacterID"].Balloon.FontSize

control.Characters.this ["CharacterID"].Balloon.FontSize = FontSize

Visual C++

balloon->get_FontSize (&FontSize)

balloon->put_FontSize (FontSize)

Part Description
FontSize A numeric value that specifies the font size, in points.

Remarks

The default values for the font of a character's word balloon are set in the Microsoft Agent Character Editor. In addition, the user can override font settings for all characters in the Double Agent Property Sheet.

Note: This property applies only to your client application's use of the character; the setting does not affect other clients of the character or other characters of your client application.
