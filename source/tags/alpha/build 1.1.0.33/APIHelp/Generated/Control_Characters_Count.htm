The Count Property

Description

Returns the count of Character objects in the Characters collection.

Syntax

Visual Basic .NET
[Count = ]control.Characters.Count
Visual Basic 6
[Count = ]control.Characters.Count
VBScript
[Count = ]control.Characters.Count
Visual C#
[Count = ]control.Characters.Count
Visual C++
characters.get_Count (&Count)
Part Description
Count A numeric value that indicates the number of Character objects in the collection.
