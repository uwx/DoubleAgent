The Advanced Character Options Window

The Advanced Character Options window allows users to customize their interactions with all characters. For example, users can disable speech input or change speech input preferences. Users can also change the output settings for the word balloon. These settings override any set by a client application or declared as part of the character definition.

Your application cannot change or disable these options, because they are the user's preferences for all characters. However, the server will notify your application with the AgentPropertyChange or DefaultCharacterChange event when the user applies changes to these settings.

The preference settings can be accessed using the Settings object. You can also display or close the window using the PropertySheet object's Visible property, and access its location through its Left, Top, Width, and Height properties.

Note: The Double Agent Advanced Character Options window combines the Microsoft Agent Advanced Character Options window and the Microsoft Agent Default Character Properties window.

Note: The pages in the Advanced Character Options window are also included in the Double Agent Control Panel.
