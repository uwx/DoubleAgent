Image Rendering

Animation frames can be rendered without or with image smoothing. The difference is subtle, but edge smoothing can have a particularly noticeable effect in moving animations. These options only apply when animation frames are rendered at their normal size. When an animation is stretched or shrunk, Double Agent uses bicubic interpolation to resize its frames.
