Double Agent API

The Advanced Character Options Window

Double Agent maintains preference settings that allow a user to control interaction with all characters. The Advanced Character Options window displays these options and their current settings, and can be shown by any hosting application using the Double Agent programming interface. Double Agent remembers the last page viewed and displays that page when the property sheet appears.

The Output Page

This page includes properties that control character output. For example, the user can decide whether to display output in the word balloon, decide how word balloon output should appear, play spoken output as audio, play character sound effects, and adjust the speaking speed.

The Speech Input Page

A user can adjust speech input options on this property page. The user can disable speech input, set the Listening key, choose whether to display the Listening tip, and choose to play a tone to indicate when speech input is available.

The Gallery Page

As an alternative to loading a specific character, applications can load a character that is a shared resource for the user, known as the default character. The default character is available to any application, but only the user can select it. The Gallery page allows the user to select the default character.

Note: The Gallery page used to be in the Default Character Properties window, which no longer exists as of Double Agent version 1.1.

The Copyright Page

This page displays the copyright and version information for Double Agent and Microsoft® Agent.
