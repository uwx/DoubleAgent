Double Agent API

Updating from Previous Versions

Applications built and compiled for Microsoft® Agent 2.0 should run without modification using Double Agent 1.0.  For compatibility, all of the Help methods, properties, and events are included in Double Agent but they're non-functional.  The same applies for methods, properties, and events that are obsolete in Microsoft® Agent 2.0.

Applications built and compiled for Microsoft® Agent 2.0 or Double Agent 1.0 should run without modification using Double Agent 1.1.  Deprecated methods, properties, and events are still implemented.  To use the new features of Double Agent 1.1, you'll need to modify your code.
